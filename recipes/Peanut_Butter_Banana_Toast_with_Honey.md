+----------------------------------------------------------------------+
| **\#79 Peanut Butter & Banana Toast with Honey** \| American ·       |
| Pre-Game Fueling                                                     |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 3 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 slices white bread or whole wheat (use GF bread for GF/WF)     |
| >                                                                    |
| > • 3 tbsp natural peanut butter (omit for PF --- use sunflower      |
| > butter)                                                            |
| >                                                                    |
| > • 2 ripe bananas, sliced                                           |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • Optional: sprinkle of chia seeds for Omega-3                     |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Toast bread to golden.                                         |
| >                                                                    |
| > 2\. Spread peanut butter evenly on each slice.                     |
| >                                                                    |
| > 3\. Layer banana slices on top; drizzle with honey.                |
| >                                                                    |
| > 4\. Eat 1--2 hours before game time.                               |
| >                                                                    |
| > 5\. This is a quick-energy snack version --- scale up to a full    |
| > meal by adding a glass of chocolate milk.                          |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 14g \| Calories: 350 kcal \|      |
| Carbs: 52g \| Fat: 10g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Fish-Free\]             |
| \[Shellfish-Free\] \[Tree Nut-Free\] \[Soy-Free\] \[Sesame-Free\]    |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Carb\] \[High Potassium\] \[High          |
| Magnesium\] \[High Folate\] \[Vitamin C\] \[Omega-3\]                |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains wheat and peanuts. Use GF bread for GF/WF; |
| use sunflower butter for PF. A fast, portable pre-game option. Add   |
| honey for quick carbs. Avoid high-fiber bread before competition to  |
| minimize GI discomfort.*                                             |
