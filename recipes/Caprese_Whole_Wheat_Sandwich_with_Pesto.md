+----------------------------------------------------------------------+
| **\#50 Caprese Whole Wheat Sandwich with Pesto** \| American · Lunch |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 1                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 slices whole wheat sourdough or multigrain bread               |
| >                                                                    |
| > • 3 oz (85 g) fresh mozzarella, sliced                             |
| >                                                                    |
| > • 1 medium tomato, sliced                                          |
| >                                                                    |
| > • 6 fresh basil leaves                                             |
| >                                                                    |
| > • 1 tbsp basil pesto (store-bought or homemade --- check for tree  |
| > nuts)                                                              |
| >                                                                    |
| > • 1 tsp balsamic glaze                                             |
| >                                                                    |
| > • Salt and black pepper to taste                                   |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Toast bread lightly if desired.                                |
| >                                                                    |
| > 2\. Spread pesto on one slice.                                     |
| >                                                                    |
| > 3\. Layer mozzarella, tomato, and basil on top. Season with salt   |
| > and pepper.                                                        |
| >                                                                    |
| > 4\. Drizzle balsamic glaze over.                                   |
| >                                                                    |
| > 5\. Top with second slice. Slice diagonally and serve.             |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 20g \| Calories: 420 kcal \|      |
| Carbs: 38g \| Fat: 18g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Shellfish-Free\] \[Fish-Free\]         |
| \[Peanut-Free\] \[Sesame-Free\] \[Soy-Free\]                         |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Calcium\] \[Vitamin C\]  |
| \[High Potassium\] \[High Folate\] \[Omega-3\]                       |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains wheat, dairy, and tree nuts (pesto). Use   |
| GF bread for GF. Use nut-free pesto (basil + olive oil + garlic +    |
| Parmesan) for TNF. Omit cheese for DF.*                              |
