+----------------------------------------------------------------------+
| **\#97 Grilled Chicken & Vegetable Recovery Bowl with Quinoa** \|    |
| American · Post-Game Recovery                                        |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs boneless chicken breast, grilled and sliced              |
| >                                                                    |
| > • 1½ cups quinoa, rinsed and cooked                                |
| >                                                                    |
| > • 2 cups roasted broccoli florets                                  |
| >                                                                    |
| > • 1 cup cherry tomatoes, halved                                    |
| >                                                                    |
| > • 1 avocado, sliced                                                |
| >                                                                    |
| > • 2 tbsp olive oil                                                 |
| >                                                                    |
| > • Juice of 1 lemon                                                 |
| >                                                                    |
| > • 1 tsp garlic powder                                              |
| >                                                                    |
| > • ½ tsp turmeric (anti-inflammatory)                               |
| >                                                                    |
| > • Salt and pepper to taste                                         |
| >                                                                    |
| > • Fresh parsley to garnish                                         |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Toss broccoli with 1 tbsp olive oil, garlic powder, salt, and  |
| > pepper; roast at 400°F for 18--20 min.                             |
| >                                                                    |
| > 2\. Grill chicken breast seasoned with turmeric, salt, and pepper, |
| > 5--6 min per side. Slice.                                          |
| >                                                                    |
| > 3\. Cook quinoa per package instructions.                          |
| >                                                                    |
| > 4\. Build bowls: quinoa base, sliced chicken, roasted broccoli,    |
| > cherry tomatoes, avocado.                                          |
| >                                                                    |
| > 5\. Drizzle with remaining olive oil and lemon juice; garnish with |
| > parsley.                                                           |
