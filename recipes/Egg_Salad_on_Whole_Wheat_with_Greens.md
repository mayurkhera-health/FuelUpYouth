+----------------------------------------------------------------------+
| **\#33 Egg Salad on Whole Wheat with Greens** \| American · Lunch    |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 10 min \| 🍽 Serves: 1                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 3 large eggs, hard-boiled and peeled                             |
| >                                                                    |
| > • 1½ tbsp plain Greek yogurt                                       |
| >                                                                    |
| > • 1 tsp Dijon mustard                                              |
| >                                                                    |
| > • 1 stalk celery, finely diced                                     |
| >                                                                    |
| > • Salt, black pepper, smoked paprika to taste                      |
| >                                                                    |
| > • 2 slices whole wheat bread, toasted                              |
| >                                                                    |
| > • Handful baby spinach or arugula                                  |
| >                                                                    |
| > • 2 slices tomato                                                  |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Hard-boil eggs: cover with cold water, bring to a boil, cook   |
| > 10 min, cool in ice water, peel.                                   |
| >                                                                    |
| > 2\. Mash eggs with a fork. Mix in yogurt, mustard, celery, salt,   |
| > pepper, and paprika.                                               |
| >                                                                    |
| > 3\. Toast bread. Layer spinach, egg salad, and tomato on one       |
| > slice. Close with second slice.                                    |
| >                                                                    |
| > 4\. Slice diagonally and serve immediately or wrap in parchment    |
| > for a packed lunch.                                                |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 24g \| Calories: 370 kcal \|      |
| Carbs: 32g \| Fat: 14g                                               |

| \[Peanut-Free\] \[Sesame-Free\] \[Soy-Free\]                         |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[Vitamin D\] \[B12-Rich\]      |

+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains eggs, wheat, and dairy. Use GF bread for   |
| GF version. Dairy-free yogurt swap works well. Egg salad keeps       |
| refrigerated up to 2 days --- store separately from bread.*          |
