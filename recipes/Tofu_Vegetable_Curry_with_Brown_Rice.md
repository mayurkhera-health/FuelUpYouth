+----------------------------------------------------------------------+
| **\#60 Tofu & Vegetable Curry with Brown Rice** \| Indian · Dinner   |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 25 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 14 oz extra-firm tofu, pressed and cubed                         |
| >                                                                    |
| > • 2 cups brown rice, cooked                                        |
| >                                                                    |
| > • 1 can (14 oz) full-fat coconut milk                              |
| >                                                                    |
| > • 1 can (14 oz) diced tomatoes                                     |
| >                                                                    |
| > • 2 cups baby spinach                                              |
| >                                                                    |
| > • 1 cup frozen peas                                                |
| >                                                                    |
| > • 1 medium onion, diced                                            |
| >                                                                    |
| > • 3 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tbsp fresh ginger, grated                                      |
| >                                                                    |
| > • 2 tbsp avocado oil                                               |
| >                                                                    |
| > • 1½ tsp curry powder                                              |
| >                                                                    |
| > • 1 tsp turmeric                                                   |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • ½ tsp garam masala                                               |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • Fresh cilantro and lemon wedges                                  |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Press tofu with paper towels or a tofu press for 10 min; cube  |
| > and pat dry.                                                       |
| >                                                                    |
| > 2\. Heat 1 tbsp oil in a large skillet over medium-high; pan-fry   |
| > tofu 3--4 min per side until golden. Remove and set aside.         |
| >                                                                    |
| > 3\. In the same skillet, heat remaining oil; sauté onion until     |
| > golden (\~6 min). Add garlic and ginger; cook 1 min.               |
| >                                                                    |
| > 4\. Add all spices; toast 30 seconds until fragrant.               |
| >                                                                    |
| > 5\. Pour in diced tomatoes; cook 3 min. Add coconut milk; stir     |
| > well.                                                              |
| >                                                                    |
| > 6\. Simmer sauce 8--10 min until slightly thickened.               |
| >                                                                    |
| > 7\. Stir in frozen peas and spinach; cook 2--3 min until wilted.   |
| >                                                                    |
| > 8\. Return tofu to the pan; gently fold in. Adjust seasoning.      |
| >                                                                    |
| > 9\. Serve over brown rice with fresh cilantro and a squeeze of     |
| > lemon.                                                             |
