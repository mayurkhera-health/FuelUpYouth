+----------------------------------------------------------------------+
| **\#55 Bulgogi Beef Bowl with Rice & Kimchi** \| Korean · Dinner     |
+======================================================================+
| ⏱ Prep: 20 min \| 🍳 Cook: 10 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 lb beef ribeye or sirloin, very thinly sliced                  |
| >                                                                    |
| > • 2 cups short-grain white rice or brown rice, cooked              |
| >                                                                    |
| > • 1 cup store-bought kimchi (or cucumber slices for                |
| > SoyF/low-sodium)                                                   |
| >                                                                    |
| > • 2 cups baby spinach or shredded lettuce                          |
| >                                                                    |
| > • 1 medium carrot, julienned                                       |
| >                                                                    |
| > • 1 cucumber, sliced                                               |
| >                                                                    |
| > • 2 tbsp toasted sesame seeds (omit for SesF)                      |
| >                                                                    |
| > • 3 tbsp low-sodium soy sauce (or coconut aminos)                  |
| >                                                                    |
| > • 1 tbsp brown sugar or honey                                      |
| >                                                                    |
| > • 1 tbsp sesame oil (omit for SesF)                                |
| >                                                                    |
| > • 3 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 2 green onions, chopped                                          |
| >                                                                    |
| > • ½ Asian pear or apple, grated (natural tenderizer)               |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Combine all marinade ingredients in a bowl; whisk until sugar  |
| > dissolves.                                                         |
| >                                                                    |
| > 2\. Add beef slices to marinade; toss well and marinate at least   |
| > 15 min (or overnight in the fridge).                               |
| >                                                                    |
| > 3\. Heat a cast-iron skillet or grill pan over high heat until     |
| > very hot.                                                          |
| >                                                                    |
| > 4\. Cook beef in batches (don\'t crowd the pan) 1--2 min per side  |
| > until caramelized.                                                 |
| >                                                                    |
| > 5\. Build bowls: rice base, topped with bulgogi beef, kimchi,      |
| > spinach, carrot, and cucumber.                                     |
| >                                                                    |
| > 6\. Sprinkle with sesame seeds and serve immediately.              |
+----------------------------------------------------------------------+

| Carbs: 58g \| Fat: 15g                                               |

| Nut-Free\] \[Peanut-Free\]                                           |
|                                                                      |
| **DIETARY:** \[Low-Sugar / Diabetic-Friendly\]                       |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Iron\] \[High Zinc\]     |
| \[B12-Rich\] \[Vitamin C\] \[High Potassium\]                        |

| **TIP / SWAP:** *Contains soy and sesame. Use coconut aminos for     |
| SoyF; omit sesame oil/seeds for SesF. Kimchi provides probiotics and |
| Vitamin C. Great post-game recovery bowl.*                           |
