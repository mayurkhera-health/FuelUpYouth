+----------------------------------------------------------------------+
| **\#14 Moong Dal Chilla with Mint-Coriander Chutney** \| Indian ·    |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 15 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 cup split yellow moong dal (soaked 4+ hours or overnight, then |
| > drained)                                                           |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 1 green chilli, deseeded (optional)                              |
| >                                                                    |
| > • ¼ cup fresh spinach, finely chopped                              |
| >                                                                    |
| > • ¼ tsp turmeric                                                   |
| >                                                                    |
| > • ¼ tsp cumin seeds                                                |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • 1 tsp oil per chilla for cooking                                 |
| >                                                                    |
| > • ½ cup plain low-fat yogurt + mint chutney to serve               |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Blend soaked moong dal with ginger, chilli, and just enough    |
| > water to make a smooth, pourable batter.                           |
| >                                                                    |
| > 2\. Stir in spinach, turmeric, cumin, and salt.                    |
| >                                                                    |
| > 3\. Heat a non-stick pan over medium; lightly oil. Pour one ladle  |
| > of batter and spread thin (like a crepe).                          |
| >                                                                    |
| > 4\. Cook 3 min until edges lift and bottom is golden. Flip and     |
| > cook 2 more min.                                                   |
| >                                                                    |
| > 5\. Serve hot with yogurt and mint chutney.                        |
