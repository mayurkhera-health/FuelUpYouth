+----------------------------------------------------------------------+
| **\#115 Kimchi Fried Rice with Eggs & Beef (Korean)** \| Korean ·    |
| Post-Game Recovery                                                   |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 15 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 3 cups cooked day-old rice                                       |
| >                                                                    |
| > • 1 cup kimchi, chopped (plus 2 tbsp kimchi brine)                 |
| >                                                                    |
| > • 8 oz lean ground beef                                            |
| >                                                                    |
| > • 4 large eggs (2 scrambled into rice + 2 fried on top)            |
| >                                                                    |
| > • 2 tbsp low-sodium soy sauce (or tamari for GF)                   |
| >                                                                    |
| > • 1 tbsp sesame oil (omit for SesF)                                |
| >                                                                    |
| > • 3 green onions, sliced                                           |
| >                                                                    |
| > • 1 tbsp neutral oil                                               |
| >                                                                    |
| > • 1 tsp gochugaru (Korean red pepper --- optional, for older       |
| > athletes)                                                          |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Brown ground beef in a wok over high heat; drain excess fat.   |
| >                                                                    |
| > 2\. Push beef to side; scramble 2 eggs in the pan.                 |
| >                                                                    |
| > 3\. Add cold rice; stir-fry everything together 3--4 min.          |
| >                                                                    |
| > 4\. Add kimchi, kimchi brine, soy sauce, and sesame oil; toss 2    |
| > min.                                                               |
| >                                                                    |
| > 5\. Serve topped with a fried egg and green onions; kimchi adds    |
| > probiotics for gut recovery.                                       |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 34g \| Calories: 470 kcal \|      |
| Carbs: 44g \| Fat: 18g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Fish-Free\] \[Shellfish-Free\]       |
| \[Tree Nut-Free\] \[Peanut-Free\] \[Gluten-Free\]                    |
|                                                                      |
| **DIETARY:** \[GF\] \[DF\] \[Low-Sugar / Diabetic-Friendly\]         |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Iron\] \[High Zinc\]     |
| \[B12-Rich\] \[Vitamin D\] \[High Potassium\] \[Vitamin C\]          |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains eggs, soy (soy sauce), sesame. Use tamari  |
| for GF; coconut aminos for SoyF; omit sesame for SesF. Kimchi\'s     |
| fermented lactobacillus bacteria support gut microbiome recovery;    |
| beef provides fast-absorbing heme iron and zinc.*                    |
