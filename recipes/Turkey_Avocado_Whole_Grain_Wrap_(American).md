+----------------------------------------------------------------------+
| **\#107 Turkey & Avocado Whole Grain Wrap (American)** \| American · |
| Post-Game Recovery                                                   |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 8 oz sliced turkey breast (low-sodium deli turkey)               |
| >                                                                    |
| > • 2 whole-wheat tortillas (10-inch)                                |
| >                                                                    |
| > • 1 ripe avocado, mashed with lime juice                           |
| >                                                                    |
| > • ½ cup baby spinach                                               |
| >                                                                    |
| > • ½ cup shredded carrots                                           |
| >                                                                    |
| > • ¼ cup sliced cucumber                                            |
| >                                                                    |
| > • 2 tbsp hummus                                                    |
| >                                                                    |
| > • Salt and pepper to taste                                         |
| >                                                                    |
| > • Optional: 1 tbsp pumpkin seeds for extra zinc                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Spread hummus on each tortilla, then layer with avocado mash.  |
| >                                                                    |
| > 2\. Add spinach, carrots, cucumber, and turkey.                    |
| >                                                                    |
| > 3\. Sprinkle pumpkin seeds if using.                               |
| >                                                                    |
| > 4\. Roll tightly and wrap in foil; cut in half for easier eating.  |
| >                                                                    |
| > 5\. Portable post-game option; turkey provides tryptophan which    |
| > aids sleep and recovery.                                           |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 32g \| Calories: 440 kcal \|      |
| Carbs: 42g \| Fat: 16g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Fish-Free\] \[Shellfish-Free\]         |
| \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]                         |
|                                                                      |
| **DIETARY:** \[Low-Sugar / Diabetic-Friendly\]                       |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Potassium\] \[High       |
| Folate\] \[Vitamin C\] \[High Magnesium\] \[High Iron\]              |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains gluten (tortilla), dairy (potential in     |
| hummus). Use GF tortillas for GF; check hummus for sesame (tahini).  |
| Turkey breast is one of the highest protein-to-calorie ratio meats;  |
| avocado provides healthy monounsaturated fats for joint recovery.*   |
