+----------------------------------------------------------------------+
| **\#26 Turkey, Avocado & Veggie Whole Wheat Wrap** \| American ·     |
| Lunch                                                                |
+======================================================================+
| ⏱ Prep: 8 min \| 🍳 Cook: 0 min \| 🍽 Serves: 1                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 large whole wheat tortilla (10-inch)                           |
| >                                                                    |
| > • 3 oz (85 g) sliced oven-roasted turkey breast (low-sodium)       |
| >                                                                    |
| > • ½ ripe avocado, sliced                                           |
| >                                                                    |
| > • 1 large romaine leaf                                             |
| >                                                                    |
| > • 4 slices cucumber                                                |
| >                                                                    |
| > • 4 cherry tomatoes, halved                                        |
| >                                                                    |
| > • 1 tbsp hummus or plain Greek yogurt spread                       |
| >                                                                    |
| > • 1 tsp Dijon mustard                                              |
| >                                                                    |
| > • Salt and black pepper to taste                                   |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Warm tortilla in a dry pan for 15 seconds so it\'s pliable.    |
| >                                                                    |
| > 2\. Spread hummus and mustard across the lower two-thirds of the   |
| > tortilla.                                                          |
| >                                                                    |
| > 3\. Layer romaine, turkey, avocado, cucumber, and tomatoes.        |
| >                                                                    |
| > 4\. Season lightly with salt and pepper.                           |
| >                                                                    |
| > 5\. Fold in the sides, then roll tightly from the bottom. Slice in |
| > half and serve.                                                    |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 28g \| Calories: 430 kcal \|      |
| Carbs: 38g \| Fat: 16g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Shellfish-Free\] \[Fish-Free\] \[Tree  |
| Nut-Free\] \[Peanut-Free\] \[Sesame-Free\]                           |
|                                                                      |
| **DIETARY:** \[Low-Sugar / Diabetic-Friendly\] \[Low-Sodium\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Potassium\] \[Omega-3\]  |
| \[High Folate\] \[Vitamin C\] \[High Iron\]                          |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains wheat, dairy (if using yogurt spread), and |
| soy (some deli meats). Use corn tortilla for GF. Hummus adds sesame  |
| (tahini) --- omit for strict SesF. Great school-packed lunch.*       |
