+----------------------------------------------------------------------+
| **\#37 Saag Paneer with Jeera Rice** \| Indian · Lunch               |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1½ cups baby spinach + ½ cup frozen peas, blended smooth         |
| >                                                                    |
| > • 7 oz (200 g) low-fat paneer, cubed                               |
| >                                                                    |
| > • 1 small onion, diced                                             |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp ginger, grated                                             |
| >                                                                    |
| > • ½ tsp cumin seeds                                                |
| >                                                                    |
| > • ½ tsp turmeric                                                   |
| >                                                                    |
| > • ½ tsp garam masala                                               |
| >                                                                    |
| > • 2 tbsp plain low-fat yogurt                                      |
| >                                                                    |
| > • 1 tbsp avocado oil                                               |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • 1 cup cooked basmati rice (cumin seeds toasted in oil, then rice |
| > cooked)                                                            |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Blend spinach and peas with a splash of water to a coarse      |
| > purée.                                                             |
| >                                                                    |
| > 2\. Heat oil; sauté onion 4 min. Add garlic, ginger, cumin, and    |
| > turmeric; stir 1 min.                                              |
| >                                                                    |
| > 3\. Add spinach purée; cook 5 min. Stir in yogurt and garam        |
| > masala.                                                            |
| >                                                                    |
| > 4\. Add paneer cubes; simmer gently 5 min until heated through.    |
| >                                                                    |
| > 5\. Serve over cumin (jeera) rice.                                 |
