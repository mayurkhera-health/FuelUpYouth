+----------------------------------------------------------------------+
| **\#112 Paneer & Chana Dal Recovery Thali (Indian)** \| Indian ·     |
| Post-Game Recovery                                                   |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 25 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 cup chana dal (split chickpeas), rinsed                        |
| >                                                                    |
| > • 8 oz paneer, cubed and pan-fried golden                          |
| >                                                                    |
| > • 1 can (14 oz) diced tomatoes                                     |
| >                                                                    |
| > • 1 large onion, diced                                             |
| >                                                                    |
| > • 3 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp turmeric, 1 tsp cumin, 1 tsp coriander, ½ tsp garam masala |
| >                                                                    |
| > • 2 tbsp ghee or neutral oil                                       |
| >                                                                    |
| > • 3 cups water or low-sodium vegetable broth                       |
| >                                                                    |
| > • Fresh cilantro and lemon juice to serve                          |
| >                                                                    |
| > • 2 cups cooked basmati rice or whole-wheat roti                   |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Pressure cook or simmer chana dal in broth 20--25 min until    |
| > tender.                                                            |
| >                                                                    |
| > 2\. Sauté onion in ghee until golden; add garlic and spices; cook  |
| > 2 min.                                                             |
| >                                                                    |
| > 3\. Stir in tomatoes and cooked dal; simmer 5 min.                 |
| >                                                                    |
| > 4\. Pan-fry paneer cubes separately until golden; add to dal.      |
| >                                                                    |
| > 5\. Serve over basmati rice with cilantro and lemon juice; pairs   |
| > protein with carbs for recovery.                                   |
