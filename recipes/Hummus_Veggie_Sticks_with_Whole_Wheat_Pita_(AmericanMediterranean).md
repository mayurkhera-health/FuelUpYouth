+----------------------------------------------------------------------+
| **\#118 Hummus & Veggie Sticks with Whole Wheat Pita                 |
| (American/Mediterranean)** \| American · Mid-Day Snacks              |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • ½ cup store-bought or homemade hummus                            |
| >                                                                    |
| > • 1 medium cucumber, cut into sticks                               |
| >                                                                    |
| > • 2 medium carrots, peeled and cut into sticks                     |
| >                                                                    |
| > • 1 red bell pepper, sliced                                        |
| >                                                                    |
| > • 2 whole wheat pita breads, cut into triangles (or rice crackers  |
| > for GF)                                                            |
| >                                                                    |
| > • Optional: ½ tsp za\'atar or smoked paprika on hummus             |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Arrange hummus in a small container or divided bento box.      |
| >                                                                    |
| > 2\. Cut all vegetables into dippable sticks; pack alongside        |
| > hummus.                                                            |
| >                                                                    |
| > 3\. Cut pita into triangles; store in a separate zip bag to keep   |
| > crisp.                                                             |
| >                                                                    |
| > 4\. Pack with small ice pack to keep hummus fresh for up to 4      |
| > hours.                                                             |
| >                                                                    |
| > 5\. The chickpea protein in hummus + fiber in vegetables =         |
| > sustained energy without sugar crashes.                            |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 10g \| Calories: 260 kcal \|      |
| Carbs: 38g \| Fat: 8g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Fish-Free\]             |
| \[Shellfish-Free\] \[Tree Nut-Free\] \[Peanut-Free\] \[Soy-Free\]    |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Vegan\] \[DF\] \[TNF\] \[PF\] \[SoyF\] |
| \[Low-Sugar / Diabetic-Friendly\]                                    |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Iron\] \[High Folate\] \[Vitamin C\]      |
| \[High Potassium\] \[High Magnesium\] \[High Zinc\]                  |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains sesame (tahini in hummus) and gluten       |
| (pita). Use rice crackers or GF pita for GF; use tahini-free hummus  |
| (white bean dip) for SesF. Bell pepper\'s Vitamin C triples the iron |
| absorption from chickpea hummus.*                                    |
