+----------------------------------------------------------------------+
| **\#9 Whole Wheat Banana Oat Pancakes** \| American · Breakfast      |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 15 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 cup whole wheat flour                                          |
| >                                                                    |
| > • ½ cup rolled oats                                                |
| >                                                                    |
| > • 2 large eggs                                                     |
| >                                                                    |
| > • 1 very ripe banana, mashed                                       |
| >                                                                    |
| > • 1 cup whole milk or almond milk                                  |
| >                                                                    |
| > • 1 tsp baking powder                                              |
| >                                                                    |
| > • ½ tsp vanilla extract                                            |
| >                                                                    |
| > • Pinch of salt                                                    |
| >                                                                    |
| > • 2 tbsp natural peanut or sunflower seed butter (for serving)     |
| >                                                                    |
| > • Sliced banana and drizzle of honey (for serving)                 |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Mix flour, oats, baking powder, and salt in a large bowl.      |
| >                                                                    |
| > 2\. In another bowl, whisk eggs, mashed banana, milk, and vanilla. |
| >                                                                    |
| > 3\. Fold wet ingredients into dry until just combined (do not      |
| > overmix).                                                          |
| >                                                                    |
| > 4\. Cook on a lightly greased griddle over medium heat --- 2--3    |
| > min per side until golden.                                         |
| >                                                                    |
| > 5\. Serve topped with nut butter, banana slices, and a honey       |
| > drizzle.                                                           |
