+----------------------------------------------------------------------+
| **\#89 Homemade Sports Drink with Watermelon & Electrolytes** \|     |
| American · Pre-Game Fueling                                          |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 4                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 cups fresh watermelon, cubed                                   |
| >                                                                    |
| > • Juice of 2 limes                                                 |
| >                                                                    |
| > • 2 cups cold water                                                |
| >                                                                    |
| > • ¼ tsp sea salt (natural sodium/electrolyte)                      |
| >                                                                    |
| > • 2 tbsp honey or maple syrup                                      |
| >                                                                    |
| > • Optional: ½ cup coconut water for extra potassium                |
| >                                                                    |
| > • Ice to serve                                                     |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Blend watermelon until smooth; strain through a fine mesh      |
| > sieve if preferred.                                                |
| >                                                                    |
| > 2\. Combine watermelon juice, lime juice, water, salt, and honey   |
| > in a pitcher.                                                      |
| >                                                                    |
| > 3\. Stir well until honey dissolves. Add coconut water if using.   |
| >                                                                    |
| > 4\. Serve over ice.                                                |
| >                                                                    |
| > 5\. Drink 30--60 min before game for hydration + quick carbs. Also |
| > excellent during half-time.                                        |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 1g \| Calories: 80 kcal \| Carbs: |
| 20g \| Fat: 0g                                                       |

| reduce muscle soreness.*                                             |
