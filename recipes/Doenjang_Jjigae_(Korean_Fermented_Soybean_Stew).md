+----------------------------------------------------------------------+
| **\#44 Doenjang Jjigae (Korean Fermented Soybean Stew)** \| Korean · |
| Lunch                                                                |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 tbsp doenjang (Korean fermented soybean paste)                 |
| >                                                                    |
| > • 3 cups low-sodium vegetable or anchovy broth                     |
| >                                                                    |
| > • 1 block (7 oz / 200 g) firm tofu, cubed                          |
| >                                                                    |
| > • 1 small zucchini, diced                                          |
| >                                                                    |
| > • ½ cup mushrooms (shiitake or oyster), sliced                     |
| >                                                                    |
| > • ½ small onion, diced                                             |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 scallion, sliced                                               |
| >                                                                    |
| > • 1 small green chilli (optional)                                  |
| >                                                                    |
| > • 1 cup cooked brown rice per person to serve                      |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Bring broth to a simmer in a clay pot or small saucepan.       |
| >                                                                    |
| > 2\. Whisk in doenjang until dissolved. Add garlic and onion.       |
| >                                                                    |
| > 3\. Add zucchini and mushrooms; simmer 8 min.                      |
| >                                                                    |
| > 4\. Add tofu and chilli; simmer 5 more min.                        |
| >                                                                    |
| > 5\. Top with scallion. Serve immediately alongside steamed brown   |
| > rice.                                                              |
