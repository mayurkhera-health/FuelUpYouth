+----------------------------------------------------------------------+
| **\#48 Pozole Verde with Chicken (Green Hominy Stew)** \| Mexican ·  |
| Lunch                                                                |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 30 min \| 🍽 Serves: 3                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 boneless chicken breasts (about 10 oz / 280 g), poached and    |
| > shredded                                                           |
| >                                                                    |
| > • 1 can (15 oz / 425 g) hominy, rinsed                             |
| >                                                                    |
| > • 4 cups low-sodium chicken broth                                  |
| >                                                                    |
| > • 4 tomatillos, husked and quartered                               |
| >                                                                    |
| > • 1 small onion, quartered                                         |
| >                                                                    |
| > • 2 cloves garlic                                                  |
| >                                                                    |
| > • 1 jalapeño, deseeded                                             |
| >                                                                    |
| > • ½ cup cilantro                                                   |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • Shredded cabbage, sliced radishes, lime wedges to serve          |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Blend tomatillos, onion, garlic, jalapeño, cilantro, and cumin |
| > with ½ cup broth until smooth.                                     |
| >                                                                    |
| > 2\. Heat a pot over medium-high. Add blended green sauce; cook 5   |
| > min until darkened slightly.                                       |
| >                                                                    |
| > 3\. Add remaining broth, hominy, and shredded chicken. Bring to a  |
| > boil.                                                              |
| >                                                                    |
| > 4\. Reduce heat; simmer 20 min. Season with salt.                  |
| >                                                                    |
| > 5\. Serve topped with shredded cabbage, radishes, and lime.        |
