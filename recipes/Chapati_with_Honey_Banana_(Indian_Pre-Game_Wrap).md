+----------------------------------------------------------------------+
| **\#88 Chapati with Honey & Banana (Indian Pre-Game Wrap)** \|       |
| Indian · Pre-Game Fueling                                            |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 5 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 small whole wheat chapati or tortillas (or GF tortilla for     |
| > GF/WF)                                                             |
| >                                                                    |
| > • 2 ripe bananas                                                   |
| >                                                                    |
| > • 2 tbsp honey                                                     |
| >                                                                    |
| > • 1 tbsp almond butter (optional --- omit for TNF/PF; skip if \<1  |
| > hr before game)                                                    |
| >                                                                    |
| > • Pinch of cardamom                                                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Warm chapati in a dry skillet 30 sec per side.                 |
| >                                                                    |
| > 2\. Spread almond butter on each (if using).                       |
| >                                                                    |
| > 3\. Slice banana lengthwise; place on chapati.                     |
| >                                                                    |
| > 4\. Drizzle honey; sprinkle cardamom.                              |
| >                                                                    |
| > 5\. Roll and eat. Best 1--2 hours before game. Skip almond butter  |
| > if within 1 hour --- fat slows digestion.                          |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 8g \| Calories: 300 kcal \|       |
| Carbs: 58g \| Fat: 4g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Fish-Free\]             |
| \[Shellfish-Free\] \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]      |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Carb\] \[High Potassium\] \[High          |
| Magnesium\] \[Vitamin C\] \[High Folate\]                            |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains wheat (chapati) and optionally tree nuts   |
| (almond butter). Use GF tortilla for GF/WF; omit almond butter for   |
| TNF/PF. Honey + banana = quick + medium-release carbs for sustained  |
| competition energy.*                                                 |
