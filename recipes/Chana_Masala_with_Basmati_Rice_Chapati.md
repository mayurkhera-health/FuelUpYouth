+----------------------------------------------------------------------+
| **\#75 Chana Masala with Basmati Rice & Chapati** \| Indian · Dinner |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 25 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cans (15 oz each) chickpeas, drained and rinsed                |
| >                                                                    |
| > • 1½ cups basmati rice, cooked                                     |
| >                                                                    |
| > • 4 store-bought chapati or whole wheat flatbread (or GF flatbread |
| > for GF/WF)                                                         |
| >                                                                    |
| > • 1 can (14 oz) diced tomatoes                                     |
| >                                                                    |
| > • 1 medium onion, finely diced                                     |
| >                                                                    |
| > • 4 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tbsp fresh ginger, grated                                      |
| >                                                                    |
| > • 2 tbsp avocado oil                                               |
| >                                                                    |
| > • 1 tsp cumin seeds                                                |
| >                                                                    |
| > • 1½ tsp coriander powder                                          |
| >                                                                    |
| > • 1 tsp turmeric                                                   |
| >                                                                    |
| > • 1 tsp garam masala                                               |
| >                                                                    |
| > • ½ tsp chili powder                                               |
| >                                                                    |
| > • Juice of ½ lemon                                                 |
| >                                                                    |
| > • Fresh cilantro and sliced onion to serve                         |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil; add cumin seeds and sizzle 30 sec. Add onion; cook   |
| > until golden, \~8 min.                                             |
| >                                                                    |
| > 2\. Add garlic and ginger; cook 1 min. Add all dry spices; toast   |
| > 30 sec.                                                            |
| >                                                                    |
| > 3\. Pour in diced tomatoes; cook 5 min until thickened.            |
| >                                                                    |
| > 4\. Add chickpeas and ½ cup water; simmer 12--15 min until sauce   |
| > is rich and chickpeas are fully flavored.                          |
| >                                                                    |
| > 5\. Squeeze in lemon juice; adjust seasoning.                      |
| >                                                                    |
| > 6\. Serve over basmati rice with chapati for dipping, topped with  |
| > cilantro and fresh onion.                                          |
