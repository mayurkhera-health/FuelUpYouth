+----------------------------------------------------------------------+
| **\#87 Tteokbokki-Style Rice Cakes with Mild Sauce** \| Korean ·     |
| Pre-Game Fueling                                                     |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 10 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 8 oz cylindrical Korean rice cakes (tteok), fresh or thawed      |
| >                                                                    |
| > • 1 cup low-sodium chicken or vegetable broth                      |
| >                                                                    |
| > • 1 tbsp gochujang paste (reduce for lower heat)                   |
| >                                                                    |
| > • 1 tbsp low-sodium soy sauce (or coconut aminos for SoyF)         |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • 1 clove garlic, minced                                           |
| >                                                                    |
| > • 1 green onion, sliced                                            |
| >                                                                    |
| > • Optional: 1 soft-boiled egg (halved)                             |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Combine broth, gochujang, soy sauce, honey, and garlic in a    |
| > skillet; bring to a simmer.                                        |
| >                                                                    |
| > 2\. Add rice cakes; cook over medium heat 5--7 min, stirring       |
| > frequently, until sauce thickens and coats the rice cakes.         |
| >                                                                    |
| > 3\. Plate; top with green onions and egg if using.                 |
| >                                                                    |
| > 4\. Best eaten 2 hours before competition. Rice cakes are made     |
| > from pure rice flour --- naturally GF and quickly digestible.      |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 8g \| Calories: 340 kcal \|       |
| Carbs: 68g \| Fat: 2g                                                |

|                                                                      |

| **TIP / SWAP:** *Contains soy (soy sauce). Use coconut aminos for    |
| SoyF. Tteok (rice cakes) are naturally gluten-free and one of the    |
| fastest-digesting complex carbohydrates available --- ideal pre-game |
| Korean fuel.*                                                        |
