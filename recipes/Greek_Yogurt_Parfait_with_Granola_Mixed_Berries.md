+----------------------------------------------------------------------+
| **\#80 Greek Yogurt Parfait with Granola & Mixed Berries** \|        |
| American · Pre-Game Fueling                                          |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1½ cups plain non-fat Greek yogurt (or coconut yogurt for DF)    |
| >                                                                    |
| > • ¾ cup low-fat granola (certified GF for GF --- look for          |
| > low-fiber variety pre-game)                                        |
| >                                                                    |
| > • 1 cup mixed berries (strawberries, blueberries, raspberries)     |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • Optional: 1 tbsp chia seeds (only if eating 2+ hours before      |
| > game)                                                              |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Layer Greek yogurt in a glass or bowl.                         |
| >                                                                    |
| > 2\. Add half the granola; layer in berries.                        |
| >                                                                    |
| > 3\. Add remaining yogurt, then top with remaining granola and      |
| > berries.                                                           |
| >                                                                    |
| > 4\. Drizzle with honey.                                            |
| >                                                                    |
| > 5\. Best eaten 1.5--2 hours before competition. Skip chia seeds if |
| > eating within 1 hour (adds fiber).                                 |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 22g \| Calories: 380 kcal \|      |
| Carbs: 58g \| Fat: 6g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Fish-Free\] \[Shellfish-Free\]         |
| \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]                         |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Carb\] \[High Calcium\]  |
| \[Vitamin C\] \[High Potassium\] \[High Folate\] \[Vitamin D\]       |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains dairy and may contain wheat (granola) and  |
| tree nuts. Use coconut yogurt for DF; use certified GF granola for   |
| GF; check granola for TNF. Greek yogurt delivers 22g protein with    |
| fast-absorbing carbs from berries and honey.*                        |
