+----------------------------------------------------------------------+
| **\#83 Whole Wheat Toast with Turkey & Avocado** \| American ·       |
| Pre-Game Fueling                                                     |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 3 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 slices whole wheat bread (or GF bread for GF/WF)               |
| >                                                                    |
| > • 6 oz deli turkey breast (low-sodium)                             |
| >                                                                    |
| > • 1 ripe avocado, sliced                                           |
| >                                                                    |
| > • 1 tsp lemon juice                                                |
| >                                                                    |
| > • Salt and pepper to taste                                         |
| >                                                                    |
| > • Optional: 1 slice tomato per toast, fresh spinach                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Toast bread until golden.                                      |
| >                                                                    |
| > 2\. Layer turkey slices on toast.                                  |
| >                                                                    |
| > 3\. Arrange avocado slices on top; squeeze lemon juice over,       |
| > season with salt and pepper.                                       |
| >                                                                    |
| > 4\. Add tomato and spinach if using.                               |
| >                                                                    |
| > 5\. Best eaten 2 hours before game time --- avocado fat slows      |
| > digestion slightly, so don\'t eat too close to competition.        |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 22g \| Calories: 360 kcal \|      |
| Carbs: 38g \| Fat: 14g                                               |

| \[Dairy-Free\]                                                       |
|                                                                      |
| **DIETARY:** \[DF\] \[Low-Sugar / Diabetic-Friendly\]                |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Potassium\] \[High       |
| Folate\] \[Vitamin C\] \[High Magnesium\] \[B12-Rich\]               |

| turkey label for GF and SoyF. Avocado provides healthy               |
| monounsaturated fats and potassium --- electrolyte-supporting        |
| pre-game fuel.*                                                      |
