+----------------------------------------------------------------------+
| **\#17 Poha with Peanuts, Edamame & Lemon** \| Indian · Breakfast    |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 12 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cups thick flattened rice (poha)                               |
| >                                                                    |
| > • ¼ cup roasted peanuts (omit for PF --- use pepitas instead)      |
| >                                                                    |
| > • ¼ cup frozen edamame, thawed                                     |
| >                                                                    |
| > • ½ small onion, finely diced                                      |
| >                                                                    |
| > • 1 small potato, diced small (optional for extra carbs)           |
| >                                                                    |
| > • ½ tsp turmeric                                                   |
| >                                                                    |
| > • ½ tsp mustard seeds                                              |
| >                                                                    |
| > • 8--10 curry leaves                                               |
| >                                                                    |
| > • 2 tbsp avocado or coconut oil                                    |
| >                                                                    |
| > • Juice of 1 lemon                                                 |
| >                                                                    |
| > • Salt to taste, fresh coriander to garnish                        |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Rinse poha in a colander under cold water; shake dry and leave |
| > to fluff up for 5 min.                                             |
| >                                                                    |
| > 2\. Heat oil in a pan; add mustard seeds until they pop. Add curry |
| > leaves, then onion and potato.                                     |
| >                                                                    |
| > 3\. Cook 5--6 min over medium until potato is tender. Add          |
| > turmeric, edamame, and peanuts.                                    |
| >                                                                    |
| > 4\. Add poha and salt; toss gently so it doesn\'t break up. Heat   |
| > through 3 min.                                                     |
| >                                                                    |
| > 5\. Squeeze lemon juice over, scatter coriander, and serve warm.   |
