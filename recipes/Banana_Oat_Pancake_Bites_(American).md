+----------------------------------------------------------------------+
| **\#129 Banana Oat Pancake Bites (American)** \| American · Mid-Day  |
| Snacks                                                               |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 15 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 ripe bananas, mashed                                           |
| >                                                                    |
| > • 2 large eggs                                                     |
| >                                                                    |
| > • 1 cup rolled oats (GF certified for GF)                          |
| >                                                                    |
| > • 2 tbsp natural peanut butter                                     |
| >                                                                    |
| > • 1 tsp cinnamon                                                   |
| >                                                                    |
| > • ½ tsp vanilla extract                                            |
| >                                                                    |
| > • ¼ cup blueberries or mini dark chocolate chips                   |
| >                                                                    |
| > • Pinch of salt                                                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 350°F; grease a mini-muffin tin.               |
| >                                                                    |
| > 2\. Mix all ingredients together until combined.                   |
| >                                                                    |
| > 3\. Fill muffin cups ¾ full.                                       |
| >                                                                    |
| > 4\. Bake 12--15 min until set and slightly golden; cool before     |
| > removing.                                                          |
| >                                                                    |
| > 5\. Makes 24 bites; each bite is \~35 kcal with real food          |
| > ingredients --- perfect for 3-4 bite pre-practice fuel.            |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 10g \| Calories: 240 kcal \|      |
| Carbs: 34g \| Fat: 8g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Fish-Free\] \[Shellfish-Free\]       |
| \[Tree Nut-Free\] \[Gluten-Free\] \[Soy-Free\] \[Sesame-Free\]       |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[DF\] \[SoyF\] \[SesF\] \[Low-Sugar /   |
| Diabetic-Friendly\]                                                  |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Potassium\] \[High       |
| Magnesium\] \[Omega-3\] \[Vitamin D\] \[High Folate\] \[B12-Rich\]   |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains eggs, peanuts (peanut butter), and gluten  |
| (oats --- use GF oats for GF). Use sunflower seed butter for PF.     |
| Banana + oats = slow-release carbohydrates ideal between morning and |
| afternoon training sessions; eggs bind without flour.*               |
