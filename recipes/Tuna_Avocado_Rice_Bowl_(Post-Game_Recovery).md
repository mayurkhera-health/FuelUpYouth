+----------------------------------------------------------------------+
| **\#100 Tuna & Avocado Rice Bowl (Post-Game Recovery)** \| Japanese  |
| · Post-Game Recovery                                                 |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cans (5 oz each) tuna in water, drained                        |
| >                                                                    |
| > • 1½ cups cooked white or brown rice                               |
| >                                                                    |
| > • 1 ripe avocado, sliced                                           |
| >                                                                    |
| > • ½ cucumber, thinly sliced                                        |
| >                                                                    |
| > • 1 cup shredded purple cabbage                                    |
| >                                                                    |
| > • 2 tbsp low-sodium soy sauce (or coconut aminos for SoyF)         |
| >                                                                    |
| > • 1 tbsp rice vinegar                                              |
| >                                                                    |
| > • 1 tsp sesame oil (omit for SesF)                                 |
| >                                                                    |
| > • 1 tsp honey                                                      |
| >                                                                    |
| > • 1 tbsp sesame seeds (omit for SesF)                              |
| >                                                                    |
| > • 2 green onions, sliced                                           |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Mix tuna with half the soy sauce and rice vinegar.             |
| >                                                                    |
| > 2\. Whisk remaining soy sauce, sesame oil, honey, and rice vinegar |
| > into a dressing.                                                   |
| >                                                                    |
| > 3\. Build bowls: rice base, topped with tuna, avocado, cucumber,   |
| > and cabbage.                                                       |
| >                                                                    |
| > 4\. Drizzle dressing over; garnish with sesame seeds and green     |
| > onions.                                                            |
| >                                                                    |
| > 5\. Eat within 45 min of game. Cold ingredients make this ideal    |
| > for hot competition days.                                          |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 38g \| Calories: 460 kcal \|      |
| Carbs: 44g \| Fat: 16g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Shellfish-Free\] \[Tree |
| Nut-Free\] \[Peanut-Free\] \[Gluten-Free\]                           |
|                                                                      |
| **DIETARY:** \[GF\] \[DF\] \[Low-Sugar / Diabetic-Friendly\]         |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[Omega-3\] \[Vitamin D\]       |
| \[High Potassium\] \[Vitamin C\] \[B12-Rich\] \[High Folate\]        |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains fish, soy, and sesame. Use coconut aminos  |
| for SoyF; omit sesame for SesF; use tamari for GF. Tuna + avocado =  |
| protein + anti-inflammatory fats. Rice replenishes glycogen. Quick   |
| no-cook post-game option.*                                           |
