+----------------------------------------------------------------------+
| **\#45 Rajma (Kidney Bean Curry) with Rice** \| Indian · Lunch       |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 25 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 can (15 oz / 425 g) red kidney beans, rinsed                   |
| >                                                                    |
| > • 1 can (14 oz / 400 g) diced tomatoes                             |
| >                                                                    |
| > • 1 small onion, finely diced                                      |
| >                                                                    |
| > • 3 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp ginger, grated                                             |
| >                                                                    |
| > • 1 tsp cumin seeds                                                |
| >                                                                    |
| > • 1 tsp coriander powder                                           |
| >                                                                    |
| > • ½ tsp turmeric                                                   |
| >                                                                    |
| > • ½ tsp garam masala                                               |
| >                                                                    |
| > • ¼ tsp cayenne (optional)                                         |
| >                                                                    |
| > • 1 tbsp avocado or coconut oil                                    |
| >                                                                    |
| > • Salt to taste, fresh coriander and lime to serve                 |
| >                                                                    |
| > • 1½ cups cooked basmati rice                                      |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil in a pot. Add cumin seeds; once they sizzle, add      |
| > onion and cook 5 min.                                              |
| >                                                                    |
| > 2\. Add garlic, ginger, coriander, turmeric, and cayenne; stir 1   |
| > min.                                                               |
| >                                                                    |
| > 3\. Add tomatoes; simmer 8 min until sauce thickens. Add kidney    |
| > beans.                                                             |
| >                                                                    |
| > 4\. Simmer 10 min, mashing a few beans for a thicker gravy. Add    |
| > garam masala.                                                      |
| >                                                                    |
| > 5\. Serve over rice with coriander and a squeeze of lime.          |
