+----------------------------------------------------------------------+
| **\#70 Paneer Tikka with Basmati Rice & Cucumber Raita** \| Indian · |
| Dinner                                                               |
+======================================================================+
| ⏱ Prep: 20 min \| 🍳 Cook: 20 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 14 oz paneer (Indian cheese), cubed                              |
| >                                                                    |
| > • 1½ cups basmati rice, cooked                                     |
| >                                                                    |
| > • 1 red bell pepper, cubed                                         |
| >                                                                    |
| > • 1 green bell pepper, cubed                                       |
| >                                                                    |
| > • 1 medium red onion, cut into chunks                              |
| >                                                                    |
| > • 2 tbsp avocado oil                                               |
| >                                                                    |
| > • Marinade: ½ cup plain yogurt                                     |
| >                                                                    |
| > • 1 tbsp lemon juice                                               |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp ginger, grated                                             |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • 1 tsp coriander                                                  |
| >                                                                    |
| > • 1 tsp garam masala                                               |
| >                                                                    |
| > • ½ tsp turmeric                                                   |
| >                                                                    |
| > • ½ tsp chili powder                                               |
| >                                                                    |
| > • Raita: 1 cup plain Greek yogurt                                  |
| >                                                                    |
| > • ½ cucumber, grated and squeezed dry                              |
| >                                                                    |
| > • 1 tsp cumin, pinch of salt, fresh mint                           |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Combine all marinade ingredients; fold in paneer cubes and     |
| > vegetables. Marinate 30 min minimum.                               |
| >                                                                    |
| > 2\. Make raita: mix Greek yogurt with grated cucumber, cumin,      |
| > salt, and mint. Refrigerate.                                       |
| >                                                                    |
| > 3\. Preheat oven to 425°F or heat a grill pan over high heat.      |
| >                                                                    |
| > 4\. Arrange paneer and vegetables on a sheet pan; drizzle with     |
| > avocado oil.                                                       |
| >                                                                    |
| > 5\. Roast 18--20 min until paneer is golden and vegetables are     |
| > charred at edges. Or thread onto skewers and grill 3--4 min per    |
| > side.                                                              |
| >                                                                    |
| > 6\. Serve over basmati rice with a generous spoonful of raita      |
| > alongside.                                                         |
