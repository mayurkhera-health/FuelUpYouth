+----------------------------------------------------------------------+
| **\#12 Tuna & Brown Rice Onigiri (Rice Balls)** \| Japanese ·        |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 25 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cups cooked short-grain brown rice (warm)                      |
| >                                                                    |
| > • 1 can (5 oz / 142 g) water-packed tuna, drained                  |
| >                                                                    |
| > • 1½ tbsp plain Greek yogurt or light mayo                         |
| >                                                                    |
| > • 1 tsp low-sodium soy sauce or tamari                             |
| >                                                                    |
| > • 2 sheets nori (dried seaweed), cut in half                       |
| >                                                                    |
| > • ¼ tsp sesame seeds (optional --- omit for SesF)                  |
| >                                                                    |
| > • Pinch of sea salt                                                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Mix tuna with Greek yogurt/mayo and soy sauce to make filling. |
| >                                                                    |
| > 2\. Wet hands lightly, sprinkle with a pinch of salt. Take ¾ cup   |
| > warm rice and flatten in palm.                                     |
| >                                                                    |
| > 3\. Place 1--2 tbsp tuna filling in the centre; fold rice around   |
| > it and press firmly into a triangle or ball.                       |
| >                                                                    |
| > 4\. Wrap base with a strip of nori. Sprinkle sesame seeds on top   |
| > if using.                                                          |
| >                                                                    |
| > 5\. Repeat to make 4 onigiri. Pack in an airtight container if     |
| > eating later.                                                      |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 26g \| Calories: 410 kcal \|      |
| Carbs: 58g \| Fat: 5g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Shellfish-Free\] \[Tree |
| Nut-Free\] \[Peanut-Free\]                                           |
|                                                                      |
| **DIETARY:** \[Low-Sodium\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Carb\] \[Omega-3\]       |
| \[B12-Rich\] \[Vitamin D\] \[High Iron\]                             |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Use tamari to make GF & SoyF. Omit sesame seeds for |
| SesF. Swap tuna for mashed edamame for a vegan version. Great        |
| pre-packed school breakfast.*                                        |
