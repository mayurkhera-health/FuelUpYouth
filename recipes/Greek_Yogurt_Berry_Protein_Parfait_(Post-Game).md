+----------------------------------------------------------------------+
| **\#99 Greek Yogurt & Berry Protein Parfait (Post-Game)** \|         |
| American · Post-Game Recovery                                        |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cups plain full-fat Greek yogurt (or high-protein coconut      |
| > yogurt for DF)                                                     |
| >                                                                    |
| > • 1 cup mixed berries (blueberries, strawberries, raspberries)     |
| >                                                                    |
| > • ¼ cup granola (certified GF for GF)                              |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • 1 tbsp chia seeds (Omega-3 and fiber for recovery)               |
| >                                                                    |
| > • Optional: 1 tbsp hemp seeds for extra protein                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Layer yogurt in a glass or bowl.                               |
| >                                                                    |
| > 2\. Add berries and granola.                                       |
| >                                                                    |
| > 3\. Top with chia seeds, honey, and hemp seeds if using.           |
| >                                                                    |
| > 4\. Eat within 30--45 min of finishing competition.                |
| >                                                                    |
| > 5\. The protein in Greek yogurt (20g per cup) initiates muscle     |
| > protein synthesis; berries reduce oxidative stress.                |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 24g \| Calories: 360 kcal \|      |
| Carbs: 44g \| Fat: 10g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Fish-Free\] \[Shellfish-Free\]         |
| \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]                         |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Calcium\] \[Vitamin C\]  |
| \[Vitamin D\] \[High Potassium\] \[Omega-3\] \[High Folate\]         |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains dairy, tree nuts (granola), and wheat      |
| (granola). Use coconut yogurt for DF; use GF certified granola for   |
| GF; check granola for TNF. Greek yogurt is one of the highest        |
| natural protein-per-calorie foods available.*                        |
