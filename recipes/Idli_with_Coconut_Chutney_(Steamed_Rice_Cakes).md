+----------------------------------------------------------------------+
| **\#94 Idli with Coconut Chutney (Steamed Rice Cakes)** \| Indian ·  |
| Pre-Game Fueling                                                     |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 15 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 12 store-bought or homemade idli (fermented rice & lentil cakes  |
| > --- naturally GF)                                                  |
| >                                                                    |
| > • Coconut Chutney: ½ cup shredded coconut (fresh or frozen)        |
| >                                                                    |
| > • 2 tbsp roasted chana dal (split chickpeas)                       |
| >                                                                    |
| > • 1 green chili (seeded for mild)                                  |
| >                                                                    |
| > • ½ inch fresh ginger                                              |
| >                                                                    |
| > • Juice of ½ lemon                                                 |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • Water to blend                                                   |
| >                                                                    |
| > • Tempering: 1 tsp avocado oil, ½ tsp mustard seeds, 8 curry       |
| > leaves                                                             |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Steam idli per package instructions (typically 10--12 min      |
| > until a toothpick comes out clean).                                |
| >                                                                    |
| > 2\. Blend coconut, chana dal, chili, ginger, lemon juice, salt,    |
| > and water into a smooth chutney.                                   |
| >                                                                    |
| > 3\. For tempering: heat oil in a small pan; add mustard seeds      |
| > until they pop. Add curry leaves; pour over chutney.               |
| >                                                                    |
| > 4\. Serve idli warm with coconut chutney on the side.              |
| >                                                                    |
| > 5\. Best eaten 1.5--2 hours before competition. Idli are one of    |
| > the most digestible pre-game foods --- light, steamed, and made    |
| > from fermented batter.                                             |
