+----------------------------------------------------------------------+
| **\#110 Cottage Cheese & Pineapple Recovery Bowl (American)** \|     |
| American · Post-Game Recovery                                        |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cups full-fat cottage cheese (or low-fat for lighter option)   |
| >                                                                    |
| > • 1 cup fresh pineapple chunks (or canned in juice, drained)       |
| >                                                                    |
| > • ½ cup sliced strawberries                                        |
| >                                                                    |
| > • ¼ cup granola (GF certified for GF)                              |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • 1 tsp vanilla extract                                            |
| >                                                                    |
| > • Optional: 2 tbsp sliced almonds for healthy fat                  |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Divide cottage cheese into two bowls.                          |
| >                                                                    |
| > 2\. Top with pineapple, strawberries, and granola.                 |
| >                                                                    |
| > 3\. Drizzle honey; add vanilla and almonds if using.               |
| >                                                                    |
| > 4\. Eat within 30 min of game; cottage cheese is 70% casein ---    |
| > the slow-digesting protein that rebuilds muscle overnight.         |
| >                                                                    |
| > 5\. Pineapple contains bromelain, a natural enzyme that reduces    |
| > sports-related muscle soreness.                                    |

| **NUTRITION PER SERVING** Protein: 28g \| Calories: 380 kcal \|      |
| Carbs: 46g \| Fat: 10g                                               |

|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Calcium\] \[Vitamin C\]  |
| \[High Potassium\] \[B12-Rich\] \[Vitamin D\] \[High Zinc\]          |

| **TIP / SWAP:** *Contains dairy, gluten (granola), tree nuts         |
| (almonds --- optional). Use GF granola for GF; omit almonds for TNF. |
| Cottage cheese is among the richest sources of casein protein ---    |
| ideal for overnight muscle repair when consumed post-evening game.*  |
