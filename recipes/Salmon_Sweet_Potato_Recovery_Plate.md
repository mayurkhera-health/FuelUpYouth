+----------------------------------------------------------------------+
| **\#98 Salmon & Sweet Potato Recovery Plate** \| American ·          |
| Post-Game Recovery                                                   |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 25 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 salmon fillets (5 oz each)                                     |
| >                                                                    |
| > • 2 large sweet potatoes, cubed and roasted                        |
| >                                                                    |
| > • 2 cups steamed or roasted broccolini                             |
| >                                                                    |
| > • 2 tbsp olive oil                                                 |
| >                                                                    |
| > • 1 tsp garlic powder                                              |
| >                                                                    |
| > • 1 tsp smoked paprika                                             |
| >                                                                    |
| > • Juice of 1 lemon                                                 |
| >                                                                    |
| > • 2 tbsp fresh dill or parsley                                     |
| >                                                                    |
| > • Salt and pepper                                                  |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 400°F. Toss sweet potato cubes with 1 tbsp     |
| > oil, salt, and pepper; roast 20--25 min.                           |
| >                                                                    |
| > 2\. Season salmon with garlic powder, smoked paprika, salt, and    |
| > pepper.                                                            |
| >                                                                    |
| > 3\. Heat remaining oil in a skillet over medium-high; cook salmon  |
| > 4 min per side until golden and cooked through.                    |
| >                                                                    |
| > 4\. Steam or roast broccolini 8--10 min until tender.              |
| >                                                                    |
| > 5\. Plate salmon alongside sweet potato and broccolini; finish     |
| > with lemon juice and fresh herbs.                                  |
