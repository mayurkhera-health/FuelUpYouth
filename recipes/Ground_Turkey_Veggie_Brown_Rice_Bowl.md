+----------------------------------------------------------------------+
| **\#32 Ground Turkey & Veggie Brown Rice Bowl** \| American · Lunch  |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 20 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 8 oz (225 g) lean ground turkey                                  |
| >                                                                    |
| > • 1½ cups cooked brown rice                                        |
| >                                                                    |
| > • 1 cup broccoli florets, steamed                                  |
| >                                                                    |
| > • 1 medium carrot, diced and steamed                               |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tbsp low-sodium soy sauce or tamari                            |
| >                                                                    |
| > • 1 tsp olive oil                                                  |
| >                                                                    |
| > • ½ tsp smoked paprika                                             |
| >                                                                    |
| > • Salt and pepper to taste                                         |
| >                                                                    |
| > • Lime wedge and fresh herbs to serve                              |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil in a skillet. Brown ground turkey over medium-high,   |
| > breaking it apart, 8--10 min.                                      |
| >                                                                    |
| > 2\. Add garlic and paprika; stir 1 min. Drizzle soy sauce over and |
| > toss.                                                              |
| >                                                                    |
| > 3\. Steam broccoli and carrot until tender-crisp, about 5--6 min.  |
| >                                                                    |
| > 4\. Divide rice between bowls. Top with turkey and vegetables.     |
| >                                                                    |
| > 5\. Squeeze lime over and garnish with fresh herbs.                |
+----------------------------------------------------------------------+
