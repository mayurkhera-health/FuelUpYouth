+----------------------------------------------------------------------+
| **\#71 One-Pan Lemon Herb Chicken Thighs with Root Vegetables** \|   |
| American · Dinner                                                    |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 35 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs bone-in, skin-on chicken thighs                          |
| >                                                                    |
| > • 2 medium sweet potatoes, cubed                                   |
| >                                                                    |
| > • 2 large carrots, cut into chunks                                 |
| >                                                                    |
| > • 1 cup green beans, trimmed                                       |
| >                                                                    |
| > • 4 cloves garlic, smashed                                         |
| >                                                                    |
| > • 2 tbsp olive oil                                                 |
| >                                                                    |
| > • Juice and zest of 1 lemon                                        |
| >                                                                    |
| > • 1 tsp dried thyme                                                |
| >                                                                    |
| > • 1 tsp dried rosemary                                             |
| >                                                                    |
| > • ½ tsp garlic powder                                              |
| >                                                                    |
| > • Salt and pepper to taste                                         |
| >                                                                    |
| > • Fresh parsley to garnish                                         |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 425°F. Toss sweet potatoes and carrots with 1  |
| > tbsp olive oil, salt, and pepper on a sheet pan.                   |
| >                                                                    |
| > 2\. Rub chicken thighs with remaining oil, lemon zest, thyme,      |
| > rosemary, garlic powder, salt, and pepper.                         |
| >                                                                    |
| > 3\. Nestle chicken thighs among the vegetables, skin-side-up.      |
| > Scatter smashed garlic across pan.                                 |
| >                                                                    |
| > 4\. Roast 30 min. Add green beans; roast 5--7 more min until       |
| > chicken skin is golden and internal temp 165°F.                    |
| >                                                                    |
| > 5\. Squeeze lemon juice over everything; garnish with parsley and  |
| > serve directly from the pan.                                       |
