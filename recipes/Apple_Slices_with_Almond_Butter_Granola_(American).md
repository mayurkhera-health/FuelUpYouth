+----------------------------------------------------------------------+
| **\#116 Apple Slices with Almond Butter & Granola (American)** \|    |
| American · Mid-Day Snacks                                            |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 medium apples, cored and sliced                                |
| >                                                                    |
| > • 4 tbsp natural almond butter (no added sugar)                    |
| >                                                                    |
| > • ¼ cup granola (GF certified for GF)                              |
| >                                                                    |
| > • 1 tsp honey (optional)                                           |
| >                                                                    |
| > • Pinch of cinnamon                                                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Core and slice apples into wedges.                             |
| >                                                                    |
| > 2\. Divide almond butter into two dipping cups or spread on apple  |
| > slices.                                                            |
| >                                                                    |
| > 3\. Sprinkle granola and cinnamon over the top.                    |
| >                                                                    |
| > 4\. Pack in a bento box with a small ice pack for portability.     |
| >                                                                    |
| > 5\. The fiber in apples + protein in almond butter provides 2--3   |
| > hours of sustained energy.                                         |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 10g \| Calories: 290 kcal \|      |
| Carbs: 36g \| Fat: 14g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Fish-Free\]             |
| \[Shellfish-Free\] \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]      |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Vegan\] \[DF\] \[PF\] \[SoyF\]         |
| \[SesF\] \[Low-Sugar / Diabetic-Friendly\]                           |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Potassium\] \[Vitamin C\] \[High          |
| Magnesium\] \[Omega-3\] \[High Calcium\] \[High Iron\]               |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains tree nuts (almonds) and gluten (granola).  |
| Use GF granola for GF; use sunflower seed butter for TNF. Almond     |
| butter is one of the highest calcium nut butters --- 2 tbsp provides |
| 10% daily calcium. Apple pectin fiber feeds gut bacteria.*           |
