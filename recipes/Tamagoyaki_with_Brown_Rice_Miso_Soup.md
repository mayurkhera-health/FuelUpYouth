+----------------------------------------------------------------------+
| **\#11 Tamagoyaki with Brown Rice & Miso Soup** \| Japanese ·        |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 15 min \| 🍽 Serves: 1                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 3 large eggs                                                     |
| >                                                                    |
| > • 1 tsp low-sodium soy sauce or tamari                             |
| >                                                                    |
| > • 1 tsp mirin (or ½ tsp honey)                                     |
| >                                                                    |
| > • 1 cup cooked short-grain brown rice                              |
| >                                                                    |
| > • 1 cup hot water + 1 tbsp white miso paste                        |
| >                                                                    |
| > • 1 scallion, finely sliced                                        |
| >                                                                    |
| > • 1 tsp neutral oil (avocado or light sesame)                      |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Whisk eggs with soy sauce and mirin until fully blended.       |
| >                                                                    |
| > 2\. Heat a non-stick pan over medium, lightly oil it. Pour         |
| > one-third of the egg mixture; let it partially set.                |
| >                                                                    |
| > 3\. Roll from one side using chopsticks or a spatula. Push roll to |
| > one end; add another third of egg, lift roll so new egg flows      |
| > underneath. Roll again. Repeat with final third.                   |
| >                                                                    |
| > 4\. Remove, rest 1 min, then slice into 4 rounds.                  |
| >                                                                    |
| > 5\. Stir miso into hot water for soup. Serve tamagoyaki alongside  |
| > brown rice, miso soup, and scallion.                               |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 22g \| Calories: 370 kcal \|      |
| Carbs: 44g \| Fat: 10g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Shellfish-Free\] \[Fish-Free\]       |
| \[Tree Nut-Free\] \[Peanut-Free\]                                    |
|                                                                      |
| **DIETARY:** \[Low-Sugar / Diabetic-Friendly\] \[Low-Sodium\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[Vitamin D\] \[B12-Rich\]      |
| \[High Iron\] \[High Magnesium\]                                     |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Use tamari to make GF & SoyF. Skip miso and use     |
| low-sodium broth for SoyF soup alternative. The egg roll keeps in    |
| the fridge up to 2 days --- slice fresh.*                            |
