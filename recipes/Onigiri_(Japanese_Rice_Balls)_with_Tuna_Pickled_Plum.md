+----------------------------------------------------------------------+
| **\#91 Onigiri (Japanese Rice Balls) with Tuna & Pickled Plum** \|   |
| Japanese · Pre-Game Fueling                                          |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 0 min \| 🍽 Serves: 4                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cups short-grain white rice, cooked and slightly warm          |
| >                                                                    |
| > • 1 can (5 oz) tuna in water, drained and mixed with 1 tsp soy     |
| > sauce                                                              |
| >                                                                    |
| > • 2 umeboshi (pickled plums), pitted and chopped (or use a pinch   |
| > of salt as substitute)                                             |
| >                                                                    |
| > • 4 sheets nori (seaweed), cut into strips (omit for FF/SesF)      |
| >                                                                    |
| > • 1 tsp sesame seeds (omit for SesF)                               |
| >                                                                    |
| > • Salt to taste                                                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Wet your hands and sprinkle lightly with salt.                 |
| >                                                                    |
| > 2\. Take ½ cup warm rice; flatten in your palm. Place 1 tbsp tuna  |
| > or a piece of umeboshi in the center.                              |
| >                                                                    |
| > 3\. Fold rice around the filling; press firmly into a triangle or  |
| > ball shape.                                                        |
| >                                                                    |
| > 4\. Wrap with a strip of nori if using; sprinkle with sesame       |
| > seeds.                                                             |
| >                                                                    |
| > 5\. Wrap individually in plastic wrap for easy transport. Best     |
| > eaten 1--2 hours before game time.                                 |
+----------------------------------------------------------------------+

| Carbs: 56g \| Fat: 2g                                                |

| **TIP / SWAP:** *Contains fish (tuna) and sesame. Omit nori for FF.  |
| Use coconut aminos for SoyF. Onigiri are the original pre-game       |
| sports food in Japan --- portable, no utensils needed, high carb,    |
| easy to digest.*                                                     |
