+----------------------------------------------------------------------+
| **\#16 Paneer & Spinach Paratha with Plain Yogurt** \| Indian ·      |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 15 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1½ cups whole wheat flour (atta), plus extra for rolling         |
| >                                                                    |
| > • Water to knead (approx. ½ cup)                                   |
| >                                                                    |
| > • 1 cup crumbled low-fat paneer (Indian cottage cheese)            |
| >                                                                    |
| > • ½ cup baby spinach, finely chopped                               |
| >                                                                    |
| > • ½ tsp cumin powder                                               |
| >                                                                    |
| > • ½ tsp garam masala                                               |
| >                                                                    |
| > • ¼ tsp turmeric                                                   |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • 1 tsp ghee or olive oil per paratha                              |
| >                                                                    |
| > • 1 cup plain low-fat yogurt to serve                              |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Knead flour with water and a pinch of salt into a soft, smooth |
| > dough. Cover and rest 10 min.                                      |
| >                                                                    |
| > 2\. Mix paneer, spinach, cumin, garam masala, turmeric, and salt   |
| > for the filling.                                                   |
| >                                                                    |
| > 3\. Divide dough into 4 balls. Flatten one, place filling in       |
| > centre, fold edges in, seal, and roll thin.                        |
| >                                                                    |
| > 4\. Cook on a hot dry griddle 2--3 min per side until golden.      |
| > Brush with a little ghee.                                          |
| >                                                                    |
| > 5\. Serve with plain yogurt.                                       |
