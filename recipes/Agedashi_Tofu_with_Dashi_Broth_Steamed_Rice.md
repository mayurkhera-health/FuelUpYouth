+----------------------------------------------------------------------+
| **\#72 Agedashi Tofu with Dashi Broth & Steamed Rice** \| Japanese · |
| Dinner                                                               |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 15 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 14 oz extra-firm tofu, pressed and cubed                         |
| >                                                                    |
| > • 2 cups jasmine rice, cooked                                      |
| >                                                                    |
| > • ¼ cup potato starch or cornstarch                                |
| >                                                                    |
| > • 2 cups dashi broth (or vegetable broth for vegan)                |
| >                                                                    |
| > • 3 tbsp low-sodium soy sauce (or tamari for GF)                   |
| >                                                                    |
| > • 2 tbsp mirin                                                     |
| >                                                                    |
| > • 1 tsp sugar                                                      |
| >                                                                    |
| > • 2 cups avocado oil or neutral oil for frying                     |
| >                                                                    |
| > • 2 green onions, thinly sliced                                    |
| >                                                                    |
| > • 1 tsp fresh ginger, finely grated                                |
| >                                                                    |
| > • 1 tsp sesame seeds (omit for SesF)                               |
| >                                                                    |
| > • Optional: bonito flakes (omit for FF/vegan)                      |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Press tofu 15 min; cut into 2-inch cubes. Pat completely dry.  |
| >                                                                    |
| > 2\. Combine dashi, soy sauce, mirin, and sugar in a saucepan;      |
| > bring to a simmer. Keep warm.                                      |
| >                                                                    |
| > 3\. Coat tofu cubes lightly in potato starch, shaking off excess.  |
| >                                                                    |
| > 4\. Heat 1 inch of oil in a deep skillet to 350°F. Fry tofu in     |
| > batches 3--4 min per side until lightly golden and slightly crisp. |
| >                                                                    |
| > 5\. Drain tofu on paper towels; arrange in bowls.                  |
| >                                                                    |
| > 6\. Pour hot dashi broth around (not over) the tofu to keep it     |
| > crisp.                                                             |
| >                                                                    |
| > 7\. Top with green onions, ginger, sesame seeds, and bonito flakes |
| > if using. Serve with rice.                                         |
