+----------------------------------------------------------------------+
| **\#20 Chilaquiles with Salsa Verde & Fried Egg** \| Mexican ·       |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 15 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cups baked corn tortilla chips (low-sodium)                    |
| >                                                                    |
| > • 1 cup store-bought or homemade salsa verde                       |
| >                                                                    |
| > • 4 large eggs                                                     |
| >                                                                    |
| > • ¼ cup crumbled queso fresco or feta                              |
| >                                                                    |
| > • ½ cup canned black beans, rinsed and warmed                      |
| >                                                                    |
| > • ½ ripe avocado, sliced                                           |
| >                                                                    |
| > • 2 tbsp plain Greek yogurt                                        |
| >                                                                    |
| > • Fresh cilantro, lime wedge                                       |
| >                                                                    |
| > • 1 tsp olive oil                                                  |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Warm salsa verde in a large skillet over medium heat. Add      |
| > chips and toss gently until coated and slightly softened, 2--3     |
| > min.                                                               |
| >                                                                    |
| > 2\. Push chips to edges; add oil to the centre and fry eggs to     |
| > your liking (sunny-side or over-easy).                             |
| >                                                                    |
| > 3\. Divide chips between two plates; top each with 2 eggs.         |
| >                                                                    |
| > 4\. Add warm beans, avocado, crumbled cheese, and a dollop of      |
| > Greek yogurt.                                                      |
| >                                                                    |
| > 5\. Scatter cilantro and squeeze lime over. Serve right away.      |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 24g \| Calories: 470 kcal \|      |
| Carbs: 44g \| Fat: 22g                                               |

|                                                                      |

| C\] \[High Folate\] \[High Iron\] \[Omega-3\] \[High Calcium\]       |

| **TIP / SWAP:** *GF if chips are 100% corn. Contains eggs and dairy. |

| anti-inflammatory breakfasts in the database.*                       |
