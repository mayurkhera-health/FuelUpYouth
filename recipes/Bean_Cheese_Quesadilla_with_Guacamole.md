+----------------------------------------------------------------------+
| **\#39 Bean & Cheese Quesadilla with Guacamole** \| Mexican · Lunch  |
+======================================================================+
| ⏱ Prep: 8 min \| 🍳 Cook: 10 min \| 🍽 Serves: 1                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 whole wheat flour tortillas (8-inch)                           |
| >                                                                    |
| > • ½ cup canned refried beans (low-sodium) or mashed black beans    |
| >                                                                    |
| > • ½ cup shredded Monterey Jack or pepper jack cheese               |
| >                                                                    |
| > • ¼ cup corn kernels                                               |
| >                                                                    |
| > • 1 ripe avocado, mashed with lime and a pinch of salt             |
| >                                                                    |
| > • 2 tbsp low-sodium salsa                                          |
| >                                                                    |
| > • Fresh cilantro to garnish                                        |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Spread refried beans on one tortilla. Sprinkle cheese and corn |
| > over.                                                              |
| >                                                                    |
| > 2\. Top with the second tortilla. Cook in a dry skillet over       |
| > medium heat 2--3 min per side until golden and cheese melts.       |
| >                                                                    |
| > 3\. Slice into wedges.                                             |
| >                                                                    |
| > 4\. Serve with mashed avocado (guacamole) and fresh salsa.         |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 22g \| Calories: 510 kcal \|      |
| Carbs: 54g \| Fat: 22g                                               |

| dairy-free cheese for GF & DF version. Beans and avocado together    |
| deliver exceptional folate, potassium, and healthy fats for youth    |
| athletes.*                                                           |
