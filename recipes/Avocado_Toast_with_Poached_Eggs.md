+----------------------------------------------------------------------+
| **\#5 Avocado Toast with Poached Eggs** \| American · Breakfast      |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 8 min \| 🍽 Serves: 1                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 slices whole wheat or sourdough bread                          |
| >                                                                    |
| > • 1 ripe avocado                                                   |
| >                                                                    |
| > • 2 large eggs                                                     |
| >                                                                    |
| > • 1 tbsp fresh lemon juice                                         |
| >                                                                    |
| > • ¼ tsp red chili flakes (optional)                                |
| >                                                                    |
| > • Salt, black pepper, everything bagel seasoning (optional)        |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Toast bread until golden. Mash avocado with lemon juice, salt, |
| > and pepper.                                                        |
| >                                                                    |
| > 2\. Bring a small saucepan of water to a gentle simmer; add a      |
| > splash of white vinegar.                                           |
| >                                                                    |
| > 3\. Crack each egg into a small cup; slide gently into simmering   |
| > water. Poach 3--4 min.                                             |
| >                                                                    |
| > 4\. Spread avocado on toast, top with poached eggs, season with    |
| > chili flakes and bagel seasoning.                                  |
