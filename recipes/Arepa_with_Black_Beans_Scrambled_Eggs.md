+----------------------------------------------------------------------+
| **\#92 Arepa with Black Beans & Scrambled Eggs** \| Mexican ·        |
| Pre-Game Fueling                                                     |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 15 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cups masarepa (pre-cooked white cornmeal --- naturally GF)     |
| >                                                                    |
| > • 2 cups warm water                                                |
| >                                                                    |
| > • ½ tsp salt                                                       |
| >                                                                    |
| > • 1 can (15 oz) black beans, drained, warmed with a pinch of cumin |
| >                                                                    |
| > • 4 eggs, scrambled (omit for EF --- use extra beans)              |
| >                                                                    |
| > • 1 tsp olive oil                                                  |
| >                                                                    |
| > • Optional: sliced avocado, salsa                                  |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Mix masarepa, water, and salt; let rest 5 min until dough      |
| > forms.                                                             |
| >                                                                    |
| > 2\. Shape into 8 flat patties (\~½ inch thick, 4 inches wide).     |
| >                                                                    |
| > 3\. Cook in a lightly oiled skillet over medium heat 5--6 min per  |
| > side until golden crust forms.                                     |
| >                                                                    |
| > 4\. Scramble eggs in the same skillet with 1 tsp oil; season with  |
| > salt and pepper.                                                   |
| >                                                                    |
| > 5\. Split arepas open; fill with warm black beans and scrambled    |
| > eggs. Add avocado and salsa if using.                              |
