+----------------------------------------------------------------------+
| **\#63 Chicken Tikka Masala with Garlic Naan** \| Indian · Dinner    |
+======================================================================+
| ⏱ Prep: 20 min \| 🍳 Cook: 30 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs boneless chicken breast, cubed                           |
| >                                                                    |
| > • 4 pieces store-bought naan (or GF flatbread for GF/WF)           |
| >                                                                    |
| > • 1 can (14 oz) crushed tomatoes                                   |
| >                                                                    |
| > • ½ cup full-fat coconut milk                                      |
| >                                                                    |
| > • 1 medium onion, diced                                            |
| >                                                                    |
| > • 4 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tbsp fresh ginger, grated                                      |
| >                                                                    |
| > • 2 tbsp ghee or avocado oil                                       |
| >                                                                    |
| > • 2 tsp garam masala                                               |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • 1 tsp turmeric                                                   |
| >                                                                    |
| > • 1 tsp coriander                                                  |
| >                                                                    |
| > • ½ tsp chili powder                                               |
| >                                                                    |
| > • ½ cup plain yogurt (or coconut yogurt for DF)                    |
| >                                                                    |
| > • Juice of ½ lemon                                                 |
| >                                                                    |
| > • Fresh cilantro to garnish                                        |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Marinate chicken in yogurt, lemon juice, 1 tsp garam masala,   |
| > and half the garlic and ginger for 30 min (or overnight).          |
| >                                                                    |
| > 2\. Heat ghee in a large pan; cook marinated chicken in batches    |
| > until charred on edges, \~5 min. Remove.                           |
| >                                                                    |
| > 3\. In same pan sauté onion until golden. Add remaining garlic and |
| > ginger; cook 1 min.                                                |
| >                                                                    |
| > 4\. Add all spices; toast 30 sec. Pour in crushed tomatoes; simmer |
| > 8--10 min.                                                         |
| >                                                                    |
| > 5\. Stir in coconut milk; simmer 5 min. Return chicken; cook 5     |
| > more min.                                                          |
| >                                                                    |
| > 6\. Warm naan in a dry skillet. Serve curry over or alongside      |
| > naan, garnished with cilantro.                                     |
