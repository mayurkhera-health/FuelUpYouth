+----------------------------------------------------------------------+
| **\#41 Grilled Chicken Caesar Salad (No Croutons)** \| American ·    |
| Lunch                                                                |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 12 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 boneless chicken breasts (about 10 oz / 280 g)                 |
| >                                                                    |
| > • 4 cups chopped romaine lettuce                                   |
| >                                                                    |
| > • 3 tbsp plain Greek yogurt                                        |
| >                                                                    |
| > • 1 tbsp fresh lemon juice                                         |
| >                                                                    |
| > • 1 tsp Worcestershire sauce                                       |
| >                                                                    |
| > • 1 small clove garlic, minced                                     |
| >                                                                    |
| > • 2 tbsp grated Parmesan                                           |
| >                                                                    |
| > • ½ tsp Dijon mustard                                              |
| >                                                                    |
| > • Salt and black pepper to taste                                   |
| >                                                                    |
| > • Lemon wedges to serve                                            |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Season chicken with salt, pepper, and a drizzle of olive oil.  |
| > Grill or pan-cook 6 min per side.                                  |
| >                                                                    |
| > 2\. Whisk yogurt, lemon juice, Worcestershire, garlic, mustard,    |
| > and Parmesan for dressing. Season.                                 |
| >                                                                    |
| > 3\. Slice chicken against the grain once rested 3 min.             |
| >                                                                    |
| > 4\. Toss romaine with dressing. Top with sliced chicken.           |
| >                                                                    |
| > 5\. Serve with lemon wedges.                                       |
+----------------------------------------------------------------------+

| Carbs: 8g \| Fat: 10g                                                |
