+----------------------------------------------------------------------+
| **\#36 Chole (Chickpea Curry) with Basmati Rice** \| Indian · Lunch  |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 25 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 can (15 oz / 425 g) chickpeas, rinsed                          |
| >                                                                    |
| > • 1 cup cooked basmati rice                                        |
| >                                                                    |
| > • 1 can (14 oz / 400 g) diced tomatoes                             |
| >                                                                    |
| > • 1 small onion, finely diced                                      |
| >                                                                    |
| > • 3 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 1½ tsp cumin                                                     |
| >                                                                    |
| > • 1 tsp coriander                                                  |
| >                                                                    |
| > • ½ tsp turmeric                                                   |
| >                                                                    |
| > • ½ tsp garam masala                                               |
| >                                                                    |
| > • ¼ tsp cayenne (optional)                                         |
| >                                                                    |
| > • 1 tbsp avocado oil                                               |
| >                                                                    |
| > • Salt to taste, fresh coriander and lemon to serve                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil in a large pan over medium. Sauté onion 5 min until   |
| > golden.                                                            |
| >                                                                    |
| > 2\. Add garlic and ginger; cook 1 min. Add cumin, coriander,       |
| > turmeric, and cayenne; stir 30 sec.                                |
| >                                                                    |
| > 3\. Pour in tomatoes; simmer 8 min until sauce thickens. Add       |
| > chickpeas.                                                         |
| >                                                                    |
| > 4\. Simmer 10 more min, mashing a few chickpeas to thicken the     |
| > curry.                                                             |
| >                                                                    |
| > 5\. Add garam masala, stir, and serve over basmati rice with       |
| > coriander and lemon.                                               |
