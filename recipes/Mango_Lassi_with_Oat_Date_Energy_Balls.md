+----------------------------------------------------------------------+
| **\#84 Mango Lassi with Oat & Date Energy Balls** \| Indian ·        |
| Pre-Game Fueling                                                     |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 0 min \| 🍽 Serves: 4                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • Mango Lassi: 2 cups plain low-fat yogurt (or coconut yogurt for  |
| > DF)                                                                |
| >                                                                    |
| > • 1½ cups frozen mango chunks                                      |
| >                                                                    |
| > • ½ cup low-fat milk or oat milk                                   |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • ¼ tsp cardamom                                                   |
| >                                                                    |
| > • Energy Balls: 1 cup rolled oats (certified GF for GF)            |
| >                                                                    |
| > • 8 Medjool dates, pitted                                          |
| >                                                                    |
| > • 2 tbsp almond butter (or sunflower butter for TNF/PF)            |
| >                                                                    |
| > • 2 tbsp shredded coconut (optional)                               |
| >                                                                    |
| > • Pinch of salt                                                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Blend all lassi ingredients until smooth. Serve chilled.       |
| >                                                                    |
| > 2\. For energy balls: pulse dates in a food processor until they   |
| > form a paste.                                                      |
| >                                                                    |
| > 3\. Combine date paste with oats, almond butter, coconut, and      |
| > salt; mix until it holds together.                                 |
| >                                                                    |
| > 4\. Roll into 12 balls (\~1 tbsp each); refrigerate 30 min to firm |
| > up.                                                                |
| >                                                                    |
| > 5\. Serve 2 energy balls with 1 glass of mango lassi as a pre-game |
| > combo 1.5--2 hours before competition.                             |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 14g \| Calories: 390 kcal \|      |
| Carbs: 65g \| Fat: 8g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Fish-Free\] \[Shellfish-Free\]         |
| \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]                         |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Carb\] \[High Potassium\] \[High          |
| Calcium\] \[Vitamin C\] \[High Magnesium\] \[High Folate\]           |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains dairy (yogurt/milk) and tree nuts (almond  |
| butter). Use coconut yogurt + oat milk for DF; use sunflower butter  |
| for TNF/PF; use certified GF oats for GF. Mango is rich in Vitamin   |
| C; dates provide quick-release glucose.*                             |
