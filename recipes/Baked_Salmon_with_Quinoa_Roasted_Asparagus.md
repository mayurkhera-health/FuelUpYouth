+----------------------------------------------------------------------+
| **\#53 Baked Salmon with Quinoa & Roasted Asparagus** \| American ·  |
| Dinner                                                               |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 25 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 salmon fillets (5 oz each)                                     |
| >                                                                    |
| > • 1½ cups quinoa, rinsed                                           |
| >                                                                    |
| > • 3 cups water                                                     |
| >                                                                    |
| > • 1 lb asparagus, tough ends trimmed                               |
| >                                                                    |
| > • 2 tbsp olive oil                                                 |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 lemon (zest + juice)                                           |
| >                                                                    |
| > • 1 tsp dried dill                                                 |
| >                                                                    |
| > • ½ tsp smoked paprika                                             |
| >                                                                    |
| > • Salt and pepper to taste                                         |
| >                                                                    |
| > • 2 tbsp fresh parsley, chopped                                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 400°F. Line a sheet pan with parchment paper.  |
| >                                                                    |
| > 2\. Cook quinoa: bring water to a boil, add quinoa, reduce heat,   |
| > cover and simmer 15 min. Fluff with a fork.                        |
| >                                                                    |
| > 3\. Toss asparagus with 1 tbsp olive oil, garlic, salt, and        |
| > pepper; arrange on half the sheet pan.                             |
| >                                                                    |
| > 4\. Place salmon fillets on the other half; drizzle with remaining |
| > olive oil, lemon juice, dill, and paprika.                         |
| >                                                                    |
| > 5\. Bake 15--18 min until salmon flakes easily with a fork and     |
| > asparagus is tender.                                               |
| >                                                                    |
| > 6\. Plate quinoa, top with salmon, and fan asparagus alongside.    |
| >                                                                    |
| > 7\. Finish with lemon zest and fresh parsley.                      |
