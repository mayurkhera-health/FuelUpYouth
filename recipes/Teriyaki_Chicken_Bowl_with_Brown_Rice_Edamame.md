+----------------------------------------------------------------------+
| **\#61 Teriyaki Chicken Bowl with Brown Rice & Edamame** \| Japanese |
| · Dinner                                                             |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs boneless skinless chicken thighs                         |
| >                                                                    |
| > • 2 cups brown rice, cooked                                        |
| >                                                                    |
| > • 1 cup shelled edamame, thawed                                    |
| >                                                                    |
| > • 1 cup shredded purple cabbage                                    |
| >                                                                    |
| > • 1 large carrot, julienned                                        |
| >                                                                    |
| > • 2 green onions, sliced                                           |
| >                                                                    |
| > • 1 tbsp sesame seeds (omit for SesF)                              |
| >                                                                    |
| > • 1 tbsp avocado oil                                               |
| >                                                                    |
| > • Teriyaki sauce: 4 tbsp low-sodium soy sauce (or coconut aminos   |
| > for SoyF)                                                          |
| >                                                                    |
| > • 2 tbsp honey or maple syrup                                      |
| >                                                                    |
| > • 1 tbsp rice vinegar                                              |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp cornstarch dissolved in 1 tsp water                        |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Whisk all teriyaki sauce ingredients in a small saucepan;      |
| > simmer 3--4 min until slightly thickened. Set aside.               |
| >                                                                    |
| > 2\. Heat avocado oil in a large skillet over medium-high. Add      |
| > chicken thighs; cook 5--6 min per side until cooked through.       |
| >                                                                    |
| > 3\. Pour half the teriyaki sauce over chicken in the last 2 min of |
| > cooking; flip to coat.                                             |
| >                                                                    |
| > 4\. Slice chicken into strips.                                     |
| >                                                                    |
| > 5\. Build bowls: brown rice base, topped with chicken, edamame,    |
| > cabbage, and carrot.                                               |
| >                                                                    |
| > 6\. Drizzle remaining teriyaki sauce over the top; garnish with    |
| > green onions and sesame seeds.                                     |

| Carbs: 55g \| Fat: 12g                                               |

| SoyF; omit sesame seeds for SesF. Use tamari for GF. Edamame adds    |
| plant-based protein and folate --- great for young female athletes.* |
