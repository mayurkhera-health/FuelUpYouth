+----------------------------------------------------------------------+
| **\#127 Savory Roasted Chickpeas (Indian-Spiced)** \| Indian ·       |
| Mid-Day Snacks                                                       |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 30 min \| 🍽 Serves: 4                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cans (15 oz each) chickpeas, drained, rinsed, and patted very  |
| > dry                                                                |
| >                                                                    |
| > • 2 tbsp olive oil                                                 |
| >                                                                    |
| > • 1 tsp cumin, 1 tsp smoked paprika, ½ tsp turmeric                |
| >                                                                    |
| > • ½ tsp garlic powder, ½ tsp salt                                  |
| >                                                                    |
| > • Optional: ¼ tsp cayenne for heat                                 |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 425°F; pat chickpeas completely dry (critical  |
| > for crunch).                                                       |
| >                                                                    |
| > 2\. Toss with oil and spices; spread in single layer on baking     |
| > sheet.                                                             |
| >                                                                    |
| > 3\. Roast 25--30 min, shaking halfway, until very crispy and       |
| > golden.                                                            |
| >                                                                    |
| > 4\. Cool completely before storing; keep in an open container at   |
| > room temp up to 3 days.                                            |
| >                                                                    |
| > 5\. 15g plant protein per serving with the same crunch as chips    |
| > --- a game-changing travel snack.                                  |
