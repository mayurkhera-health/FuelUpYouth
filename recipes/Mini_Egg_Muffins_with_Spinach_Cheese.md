+----------------------------------------------------------------------+
| **\#4 Mini Egg Muffins with Spinach & Cheese** \| American ·         |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 6 large eggs                                                     |
| >                                                                    |
| > • 1 cup baby spinach, chopped                                      |
| >                                                                    |
| > • ½ cup shredded mozzarella or cheddar                             |
| >                                                                    |
| > • ¼ cup diced bell pepper (any colour)                             |
| >                                                                    |
| > • 2 tbsp milk                                                      |
| >                                                                    |
| > • Salt, black pepper, garlic powder to taste                       |
| >                                                                    |
| > • Cooking spray                                                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 375°F (190°C). Spray a 6-cup muffin tin well.  |
| >                                                                    |
| > 2\. Whisk eggs with milk, salt, pepper, and garlic powder.         |
| >                                                                    |
| > 3\. Divide spinach, bell pepper, and cheese evenly among muffin    |
| > cups.                                                              |
| >                                                                    |
| > 4\. Pour egg mixture over vegetables, filling each cup ¾ full.     |
| >                                                                    |
| > 5\. Bake 18--20 min until puffed and set. Cool 5 min before        |
| > removing.                                                          |
