+----------------------------------------------------------------------+
| **\#82 Chicken & White Rice Congee (Rice Porridge)** \| Chinese ·    |
| Pre-Game Fueling                                                     |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 30 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 cup long-grain white rice                                      |
| >                                                                    |
| > • 6 cups low-sodium chicken broth                                  |
| >                                                                    |
| > • 1 lb boneless chicken breast, thinly sliced                      |
| >                                                                    |
| > • 2 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tbsp low-sodium soy sauce (or coconut aminos for SoyF)         |
| >                                                                    |
| > • 2 green onions, sliced                                           |
| >                                                                    |
| > • ½ tsp white pepper                                               |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • Optional toppings: a drizzle of sesame oil (omit for SesF),      |
| > soft-boiled egg                                                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Bring broth to a boil in a large pot. Add rice, ginger, and    |
| > garlic.                                                            |
| >                                                                    |
| > 2\. Reduce heat to low; cook uncovered 25 min, stirring            |
| > occasionally until rice breaks down into a thick porridge.         |
| >                                                                    |
| > 3\. Add sliced chicken; cook 5 min until cooked through. Season    |
| > with soy sauce and white pepper.                                   |
| >                                                                    |
| > 4\. Ladle into bowls; top with green onions and optional toppings. |
| >                                                                    |
| > 5\. Best eaten 2--3 hours before competition. Easy on the stomach  |
| > --- excellent choice before morning tournaments.                   |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 32g \| Calories: 380 kcal \|      |
| Carbs: 50g \| Fat: 5g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Fish-Free\]             |
| \[Shellfish-Free\] \[Tree Nut-Free\] \[Peanut-Free\] \[Sesame-Free\] |
| \[Gluten-Free\]                                                      |
|                                                                      |
| **DIETARY:** \[GF\] \[DF\] \[Low-Sugar / Diabetic-Friendly\]         |
| \[Low-Sodium\]                                                       |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Carb\] \[B12-Rich\]      |
| \[High Potassium\] \[High Zinc\]                                     |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains soy (soy sauce). Use coconut aminos for    |
| SoyF. Naturally GF and DF. Congee is one of the most easily          |
| digestible pre-game meals --- the broken-down rice causes minimal GI |
| distress during competition.*                                        |
