+----------------------------------------------------------------------+
| **\#30 Chicken & Black Bean Tortilla Soup** \| Mexican · Lunch       |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 25 min \| 🍽 Serves: 3                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 boneless chicken breasts (about 12 oz / 340 g)                 |
| >                                                                    |
| > • 1 can (15 oz / 425 g) black beans, rinsed                        |
| >                                                                    |
| > • 1 can (14 oz / 400 g) fire-roasted diced tomatoes                |
| >                                                                    |
| > • 3 cups low-sodium chicken broth                                  |
| >                                                                    |
| > • 1 cup frozen corn kernels                                        |
| >                                                                    |
| > • 1 small onion, diced                                             |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • ½ tsp smoked paprika                                             |
| >                                                                    |
| > • ½ tsp chili powder                                               |
| >                                                                    |
| > • 1 tbsp avocado oil                                               |
| >                                                                    |
| > • Baked corn tortilla strips, avocado, plain Greek yogurt, and     |
| > lime to serve                                                      |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil in a pot. Sauté onion and garlic 3 min. Add cumin,    |
| > paprika, and chili powder; stir 1 min.                             |
| >                                                                    |
| > 2\. Add chicken breasts, tomatoes, broth, black beans, and corn.   |
| > Bring to a boil.                                                   |
| >                                                                    |
| > 3\. Reduce heat; simmer 18--20 min until chicken is cooked         |
| > through.                                                           |
| >                                                                    |
| > 4\. Remove chicken; shred with two forks and return to the pot.    |
| >                                                                    |
| > 5\. Ladle into bowls; top with tortilla strips, avocado slices, a  |
| > spoon of Greek yogurt, and a squeeze of lime.                      |
