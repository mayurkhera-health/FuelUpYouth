+----------------------------------------------------------------------+
| **\#125 Guacamole & Baked Tortilla Chips (Mexican)** \| Mexican ·    |
| Mid-Day Snacks                                                       |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 0 min \| 🍽 Serves: 4                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 3 ripe avocados, pitted and mashed                               |
| >                                                                    |
| > • 1 lime, juiced                                                   |
| >                                                                    |
| > • ¼ cup red onion, finely diced                                    |
| >                                                                    |
| > • 1 jalapeño, seeded and minced (optional)                         |
| >                                                                    |
| > • 2 tbsp fresh cilantro, chopped                                   |
| >                                                                    |
| > • Salt and pepper to taste                                         |
| >                                                                    |
| > • 4 oz baked corn tortilla chips (GF)                              |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Mash avocados with lime juice until mostly smooth but slightly |
| > chunky.                                                            |
| >                                                                    |
| > 2\. Fold in onion, jalapeño, and cilantro; season with salt and    |
| > pepper.                                                            |
| >                                                                    |
| > 3\. Press plastic wrap directly on surface to prevent browning;    |
| > refrigerate until serving.                                         |
| >                                                                    |
| > 4\. Portion ¼ cup guacamole per athlete with a serving of chips.   |
| >                                                                    |
| > 5\. Avocado\'s monounsaturated fats support joint lubrication and  |
| > help absorb fat-soluble vitamins A, D, E, K.                       |
