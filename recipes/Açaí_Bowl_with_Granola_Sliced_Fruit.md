+----------------------------------------------------------------------+
| **\#85 Açaí Bowl with Granola & Sliced Fruit** \| American ·         |
| Pre-Game Fueling                                                     |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 packets (3.5 oz each) unsweetened frozen açaí purée            |
| >                                                                    |
| > • 1 cup frozen blueberries                                         |
| >                                                                    |
| > • 1 frozen banana                                                  |
| >                                                                    |
| > • ½ cup oat milk or almond milk                                    |
| >                                                                    |
| > • Toppings: ½ cup low-fat granola (certified GF for GF)            |
| >                                                                    |
| > • 1 banana, sliced                                                 |
| >                                                                    |
| > • ½ cup fresh strawberries, sliced                                 |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • Optional: 1 tbsp hemp seeds for extra protein                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Blend açaí packets, frozen blueberries, frozen banana, and     |
| > milk until thick and smooth (add milk gradually --- keep it        |
| > thick).                                                            |
| >                                                                    |
| > 2\. Pour into bowls.                                               |
| >                                                                    |
| > 3\. Top with granola, fresh fruit, and honey.                      |
| >                                                                    |
| > 4\. Add hemp seeds if using.                                       |
| >                                                                    |
| > 5\. Eat 1.5--2 hours before competition. Skip heavy granola        |
| > toppings if eating within 1 hour.                                  |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 10g \| Calories: 370 kcal \|      |
| Carbs: 68g \| Fat: 7g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Fish-Free\]             |
| \[Shellfish-Free\] \[Tree Nut-Free\] \[Peanut-Free\] \[Soy-Free\]    |
| \[Sesame-Free\]                                                      |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Ve\] \[GF\] \[DF\] \[Low-Sugar /       |
| Diabetic-Friendly\]                                                  |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Carb\] \[Vitamin C\] \[High Potassium\]   |
| \[Omega-3\] \[High Folate\] \[High Magnesium\]                       |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Use certified GF granola for GF; check granola for  |
| TNF. Açaí is rich in antioxidants --- great for reducing pre-game    |
| inflammation. Use oat milk for nut-free version. Visually exciting   |
| for young athletes.*                                                 |
