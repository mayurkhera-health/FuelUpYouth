+----------------------------------------------------------------------+
| **\#13 Korean Veggie & Egg Fried Rice (Bokkeumbap)** \| Korean ·     |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 12 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cups day-old cooked brown rice                                 |
| >                                                                    |
| > • 3 large eggs                                                     |
| >                                                                    |
| > • ½ cup frozen peas and diced carrots, thawed                      |
| >                                                                    |
| > • ½ cup shredded purple cabbage                                    |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1½ tbsp low-sodium soy sauce or tamari                           |
| >                                                                    |
| > • 1 tsp sesame oil (omit for SesF)                                 |
| >                                                                    |
| > • 1 tbsp avocado or vegetable oil                                  |
| >                                                                    |
| > • 2 scallions, sliced                                              |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil in a wok or large skillet over high heat. Add garlic, |
| > stir 30 seconds until fragrant.                                    |
| >                                                                    |
| > 2\. Add peas, carrots, and cabbage; stir-fry 2--3 min.             |
| >                                                                    |
| > 3\. Push vegetables to one side; crack eggs in and scramble until  |
| > just set.                                                          |
| >                                                                    |
| > 4\. Add rice, breaking up any clumps. Pour soy sauce over and toss |
| > everything together 2 min.                                         |
| >                                                                    |
| > 5\. Drizzle sesame oil, scatter scallions, and serve immediately.  |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 18g \| Calories: 380 kcal \|      |
| Carbs: 52g \| Fat: 11g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Shellfish-Free\] \[Fish-Free\]       |
| \[Tree Nut-Free\] \[Peanut-Free\]                                    |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Carb\] \[High Protein\] \[Vitamin C\]     |
| \[High Iron\] \[High Folate\] \[B12-Rich\]                           |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Use tamari for GF & SoyF. Omit sesame oil for SesF. |
| Day-old rice is key --- it fries without clumping. Great use of      |
| leftover dinner rice.*                                               |
