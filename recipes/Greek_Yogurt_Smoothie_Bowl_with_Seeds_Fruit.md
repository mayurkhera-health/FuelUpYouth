+----------------------------------------------------------------------+
| **\#25 Greek Yogurt Smoothie Bowl with Seeds & Fruit** \| American · |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 1                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • ¾ cup plain full-fat Greek yogurt                                |
| >                                                                    |
| > • ½ frozen banana                                                  |
| >                                                                    |
| > • ¼ cup frozen mango chunks                                        |
| >                                                                    |
| > • Splash of whole milk or oat milk to blend                        |
| >                                                                    |
| > • 1 tbsp chia seeds                                                |
| >                                                                    |
| > • 1 tbsp hemp hearts (hulled hemp seeds)                           |
| >                                                                    |
| > • ½ cup fresh berries (blueberries, raspberries)                   |
| >                                                                    |
| > • 1 tbsp raw honey                                                 |
| >                                                                    |
| > • 2 tbsp granola (nut-free if needed)                              |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Blend Greek yogurt, frozen banana, frozen mango, and a splash  |
| > of milk until thick and smooth.                                    |
| >                                                                    |
| > 2\. Pour into a wide bowl --- it should be thick enough to hold    |
| > toppings.                                                          |
| >                                                                    |
| > 3\. Arrange berries, granola, chia seeds, and hemp hearts in rows  |
| > on top.                                                            |
| >                                                                    |
| > 4\. Drizzle with honey. Eat immediately with a spoon.              |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 24g \| Calories: 415 kcal \|      |
| Carbs: 58g \| Fat: 10g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Gluten-Free\] \[Shellfish-Free\]       |
| \[Fish-Free\] \[Tree Nut-Free\] \[Peanut-Free\] \[Sesame-Free\]      |
| \[Soy-Free\]                                                         |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Calcium\] \[Omega-3\]    |
| \[Vitamin C\] \[High Potassium\] \[High Magnesium\]                  |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *No protein powder needed --- Greek yogurt and hemp  |
| hearts supply all the protein. Use dairy-free coconut yogurt + oat   |
| milk to make fully vegan. Use TNF granola for nut allergy.*          |
