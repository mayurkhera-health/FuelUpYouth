+----------------------------------------------------------------------+
| **\#120 Smoked Salmon & Cream Cheese Cucumber Rounds (American)** \| |
| American · Mid-Day Snacks                                            |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 large English cucumber, sliced into ¼-inch rounds              |
| >                                                                    |
| > • 4 oz smoked salmon, torn into small pieces                       |
| >                                                                    |
| > • 4 tbsp cream cheese (or dairy-free cream cheese for DF)          |
| >                                                                    |
| > • 1 tbsp capers (optional)                                         |
| >                                                                    |
| > • Fresh dill or chives for garnish                                 |
| >                                                                    |
| > • Lemon zest and black pepper                                      |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Arrange cucumber rounds on a plate or in a container.          |
| >                                                                    |
| > 2\. Spread ½ tsp cream cheese on each round.                       |
| >                                                                    |
| > 3\. Top with a piece of smoked salmon; add a caper if using.       |
| >                                                                    |
| > 4\. Garnish with fresh dill, lemon zest, and black pepper.         |
| >                                                                    |
| > 5\. Keep refrigerated; eat within 4 hours --- salmon\'s Omega-3s   |
| > support joint recovery between practices.                          |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 14g \| Calories: 200 kcal \|      |
| Carbs: 8g \| Fat: 12g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Shellfish-Free\] \[Tree Nut-Free\]     |
| \[Peanut-Free\] \[Gluten-Free\] \[Soy-Free\] \[Sesame-Free\]         |
|                                                                      |
| **DIETARY:** \[GF\] \[SoyF\] \[SesF\] \[Low-Sugar /                  |
| Diabetic-Friendly\]                                                  |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[Omega-3\] \[Vitamin D\]       |
| \[B12-Rich\] \[High Calcium\] \[High Potassium\]                     |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains fish (salmon) and dairy (cream cheese).    |
| Use dairy-free cream cheese for DF. Smoked salmon is one of the      |
| richest natural sources of Vitamin D and Omega-3 EPA/DHA; cucumber   |
| provides hydration and potassium with almost no calories.*           |
