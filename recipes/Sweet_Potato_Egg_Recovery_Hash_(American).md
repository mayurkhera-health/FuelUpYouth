+----------------------------------------------------------------------+
| **\#114 Sweet Potato & Egg Recovery Hash (American)** \| American ·  |
| Post-Game Recovery                                                   |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 large sweet potatoes, peeled and diced small                   |
| >                                                                    |
| > • 6 large eggs                                                     |
| >                                                                    |
| > • 1 red bell pepper, diced                                         |
| >                                                                    |
| > • 1 small onion, diced                                             |
| >                                                                    |
| > • 2 cups baby spinach                                              |
| >                                                                    |
| > • 2 tbsp olive oil                                                 |
| >                                                                    |
| > • 1 tsp smoked paprika, ½ tsp cumin, salt & pepper                 |
| >                                                                    |
| > • Optional: hot sauce or salsa on the side                         |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil in a large cast-iron skillet over medium-high; add    |
| > sweet potato and cook 10 min, stirring occasionally.               |
| >                                                                    |
| > 2\. Add bell pepper and onion; cook 5 min until softened.          |
| >                                                                    |
| > 3\. Season with paprika, cumin, salt, and pepper.                  |
| >                                                                    |
| > 4\. Create wells in the hash; crack an egg into each well; cover   |
| > and cook 3--4 min for runny yolks.                                 |
| >                                                                    |
| > 5\. Stir in spinach until wilted; serve immediately --- complete   |
| > carb + protein post-game meal.                                     |
