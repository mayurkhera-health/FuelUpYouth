+----------------------------------------------------------------------+
| **\#126 Greek Yogurt Bark with Berries & Granola (American)** \|     |
| American · Mid-Day Snacks                                            |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 0 min \| 🍽 Serves: 4                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cups plain Greek yogurt                                        |
| >                                                                    |
| > • 2 tbsp honey                                                     |
| >                                                                    |
| > • 1 cup mixed berries (blueberries, raspberries, sliced            |
| > strawberries)                                                      |
| >                                                                    |
| > • ¼ cup granola (GF certified for GF)                              |
| >                                                                    |
| > • 1 tbsp chia seeds                                                |
| >                                                                    |
| > • Optional: 1 tbsp dark chocolate chips                            |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Mix Greek yogurt with honey; spread into a thin layer on a     |
| > parchment-lined baking sheet.                                      |
| >                                                                    |
| > 2\. Top with berries, granola, chia seeds, and chocolate chips if  |
| > using.                                                             |
| >                                                                    |
| > 3\. Freeze at least 2 hours until solid.                           |
| >                                                                    |
| > 4\. Break into pieces; store in a freezer bag up to 2 weeks.       |
| >                                                                    |
| > 5\. Eat straight from freezer --- doubles as a hot-weather         |
| > cool-down snack after outdoor games.                               |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 14g \| Calories: 230 kcal \|      |
| Carbs: 30g \| Fat: 6g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Fish-Free\] \[Shellfish-Free\]         |
| \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]                         |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[PF\] \[SoyF\] \[SesF\] \[Low-Sugar /   |
| Diabetic-Friendly\]                                                  |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Calcium\] \[High Protein\] \[Vitamin C\]  |
| \[Vitamin D\] \[High Potassium\] \[Omega-3\] \[B12-Rich\]            |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains dairy (yogurt), gluten (granola), tree     |
| nuts (granola may contain). Use GF granola for GF; DF yogurt         |
| alternative for DF. Greek yogurt bark is particularly popular for    |
| hot-season tournament play --- the frozen format acts as hydration   |
| support.*                                                            |
