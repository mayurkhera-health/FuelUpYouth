+----------------------------------------------------------------------+
| **\#6 Cottage Cheese Bowl with Pineapple & Walnuts** \| American ·   |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 1                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 cup low-fat cottage cheese                                     |
| >                                                                    |
| > • ½ cup fresh pineapple chunks                                     |
| >                                                                    |
| > • 2 tbsp chopped walnuts                                           |
| >                                                                    |
| > • 1 tbsp raw honey                                                 |
| >                                                                    |
| > • Pinch of cinnamon                                                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Spoon cottage cheese into a bowl.                              |
| >                                                                    |
| > 2\. Top with pineapple, walnuts, and a drizzle of honey.           |
| >                                                                    |
| > 3\. Sprinkle cinnamon over the top and serve immediately.          |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 28g \| Calories: 340 kcal \|      |
| Carbs: 34g \| Fat: 10g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Gluten-Free\] \[Shellfish-Free\]       |
| \[Fish-Free\] \[Peanut-Free\] \[Sesame-Free\] \[Soy-Free\]           |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Calcium\] \[Omega-3\]    |
| \[Vitamin C\]                                                        |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains tree nuts (walnuts). Omit walnuts or swap  |
| with pumpkin seeds (pepitas) to make TNF & PF.*                      |
