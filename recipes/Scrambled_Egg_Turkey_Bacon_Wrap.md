+----------------------------------------------------------------------+
| **\#1 Scrambled Egg & Turkey Bacon Wrap** \| American · Breakfast    |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 10 min \| 🍽 Serves: 1                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 3 large eggs                                                     |
| >                                                                    |
| > • 2 strips turkey bacon                                            |
| >                                                                    |
| > • 1 whole wheat flour tortilla (10-inch)                           |
| >                                                                    |
| > • ¼ cup shredded cheddar cheese                                    |
| >                                                                    |
| > • Handful baby spinach                                             |
| >                                                                    |
| > • Salt and black pepper to taste                                   |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Cook turkey bacon in a skillet over medium heat until crispy,  |
| > \~5 min. Remove and chop.                                          |
| >                                                                    |
| > 2\. Beat eggs with salt and pepper. Scramble in same pan over      |
| > medium-low until just set.                                         |
| >                                                                    |
| > 3\. Warm tortilla in a dry pan for 15 seconds.                     |
| >                                                                    |
| > 4\. Layer spinach, eggs, bacon, and cheese on tortilla. Roll       |
| > tightly and slice in half.                                         |
+----------------------------------------------------------------------+

| Carbs: 28g \| Fat: 18g                                               |
