+----------------------------------------------------------------------+
| **\#111 Edamame & Brown Rice Recovery Bowl with Ginger Miso          |
| Dressing** \| Japanese · Post-Game Recovery                          |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 5 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cups cooked brown rice                                         |
| >                                                                    |
| > • 1½ cups shelled edamame (thawed from frozen)                     |
| >                                                                    |
| > • 1 cup shredded red cabbage                                       |
| >                                                                    |
| > • 1 medium carrot, julienned or grated                             |
| >                                                                    |
| > • ½ cucumber, thinly sliced                                        |
| >                                                                    |
| > • 2 tbsp white miso paste                                          |
| >                                                                    |
| > • 1 tbsp rice vinegar                                              |
| >                                                                    |
| > • 1 tbsp low-sodium soy sauce (or tamari for GF)                   |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 1 tsp sesame oil (omit for SesF)                                 |
| >                                                                    |
| > • 1 tbsp sesame seeds (omit for SesF)                              |
| >                                                                    |
| > • 2 green onions, thinly sliced                                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Whisk miso, rice vinegar, soy sauce, ginger, and sesame oil    |
| > with 2 tbsp warm water.                                            |
| >                                                                    |
| > 2\. Warm edamame briefly in microwave or hot water.                |
| >                                                                    |
| > 3\. Build bowls: brown rice base, edamame, cabbage, carrot,        |
| > cucumber.                                                          |
| >                                                                    |
| > 4\. Drizzle miso dressing over; garnish with sesame seeds and      |
| > green onions.                                                      |
| >                                                                    |
| > 5\. Eat within 45 min of competition; edamame provides complete    |
| > plant protein + glycogen-replenishing carbs.                       |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 22g \| Calories: 390 kcal \|      |
| Carbs: 56g \| Fat: 10g                                               |

|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Vegan\] \[GF\] \[DF\] \[Low-Sugar /    |
| Diabetic-Friendly\]                                                  |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Carb\] \[High Iron\]     |
| \[High Folate\] \[High Potassium\] \[High Magnesium\] \[High Zinc\]  |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains soy (edamame, miso, soy sauce) and sesame. |
| Use tamari for GF; coconut aminos for SoyF; omit sesame for SesF.    |
| Edamame is one of the few plant foods with all 9 essential amino     |
| acids; miso adds gut-supporting probiotics.*                         |
