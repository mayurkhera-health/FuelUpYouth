+----------------------------------------------------------------------+
| **\#106 Lentil & Vegetable Recovery Soup (Indian Masoor Dal)** \|    |
| Indian · Post-Game Recovery                                          |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 30 min \| 🍽 Serves: 6                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cups red lentils, rinsed                                       |
| >                                                                    |
| > • 1 can (14 oz) diced tomatoes                                     |
| >                                                                    |
| > • 1 large onion, diced                                             |
| >                                                                    |
| > • 3 cloves garlic, minced                                          |
| >                                                                    |
| > • 1-inch ginger, grated                                            |
| >                                                                    |
| > • 1 tsp turmeric, 1 tsp cumin, 1 tsp coriander                     |
| >                                                                    |
| > • ½ tsp garam masala                                               |
| >                                                                    |
| > • 6 cups low-sodium vegetable broth                                |
| >                                                                    |
| > • 2 cups spinach, roughly chopped                                  |
| >                                                                    |
| > • 2 tbsp olive oil                                                 |
| >                                                                    |
| > • Lemon juice and cilantro to serve                                |
| >                                                                    |
| > • Whole wheat naan or brown rice (serve alongside)                 |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Sauté onion in oil until golden (8--10 min); add garlic,       |
| > ginger, spices; cook 2 min.                                        |
| >                                                                    |
| > 2\. Add lentils, tomatoes, and broth; bring to boil; reduce heat   |
| > and simmer 20 min.                                                 |
| >                                                                    |
| > 3\. Stir in spinach until wilted; finish with lemon juice.         |
| >                                                                    |
| > 4\. Serve with naan or rice for complete recovery carbs + protein. |
| >                                                                    |
| > 5\. Turmeric and ginger together are among the strongest dietary   |
| > anti-inflammatory combinations.                                    |
