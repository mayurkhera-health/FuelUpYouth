+----------------------------------------------------------------------+
| **\#38 Arroz con Pollo (Chicken & Rice)** \| Mexican · Lunch         |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 30 min \| 🍽 Serves: 3                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 3 boneless chicken thighs (about 14 oz / 400 g)                  |
| >                                                                    |
| > • 1 cup long-grain or basmati rice                                 |
| >                                                                    |
| > • 1¾ cups low-sodium chicken broth                                 |
| >                                                                    |
| > • 1 can (14 oz / 400 g) diced tomatoes                             |
| >                                                                    |
| > • ½ small onion, diced                                             |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • ½ cup frozen peas                                                |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • ½ tsp smoked paprika                                             |
| >                                                                    |
| > • ½ tsp turmeric (for colour)                                      |
| >                                                                    |
| > • 1 tbsp avocado oil                                               |
| >                                                                    |
| > • Salt and pepper, fresh cilantro and lime to serve                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Season chicken with cumin, paprika, salt, and pepper.          |
| >                                                                    |
| > 2\. Heat oil in a deep skillet or Dutch oven. Brown chicken 3 min  |
| > per side. Remove.                                                  |
| >                                                                    |
| > 3\. In same pan, sauté onion 3 min. Add garlic and rice; toast 2   |
| > min until rice is lightly golden.                                  |
| >                                                                    |
| > 4\. Add tomatoes, broth, and turmeric. Nestle chicken back in,     |
| > bring to a boil.                                                   |
| >                                                                    |
| > 5\. Cover and simmer 20 min. Scatter peas on top in last 5 min.    |
| > Serve with cilantro and lime.                                      |
