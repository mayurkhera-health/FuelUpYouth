+----------------------------------------------------------------------+
| **\#90 Blueberry & Banana Smoothie with Oats** \| American ·         |
| Pre-Game Fueling                                                     |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 ripe bananas (1 frozen for thickness)                          |
| >                                                                    |
| > • 1 cup frozen blueberries                                         |
| >                                                                    |
| > • ½ cup old-fashioned oats (certified GF for GF)                   |
| >                                                                    |
| > • 1½ cups low-fat milk or oat milk (for DF)                        |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • ½ tsp vanilla extract                                            |
| >                                                                    |
| > • Optional: 1 scoop of protein-free collagen powder (whole food    |
| > only)                                                              |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Add all ingredients to a blender.                              |
| >                                                                    |
| > 2\. Blend 60 sec until completely smooth.                          |
| >                                                                    |
| > 3\. Pour into glasses; drink immediately.                          |
| >                                                                    |
| > 4\. Best consumed 1.5--2 hours before game time.                   |
| >                                                                    |
| > 5\. For 30-min pre-game: skip oats (too much fiber) and just blend |
| > banana, blueberries, honey, and milk.                              |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 12g \| Calories: 360 kcal \|      |
| Carbs: 68g \| Fat: 4g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Fish-Free\] \[Shellfish-Free\] \[Tree  |
| Nut-Free\] \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]              |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Carb\] \[High Potassium\] \[Vitamin C\]   |
| \[High Calcium\] \[High Magnesium\] \[High Folate\] \[Omega-3\]      |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains dairy (if using milk) and oats (may        |
| contain gluten). Use oat milk for DF; use certified GF oats for GF.  |
| Blueberries are antioxidant-rich --- reduce exercise-induced         |
| oxidative stress. Quick to make on game-day mornings.*               |
