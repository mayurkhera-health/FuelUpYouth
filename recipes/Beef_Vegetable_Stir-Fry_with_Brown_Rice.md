+----------------------------------------------------------------------+
| **\#52 Beef & Vegetable Stir-Fry with Brown Rice** \| Chinese ·      |
| Dinner                                                               |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 15 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 lb flank steak, thinly sliced against the grain                |
| >                                                                    |
| > • 2 cups brown rice, cooked                                        |
| >                                                                    |
| > • 2 cups broccoli florets                                          |
| >                                                                    |
| > • 1 red bell pepper, sliced                                        |
| >                                                                    |
| > • 1 cup snap peas                                                  |
| >                                                                    |
| > • 1 medium carrot, julienned                                       |
| >                                                                    |
| > • 3 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 3 tbsp low-sodium soy sauce (or coconut aminos for SoyF)         |
| >                                                                    |
| > • 1 tbsp rice vinegar                                              |
| >                                                                    |
| > • 1 tsp sesame oil (omit for SesF)                                 |
| >                                                                    |
| > • 1 tbsp cornstarch                                                |
| >                                                                    |
| > • 1 tbsp avocado oil                                               |
| >                                                                    |
| > • 2 green onions, sliced                                           |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Toss beef slices with cornstarch and 1 tbsp soy sauce; let     |
| > marinate 10 min.                                                   |
| >                                                                    |
| > 2\. Mix remaining soy sauce, rice vinegar, and sesame oil in a     |
| > small bowl; set aside.                                             |
| >                                                                    |
| > 3\. Heat avocado oil in a wok or large skillet over high heat      |
| > until smoking.                                                     |
| >                                                                    |
| > 4\. Add beef in a single layer; cook 1--2 min per side until       |
| > browned. Remove and set aside.                                     |
| >                                                                    |
| > 5\. Add garlic and ginger to the wok; stir-fry 30 seconds until    |
| > fragrant.                                                          |
| >                                                                    |
| > 6\. Add broccoli, bell pepper, snap peas, and carrot; stir-fry     |
| > 4--5 min until tender-crisp.                                       |
| >                                                                    |
| > 7\. Return beef to the wok; pour sauce over everything and toss to |
| > coat.                                                              |
| >                                                                    |
| > 8\. Serve over brown rice and garnish with green onions.           |
