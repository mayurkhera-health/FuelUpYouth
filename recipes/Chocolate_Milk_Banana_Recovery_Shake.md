+----------------------------------------------------------------------+
| **\#96 Chocolate Milk & Banana Recovery Shake** \| American ·        |
| Post-Game Recovery                                                   |
+======================================================================+
| ⏱ Prep: 3 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cups low-fat chocolate milk (or oat milk + 2 tbsp cocoa + 1    |
| > tbsp honey for DF)                                                 |
| >                                                                    |
| > • 2 frozen bananas                                                 |
| >                                                                    |
| > • 1 tbsp almond butter (optional --- omit for TNF/PF)              |
| >                                                                    |
| > • ½ tsp vanilla extract                                            |
| >                                                                    |
| > • Pinch of sea salt (replenishes sodium lost in sweat)             |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Blend all ingredients until smooth.                            |
| >                                                                    |
| > 2\. Drink within 30--45 min of finishing competition or practice.  |
| >                                                                    |
| > 3\. The carb-to-protein ratio of chocolate milk (\~3:1 to 4:1) is  |
| > ideal for glycogen replenishment.                                  |
| >                                                                    |
| > 4\. Pinch of salt replaces electrolytes lost in sweat.             |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 16g \| Calories: 360 kcal \|      |
| Carbs: 58g \| Fat: 8g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Fish-Free\] \[Shellfish-Free\]         |
| \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]                         |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Carb\] \[High Calcium\] \[High Protein\]  |
| \[High Potassium\] \[Vitamin D\] \[B12-Rich\] \[High Magnesium\]     |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains dairy and tree nuts (almond butter). Use   |
| DF chocolate milk substitute for DF. Omit almond butter for TNF/PF.  |
| Chocolate milk is one of the most researched post-exercise recovery  |
| drinks --- outperforms many commercial sports drinks.*               |
