+----------------------------------------------------------------------+
| **\#51 Sheet Pan Chicken Thighs with Sweet Potato & Broccoli** \|    |
| American · Dinner                                                    |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 30 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 bone-in chicken thighs (skin removed, about 1.5 lb / 680 g)    |
| >                                                                    |
| > • 2 medium sweet potatoes, cubed                                   |
| >                                                                    |
| > • 2 cups broccoli florets                                          |
| >                                                                    |
| > • 2 tbsp olive oil                                                 |
| >                                                                    |
| > • 1 tsp garlic powder                                              |
| >                                                                    |
| > • 1 tsp smoked paprika                                             |
| >                                                                    |
| > • ½ tsp onion powder                                               |
| >                                                                    |
| > • Salt and black pepper to taste                                   |
| >                                                                    |
| > • Fresh parsley and lemon wedges to serve                          |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 425°F (220°C). Line a large baking sheet with  |
| > parchment.                                                         |
| >                                                                    |
| > 2\. Toss sweet potato cubes with 1 tbsp oil, half the spices, salt |
| > and pepper. Spread on one side of pan.                             |
| >                                                                    |
| > 3\. Roast 10 min. Push sweet potato aside; add chicken and         |
| > broccoli, both tossed with remaining oil and spices.               |
| >                                                                    |
| > 4\. Roast 20--22 more min until chicken reaches 165°F and sweet    |
| > potato is tender.                                                  |
| >                                                                    |
| > 5\. Garnish with parsley and a squeeze of lemon. Serve directly    |
| > from the pan.                                                      |
