+----------------------------------------------------------------------+
| **\#28 Bibimbap (Korean Mixed Rice Bowl with Beef & Egg)** \| Korean |
| · Lunch                                                              |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 20 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1½ cups cooked short-grain brown rice                            |
| >                                                                    |
| > • 6 oz (170 g) lean ground beef or thinly sliced sirloin           |
| >                                                                    |
| > • 2 large eggs (fried or soft-boiled)                              |
| >                                                                    |
| > • 1 cup baby spinach, blanched and squeezed dry                    |
| >                                                                    |
| > • 1 medium carrot, julienned and sautéed                           |
| >                                                                    |
| > • ½ cup bean sprouts, blanched                                     |
| >                                                                    |
| > • 2 tbsp low-sodium soy sauce or tamari                            |
| >                                                                    |
| > • 1 tsp sesame oil (omit for SesF)                                 |
| >                                                                    |
| > • 1 tsp honey                                                      |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tbsp gochujang (Korean chili paste, optional --- omit for      |
| > strict LS)                                                         |
| >                                                                    |
| > • 1 tsp avocado oil                                                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Mix soy sauce, honey, garlic, and sesame oil. Marinate beef 5  |
| > min; cook in a pan over medium-high until browned.                 |
| >                                                                    |
| > 2\. Prepare each vegetable separately: sauté spinach with garlic,  |
| > blanch bean sprouts, sauté carrots.                                |
| >                                                                    |
| > 3\. Fry eggs sunny-side up in a lightly oiled pan.                 |
| >                                                                    |
| > 4\. Divide rice between two bowls. Arrange beef and all vegetables |
| > in sections around the bowl.                                       |
| >                                                                    |
| > 5\. Top each bowl with a fried egg. Add a small spoonful of        |
| > gochujang. Mix everything together before eating.                  |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 32g \| Calories: 520 kcal \|      |
| Carbs: 54g \| Fat: 15g                                               |

| \[Tree Nut-Free\] \[Peanut-Free\]                                    |
|                                                                      |
| **DIETARY:** \[Low-Sugar / Diabetic-Friendly\]                       |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Carb\] \[High Iron\]     |
| \[Vitamin C\] \[B12-Rich\] \[High Zinc\] \[High Folate\]             |

| **TIP / SWAP:** *Use tamari for GF & SoyF. Omit sesame oil for SesF. |
| Gochujang contains some wheat --- use GF gochujang for strict GF.    |
| One of the most balanced athlete meals in the database.*             |
