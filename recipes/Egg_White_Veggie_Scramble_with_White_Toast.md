+----------------------------------------------------------------------+
| **\#86 Egg White Veggie Scramble with White Toast** \| American ·    |
| Pre-Game Fueling                                                     |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 8 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 6 egg whites (or 3 whole eggs if EF isn\'t a concern)            |
| >                                                                    |
| > • 2 slices white bread (or GF bread for GF/WF)                     |
| >                                                                    |
| > • ½ cup baby spinach                                               |
| >                                                                    |
| > • ¼ cup diced red bell pepper                                      |
| >                                                                    |
| > • 2 tbsp diced onion                                               |
| >                                                                    |
| > • 1 tsp olive oil                                                  |
| >                                                                    |
| > • Salt and pepper to taste                                         |
| >                                                                    |
| > • Optional: 1 tbsp salsa on the side                               |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil in a non-stick skillet over medium. Sauté onion and   |
| > pepper 2 min until softened.                                       |
| >                                                                    |
| > 2\. Add spinach; cook 30 sec until wilted.                         |
| >                                                                    |
| > 3\. Pour in egg whites; scramble gently until just set, \~2--3     |
| > min. Season with salt and pepper.                                  |
| >                                                                    |
| > 4\. Toast bread.                                                   |
| >                                                                    |
| > 5\. Serve scramble on toast. Best eaten 2--2.5 hours before game   |
| > time. Egg whites are easier to digest than whole eggs pre-game.    |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 20g \| Calories: 260 kcal \|      |
| Carbs: 28g \| Fat: 4g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Fish-Free\] \[Shellfish-Free\]       |
| \[Tree Nut-Free\] \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]       |
|                                                                      |
| **DIETARY:** \[DF\] \[Low-Sugar / Diabetic-Friendly\] \[Low-Sodium\] |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Carb\] \[Vitamin C\]     |
| \[High Folate\] \[High Potassium\] \[B12-Rich\]                      |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains eggs and wheat. Use GF bread for GF/WF.    |
| Egg whites are a pure lean protein source with almost no fat ---     |
| perfect for athletes who want protein without slowing digestion      |
| before competition.*                                                 |
