+----------------------------------------------------------------------+
| **\#73 Black Bean & Sweet Potato Enchilada Casserole** \| Mexican ·  |
| Dinner                                                               |
+======================================================================+
| ⏱ Prep: 20 min \| 🍳 Cook: 30 min \| 🍽 Serves: 6                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cans (15 oz each) black beans, drained and rinsed              |
| >                                                                    |
| > • 2 medium sweet potatoes, peeled, cubed, and roasted              |
| >                                                                    |
| > • 8 corn tortillas, cut into strips (certified GF)                 |
| >                                                                    |
| > • 2 cups red enchilada sauce (low-sodium)                          |
| >                                                                    |
| > • 1 cup frozen corn                                                |
| >                                                                    |
| > • 1 red bell pepper, diced                                         |
| >                                                                    |
| > • 1 small onion, diced                                             |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • 1 tsp chili powder                                               |
| >                                                                    |
| > • 1 cup shredded vegan cheese (or regular Mexican-blend --- omit   |
| > for DF/Ve)                                                         |
| >                                                                    |
| > • 1 tbsp avocado oil                                               |
| >                                                                    |
| > • Fresh cilantro, avocado, and lime to serve                       |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 375°F. Roast sweet potato cubes tossed in oil  |
| > at 400°F for 20 min until tender.                                  |
| >                                                                    |
| > 2\. Sauté onion and bell pepper in oil 5 min. Add garlic, cumin,   |
| > and chili powder; cook 1 min.                                      |
| >                                                                    |
| > 3\. Stir in black beans, corn, and roasted sweet potato. Season    |
| > with salt.                                                         |
| >                                                                    |
| > 4\. Spread ¼ cup enchilada sauce in a 9×13 baking dish. Layer      |
| > tortilla strips, bean mixture, and sauce. Repeat layers ending     |
| > with sauce and cheese.                                             |
| >                                                                    |
| > 5\. Bake covered 20 min; uncover last 10 min until bubbling and    |
| > cheese is melted.                                                  |
| >                                                                    |
| > 6\. Rest 5 min; serve topped with cilantro, avocado slices, and a  |
| > squeeze of lime.                                                   |
