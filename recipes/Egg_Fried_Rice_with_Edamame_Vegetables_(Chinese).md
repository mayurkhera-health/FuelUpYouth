+----------------------------------------------------------------------+
| **\#103 Egg Fried Rice with Edamame & Vegetables (Chinese)** \|      |
| Chinese · Post-Game Recovery                                         |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 15 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 3 cups cooked cold rice (day-old works best)                     |
| >                                                                    |
| > • 3 large eggs, beaten                                             |
| >                                                                    |
| > • 1 cup frozen edamame, thawed                                     |
| >                                                                    |
| > • 1 cup frozen peas and carrots mix                                |
| >                                                                    |
| > • 3 green onions, chopped                                          |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 2 tbsp low-sodium soy sauce (or tamari for GF)                   |
| >                                                                    |
| > • 1 tbsp sesame oil (omit for SesF)                                |
| >                                                                    |
| > • 1 tbsp neutral oil                                               |
| >                                                                    |
| > • ½ tsp white pepper                                               |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat neutral oil in a wok or large skillet over high heat.     |
| >                                                                    |
| > 2\. Scramble eggs in the pan; push to side.                        |
| >                                                                    |
| > 3\. Add garlic; stir-fry 30 sec. Add vegetables and edamame;       |
| > stir-fry 3 min.                                                    |
| >                                                                    |
| > 4\. Add cold rice; press and toss to break up clumps for 3--4 min. |
| >                                                                    |
| > 5\. Drizzle soy sauce and sesame oil; toss with green onions;      |
| > serve immediately.                                                 |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 24g \| Calories: 420 kcal \|      |
| Carbs: 58g \| Fat: 12g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Fish-Free\] \[Shellfish-Free\]       |
| \[Tree Nut-Free\] \[Peanut-Free\]                                    |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[DF\] \[Low-Sugar / Diabetic-Friendly\] |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Carb\] \[High Folate\]   |
| \[High Iron\] \[Vitamin C\] \[High Potassium\] \[High Zinc\]         |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains eggs, soy, sesame. Use tamari for GF;      |
| coconut aminos for SoyF; omit sesame oil for SesF. Edamame is a      |
| complete plant protein; cold rice has resistant starch that slows    |
| digestion beneficially for recovery.*                                |
