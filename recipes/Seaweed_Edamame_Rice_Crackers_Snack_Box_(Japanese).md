+----------------------------------------------------------------------+
| **\#123 Seaweed & Edamame Rice Crackers Snack Box (Japanese)** \|    |
| Japanese · Mid-Day Snacks                                            |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • ½ cup shelled edamame (thawed)                                   |
| >                                                                    |
| > • 1 oz roasted seaweed snack sheets (nori)                         |
| >                                                                    |
| > • 1 oz brown rice crackers (GF)                                    |
| >                                                                    |
| > • 1 tbsp tamari dipping sauce (or low-sodium soy sauce)            |
| >                                                                    |
| > • ½ tsp sesame seeds (omit for SesF)                               |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Arrange edamame, nori sheets, and rice crackers in a snack     |
| > container.                                                         |
| >                                                                    |
| > 2\. Pack tamari in a small dipping container.                      |
| >                                                                    |
| > 3\. Eat nori wrapped around edamame or use crackers for dipping.   |
| >                                                                    |
| > 4\. Seaweed is one of the only plant sources of iodine ---         |
| > essential for thyroid function in growing athletes.                |
| >                                                                    |
| > 5\. The entire snack is under 200 calories but delivers complete   |
| > plant protein from edamame.                                        |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 10g \| Calories: 190 kcal \|      |
| Carbs: 22g \| Fat: 6g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Fish-Free\]             |
| \[Shellfish-Free\] \[Tree Nut-Free\] \[Peanut-Free\] \[Gluten-Free\] |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Vegan\] \[GF\] \[DF\] \[Low-Sugar /    |
| Diabetic-Friendly\]                                                  |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Iron\] \[High Zinc\]     |
| \[High Folate\] \[High Potassium\] \[High Magnesium\] \[Vitamin C\]  |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains soy (edamame, tamari) and sesame           |
| (optional). Use coconut aminos for SoyF; omit sesame for SesF. Nori  |
| (seaweed) is the richest plant source of B12 --- making it           |
| particularly valuable for vegetarian and vegan athletes.*            |
