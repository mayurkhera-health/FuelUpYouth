+----------------------------------------------------------------------+
| **\#66 Miso-Glazed Salmon with Bok Choy & Jasmine Rice** \| Japanese |
| · Dinner                                                             |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 salmon fillets (5 oz each)                                     |
| >                                                                    |
| > • 2 cups jasmine rice, cooked                                      |
| >                                                                    |
| > • 4 baby bok choy, halved lengthwise                               |
| >                                                                    |
| > • 2 tbsp white miso paste                                          |
| >                                                                    |
| > • 1 tbsp honey or maple syrup                                      |
| >                                                                    |
| > • 1 tbsp low-sodium soy sauce (or tamari for GF)                   |
| >                                                                    |
| > • 1 tbsp rice vinegar                                              |
| >                                                                    |
| > • 1 tsp sesame oil (omit for SesF)                                 |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 1 tbsp avocado oil                                               |
| >                                                                    |
| > • 1 tbsp sesame seeds, to garnish (omit for SesF)                  |
| >                                                                    |
| > • 2 green onions, sliced                                           |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven broiler to high. Line a sheet pan with foil.      |
| >                                                                    |
| > 2\. Whisk miso, honey, soy sauce, rice vinegar, sesame oil, and    |
| > ginger into a smooth glaze.                                        |
| >                                                                    |
| > 3\. Pat salmon dry; spread glaze over the top of each fillet.      |
| >                                                                    |
| > 4\. Broil salmon 8--10 min until caramelized and cooked through.   |
| > Watch closely --- glaze can burn quickly.                          |
| >                                                                    |
| > 5\. While salmon broils, heat avocado oil in a skillet over        |
| > medium-high.                                                       |
| >                                                                    |
| > 6\. Add bok choy cut-side-down; cook 2--3 min until charred. Flip  |
| > and cook 1 min more. Season lightly with soy sauce.                |
| >                                                                    |
| > 7\. Plate rice, top with salmon and bok choy; garnish with sesame  |
| > seeds and green onions.                                            |
+----------------------------------------------------------------------+

| Carbs: 44g \| Fat: 18g                                               |

| **NUTRIENT FOCUS:** \[High Protein\] \[Omega-3\] \[Vitamin D\]       |

| **TIP / SWAP:** *Contains soy (miso/soy sauce) and sesame. Use       |
| tamari for GF; use coconut aminos for SoyF; omit sesame for SesF.    |
| Miso provides probiotics; salmon delivers Omega-3 and Vitamin D      |
| essential for bone density in teen athletes.*                        |
