+----------------------------------------------------------------------+
| **\#121 Hard-Boiled Eggs & Cheese Snack Box (American)** \| American |
| · Mid-Day Snacks                                                     |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 12 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 large eggs                                                     |
| >                                                                    |
| > • 2 oz sharp cheddar or string cheese                              |
| >                                                                    |
| > • 1 cup grape tomatoes                                             |
| >                                                                    |
| > • ½ cup baby carrots                                               |
| >                                                                    |
| > • 1 tbsp Everything Bagel Seasoning (omit for SesF)                |
| >                                                                    |
| > • Salt and pepper to taste                                         |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Place eggs in cold water; bring to boil; cook 10--12 min;      |
| > transfer to ice bath.                                              |
| >                                                                    |
| > 2\. Peel eggs; sprinkle with Everything Bagel Seasoning.           |
| >                                                                    |
| > 3\. Pack eggs, cheese, tomatoes, and carrots in a divided          |
| > container.                                                         |
| >                                                                    |
| > 4\. Refrigerate up to 3 days for weekly meal prep.                 |
| >                                                                    |
| > 5\. Hard-boiled eggs deliver all 9 essential amino acids in one of |
| > the highest protein-per-calorie ratios of any whole food.          |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 22g \| Calories: 280 kcal \|      |
| Carbs: 10g \| Fat: 18g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Fish-Free\] \[Shellfish-Free\] \[Tree Nut-Free\]    |
| \[Peanut-Free\] \[Gluten-Free\] \[Soy-Free\]                         |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[GF\] \[SoyF\] \[Low-Sugar /            |
| Diabetic-Friendly\]                                                  |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[Vitamin D\] \[High Calcium\]  |
| \[B12-Rich\] \[Vitamin C\] \[High Zinc\] \[High Potassium\]          |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains eggs, dairy (cheese), sesame (Everything   |
| Bagel Seasoning). Omit seasoning for SesF; use DF cheese for DF.     |
| Eggs are the gold standard reference protein for bioavailability --- |
| rated 100 on the Biological Value scale, higher than meat, fish, or  |
| any plant protein.*                                                  |
