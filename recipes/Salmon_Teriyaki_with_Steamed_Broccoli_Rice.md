+----------------------------------------------------------------------+
| **\#43 Salmon Teriyaki with Steamed Broccoli & Rice** \| Japanese ·  |
| Lunch                                                                |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 18 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 salmon fillets (5 oz / 140 g each), skin-on                    |
| >                                                                    |
| > • 3 tbsp low-sodium soy sauce or tamari                            |
| >                                                                    |
| > • 1½ tbsp honey                                                    |
| >                                                                    |
| > • 1 tsp rice vinegar                                               |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 1 tsp cornstarch dissolved in 1 tsp water                        |
| >                                                                    |
| > • 1½ cups cooked brown rice                                        |
| >                                                                    |
| > • 2 cups broccoli florets, steamed                                 |
| >                                                                    |
| > • 1 tsp sesame seeds (omit for SesF)                               |
| >                                                                    |
| > • Lemon wedge to serve                                             |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Whisk soy sauce, honey, vinegar, ginger, and cornstarch slurry |
| > for teriyaki sauce.                                                |
| >                                                                    |
| > 2\. Heat a non-stick pan over medium-high. Cook salmon skin-side   |
| > down 5--6 min. Flip, cook 3 more min.                              |
| >                                                                    |
| > 3\. Pour sauce over salmon; simmer 2 min, basting frequently until |
| > sauce thickens.                                                    |
| >                                                                    |
| > 4\. Steam broccoli until bright green and tender-crisp, 5--6 min.  |
| >                                                                    |
| > 5\. Serve salmon over brown rice with broccoli. Sprinkle sesame    |
| > seeds and squeeze lemon.                                           |
