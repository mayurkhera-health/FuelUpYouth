+----------------------------------------------------------------------+
| **\#77 White Rice & Grilled Chicken with Light Teriyaki** \|         |
| Japanese · Pre-Game Fueling                                          |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 15 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs boneless skinless chicken breast                         |
| >                                                                    |
| > • 2½ cups white jasmine rice, cooked (white rice preferred over    |
| > brown pre-game)                                                    |
| >                                                                    |
| > • Light teriyaki: 3 tbsp low-sodium soy sauce (or coconut aminos   |
| > for SoyF)                                                          |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • 1 tsp rice vinegar                                               |
| >                                                                    |
| > • ½ tsp cornstarch                                                 |
| >                                                                    |
| > • 1 tbsp avocado oil                                               |
| >                                                                    |
| > • Steamed edamame or steamed broccoli (small portion --- limit     |
| > fiber pre-game)                                                    |
| >                                                                    |
| > • Pickled ginger (optional)                                        |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Mix soy sauce, honey, vinegar, and cornstarch into a simple    |
| > teriyaki sauce.                                                    |
| >                                                                    |
| > 2\. Grill or pan-cook chicken breast 5--6 min per side until       |
| > cooked through.                                                    |
| >                                                                    |
| > 3\. Brush lightly with teriyaki sauce in last 2 min of cooking.    |
| >                                                                    |
| > 4\. Slice chicken; serve over generous portion of white rice with  |
| > a small side of steamed veg.                                       |
| >                                                                    |
| > 5\. Keep the meal simple, light, and carb-forward --- white rice   |
| > is the star pre-game.                                              |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 38g \| Calories: 490 kcal \|      |
| Carbs: 60g \| Fat: 8g                                                |

| \[Shellfish-Free\] \[Tree Nut-Free\] \[Peanut-Free\] \[Gluten-Free\] |
| \[Sesame-Free\] \[WF\]                                               |
|                                                                      |
| **DIETARY:** \[GF\] \[DF\] \[Low-Sugar / Diabetic-Friendly\]         |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Carb\] \[B12-Rich\]      |
| \[High Potassium\] \[High Zinc\]                                     |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains soy. Use coconut aminos for SoyF. Use      |
| tamari for GF. White rice over brown rice pre-game --- faster        |
| digestion, less GI stress. Eat 2--3 hours before competition. Avoid  |
| heavy sauces.*                                                       |
