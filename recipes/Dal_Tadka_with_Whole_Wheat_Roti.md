+----------------------------------------------------------------------+
| **\#29 Dal Tadka with Whole Wheat Roti** \| Indian · Lunch           |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 25 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • ¾ cup split red lentils (masoor dal), rinsed                     |
| >                                                                    |
| > • 2½ cups water                                                    |
| >                                                                    |
| > • 1 medium tomato, diced                                           |
| >                                                                    |
| > • ½ small onion, diced                                             |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • ½ tsp turmeric                                                   |
| >                                                                    |
| > • ½ tsp cumin seeds                                                |
| >                                                                    |
| > • ½ tsp coriander powder                                           |
| >                                                                    |
| > • 1 tbsp ghee or avocado oil for tadka                             |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • 4 small whole wheat rotis (store-bought or homemade)             |
| >                                                                    |
| > • Fresh coriander and lime wedge to serve                          |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Boil lentils in water with turmeric and a pinch of salt for    |
| > 15--20 min until completely soft. Mash lightly.                    |
| >                                                                    |
| > 2\. In a separate pan, heat ghee. Add cumin seeds; once they       |
| > sizzle, add onion and cook 3 min.                                  |
| >                                                                    |
| > 3\. Add garlic, ginger, and tomato; cook 4 min until mashy. Stir   |
| > in coriander powder.                                               |
| >                                                                    |
| > 4\. Pour the tadka (tempering) into the lentils and stir. Simmer 3 |
| > min.                                                               |
| >                                                                    |
| > 5\. Warm rotis on a dry griddle. Serve dal with rotis, a squeeze   |
| > of lime, and coriander.                                            |
