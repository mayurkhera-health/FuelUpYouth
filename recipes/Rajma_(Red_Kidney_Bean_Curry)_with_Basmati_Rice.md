+----------------------------------------------------------------------+
| **\#68 Rajma (Red Kidney Bean Curry) with Basmati Rice** \| Indian · |
| Dinner                                                               |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 30 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cans (15 oz each) red kidney beans, drained and rinsed         |
| >                                                                    |
| > • 1½ cups basmati rice, cooked                                     |
| >                                                                    |
| > • 1 can (14 oz) crushed tomatoes                                   |
| >                                                                    |
| > • 1 medium onion, finely diced                                     |
| >                                                                    |
| > • 4 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tbsp fresh ginger, grated                                      |
| >                                                                    |
| > • 2 tbsp avocado oil                                               |
| >                                                                    |
| > • 1 tsp cumin seeds                                                |
| >                                                                    |
| > • 1 tsp turmeric                                                   |
| >                                                                    |
| > • 1 tsp coriander powder                                           |
| >                                                                    |
| > • 1 tsp garam masala                                               |
| >                                                                    |
| > • ½ tsp chili powder                                               |
| >                                                                    |
| > • 1 cup water                                                      |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • Fresh cilantro and lemon wedges to serve                         |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil in a heavy pot over medium; add cumin seeds and       |
| > sizzle 30 sec.                                                     |
| >                                                                    |
| > 2\. Add onion; cook until golden brown, \~10 min.                  |
| >                                                                    |
| > 3\. Add garlic and ginger; cook 1 min. Add all dry spices; toast   |
| > 30 sec.                                                            |
| >                                                                    |
| > 4\. Pour in crushed tomatoes; cook 5--7 min until oil separates.   |
| >                                                                    |
| > 5\. Add kidney beans and water; stir well. Bring to a boil, then   |
| > reduce heat.                                                       |
| >                                                                    |
| > 6\. Simmer 15--20 min. Use the back of a spoon to mash some beans  |
| > --- this thickens the curry naturally.                             |
| >                                                                    |
| > 7\. Adjust seasoning; serve over basmati rice with cilantro and a  |
| > squeeze of lemon.                                                  |
