+----------------------------------------------------------------------+
| **\#46 Aloo Gobi Wrap with Mint Yogurt** \| Indian · Lunch           |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 medium potatoes, diced small                                   |
| >                                                                    |
| > • 2 cups cauliflower florets, chopped                              |
| >                                                                    |
| > • 1 small onion, sliced                                            |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • ½ tsp turmeric                                                   |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • ½ tsp garam masala                                               |
| >                                                                    |
| > • 1 tbsp avocado oil                                               |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • 4 small whole wheat rotis or wraps                               |
| >                                                                    |
| > • ¼ cup plain low-fat yogurt + 2 tbsp chopped fresh mint + pinch   |
| > of cumin (dip)                                                     |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil in a skillet. Add cumin seeds, then onion; cook 4     |
| > min.                                                               |
| >                                                                    |
| > 2\. Add garlic, potato, and cauliflower. Season with turmeric,     |
| > cumin, and salt.                                                   |
| >                                                                    |
| > 3\. Cover and cook 12--15 min, stirring occasionally, until potato |
| > is tender.                                                         |
| >                                                                    |
| > 4\. Add garam masala, stir, and remove from heat.                  |
| >                                                                    |
| > 5\. Mix yogurt with mint and cumin for dip. Fill rotis with aloo   |
| > gobi, roll up, and serve with mint yogurt.                         |
