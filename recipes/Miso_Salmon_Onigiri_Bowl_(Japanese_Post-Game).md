+----------------------------------------------------------------------+
| **\#108 Miso Salmon Onigiri Bowl (Japanese Post-Game)** \| Japanese  |
| · Post-Game Recovery                                                 |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 15 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 salmon fillets (6 oz each)                                     |
| >                                                                    |
| > • 2 tbsp white miso paste                                          |
| >                                                                    |
| > • 1 tbsp mirin (or honey + rice vinegar)                           |
| >                                                                    |
| > • 1 tbsp low-sodium soy sauce (or tamari for GF)                   |
| >                                                                    |
| > • 1½ cups cooked Japanese short-grain rice                         |
| >                                                                    |
| > • 1 cup edamame (shelled, thawed)                                  |
| >                                                                    |
| > • ½ cup shredded purple cabbage                                    |
| >                                                                    |
| > • 1 tsp sesame seeds (omit for SesF)                               |
| >                                                                    |
| > • 2 sheets nori (seaweed), cut into strips                         |
| >                                                                    |
| > • 1 tsp rice vinegar                                               |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Whisk miso, mirin, and soy sauce into a marinade; coat salmon  |
| > and let sit 10 min.                                                |
| >                                                                    |
| > 2\. Broil or pan-fry salmon 5--6 min per side until cooked through |
| > and caramelized.                                                   |
| >                                                                    |
| > 3\. Flake salmon over rice in bowls.                               |
| >                                                                    |
| > 4\. Top with edamame, cabbage, nori strips, and sesame seeds.      |
| >                                                                    |
| > 5\. Drizzle with rice vinegar; serve within 45 min of game for     |
| > peak glycogen restoration.                                         |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 44g \| Calories: 510 kcal \|      |
| Carbs: 46g \| Fat: 18g                                               |

| Nut-Free\] \[Peanut-Free\] \[Gluten-Free\]                           |
|                                                                      |
| **DIETARY:** \[GF\] \[DF\] \[Low-Sugar / Diabetic-Friendly\]         |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[Omega-3\] \[Vitamin D\]       |
| \[B12-Rich\] \[High Potassium\] \[High Iron\] \[High Magnesium\]     |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains fish, soy (miso), sesame. Use tamari for   |
| GF; omit sesame for SesF; use coconut aminos for SoyF. Miso is       |
| fermented --- supports gut microbiome recovery; salmon\'s EPA/DHA    |
| fats reduce muscle inflammation within 24 hours.*                    |
