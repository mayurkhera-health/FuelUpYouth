+----------------------------------------------------------------------+
| **\#64 Sheet Pan Shrimp Fajitas with Guacamole** \| Mexican · Dinner |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 15 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs large shrimp, peeled and deveined                        |
| >                                                                    |
| > • 8 small flour tortillas (or corn for GF/WF)                      |
| >                                                                    |
| > • 3 bell peppers (red, yellow, green), sliced                      |
| >                                                                    |
| > • 1 large onion, sliced into strips                                |
| >                                                                    |
| > • 2 tbsp avocado oil                                               |
| >                                                                    |
| > • 2 tsp chili powder                                               |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • 1 tsp garlic powder                                              |
| >                                                                    |
| > • ½ tsp smoked paprika                                             |
| >                                                                    |
| > • Juice of 1 lime                                                  |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • Guacamole: 2 avocados mashed with lime juice, salt, ½ diced      |
| > tomato, cilantro                                                   |
| >                                                                    |
| > • Optional: plain Greek yogurt (or coconut yogurt for DF) as sour  |
| > cream substitute                                                   |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 425°F. Line a large sheet pan with parchment   |
| > paper.                                                             |
| >                                                                    |
| > 2\. Toss bell peppers and onion with 1 tbsp oil, chili powder,     |
| > cumin, garlic powder, paprika, and salt.                           |
| >                                                                    |
| > 3\. Spread on sheet pan; roast 10 min until starting to soften.    |
| >                                                                    |
| > 4\. Toss shrimp with remaining 1 tbsp oil, lime juice, and a pinch |
| > of all the same spices.                                            |
| >                                                                    |
| > 5\. Add shrimp to the sheet pan; roast 5--7 more min until shrimp  |
| > are pink and opaque.                                               |
| >                                                                    |
| > 6\. While shrimp cook, make guacamole.                             |
| >                                                                    |
| > 7\. Warm tortillas; serve fajitas loaded with shrimp, peppers,     |
| > onions, and guacamole.                                             |
