+----------------------------------------------------------------------+
| **\#74 Honey Garlic Baked Salmon with Quinoa & Broccolini** \|       |
| American · Dinner                                                    |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 salmon fillets (5 oz each)                                     |
| >                                                                    |
| > • 1½ cups quinoa, rinsed and cooked                                |
| >                                                                    |
| > • 1 lb broccolini (or broccoli florets), trimmed                   |
| >                                                                    |
| > • 3 cloves garlic, minced                                          |
| >                                                                    |
| > • 2 tbsp honey                                                     |
| >                                                                    |
| > • 2 tbsp low-sodium soy sauce (or coconut aminos for SoyF)         |
| >                                                                    |
| > • 1 tbsp apple cider vinegar                                       |
| >                                                                    |
| > • 1 tbsp olive oil                                                 |
| >                                                                    |
| > • ½ tsp smoked paprika                                             |
| >                                                                    |
| > • Lemon wedges and fresh dill to serve                             |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 400°F. Line a sheet pan with parchment.        |
| >                                                                    |
| > 2\. Whisk garlic, honey, soy sauce, vinegar, and paprika into a    |
| > glaze.                                                             |
| >                                                                    |
| > 3\. Place salmon fillets on one side of the pan; spoon glaze over  |
| > each fillet.                                                       |
| >                                                                    |
| > 4\. Toss broccolini with olive oil, salt, and pepper; arrange      |
| > alongside salmon.                                                  |
| >                                                                    |
| > 5\. Bake 15--18 min until salmon is cooked through and broccolini  |
| > is tender with slightly charred tips.                              |
| >                                                                    |
| > 6\. Serve over quinoa with lemon wedges and fresh dill.            |
