+----------------------------------------------------------------------+
| **\#23 Idli with Sambar & Coconut Chutney** \| Indian · Breakfast    |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 8 store-bought or homemade idli (fermented rice & lentil cakes)  |
| >                                                                    |
| > • 1½ cups ready-made low-sodium sambar (lentil vegetable stew)     |
| >                                                                    |
| > • ½ cup fresh coconut chutney (blend: ½ cup grated coconut, 1      |
| > green chilli, 1 tsp ginger, salt, water)                           |
| >                                                                    |
| > • ½ tsp mustard seeds + curry leaves for chutney tempering         |
| >                                                                    |
| > • 1 tsp oil for tempering                                          |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Steam idli for 8--10 min in an idli steamer or over a pot of   |
| > boiling water using a colander.                                    |
| >                                                                    |
| > 2\. Heat sambar gently in a saucepan, adding a splash of water to  |
| > thin if needed.                                                    |
| >                                                                    |
| > 3\. For chutney: blend all ingredients until smooth. Heat oil, pop |
| > mustard seeds with curry leaves, pour over chutney.                |
| >                                                                    |
| > 4\. Arrange 4 idli per plate. Serve with a bowl of hot sambar and  |
| > chutney on the side.                                               |
