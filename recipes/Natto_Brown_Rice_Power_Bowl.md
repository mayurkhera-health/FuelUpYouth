+----------------------------------------------------------------------+
| **\#21 Natto & Brown Rice Power Bowl** \| Japanese · Breakfast       |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 25 min \| 🍽 Serves: 1                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 pack natto (fermented soybeans, \~45 g --- available at Asian  |
| > grocery stores)                                                    |
| >                                                                    |
| > • 1 cup cooked short-grain brown rice                              |
| >                                                                    |
| > • 1 tsp low-sodium soy sauce or tamari                             |
| >                                                                    |
| > • 1 tsp Japanese mustard (optional)                                |
| >                                                                    |
| > • 1 scallion, finely sliced                                        |
| >                                                                    |
| > • 1 sheet nori, cut into thin strips                               |
| >                                                                    |
| > • 1 tsp rice vinegar                                               |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Cook brown rice according to package instructions.             |
| >                                                                    |
| > 2\. Open natto pack; add the included sauce packet, soy sauce, and |
| > mustard. Stir vigorously until sticky strings form.                |
| >                                                                    |
| > 3\. Spoon warm rice into a bowl. Place natto on top.               |
| >                                                                    |
| > 4\. Scatter scallion and nori strips over the bowl. Drizzle rice   |
| > vinegar.                                                           |
| >                                                                    |
| > 5\. Mix gently before eating. Pair with a small bowl of miso soup  |
| > if desired.                                                        |
