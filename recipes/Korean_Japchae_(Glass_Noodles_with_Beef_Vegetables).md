+----------------------------------------------------------------------+
| **\#35 Korean Japchae (Glass Noodles with Beef & Vegetables)** \|    |
| Korean · Lunch                                                       |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 20 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 oz (115 g) dried sweet potato glass noodles (dangmyeon)        |
| >                                                                    |
| > • 5 oz (140 g) lean beef sirloin, thinly sliced                    |
| >                                                                    |
| > • 1 cup baby spinach                                               |
| >                                                                    |
| > • 1 medium carrot, julienned                                       |
| >                                                                    |
| > • ½ red bell pepper, thinly sliced                                 |
| >                                                                    |
| > • 3 dried shiitake mushrooms, soaked and sliced                    |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 2 tbsp low-sodium soy sauce or tamari                            |
| >                                                                    |
| > • 1 tsp sesame oil (omit for SesF)                                 |
| >                                                                    |
| > • 1 tsp honey                                                      |
| >                                                                    |
| > • 1 tsp avocado oil                                                |
| >                                                                    |
| > • 1 scallion, sliced                                               |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Soak glass noodles in warm water 20 min; cook in boiling water |
| > 5 min. Drain and snip into shorter lengths.                        |
| >                                                                    |
| > 2\. Mix soy sauce, honey, and sesame oil for sauce.                |
| >                                                                    |
| > 3\. Stir-fry beef in oil over high heat 3 min. Remove. Stir-fry    |
| > carrot and bell pepper 3 min; add spinach and mushrooms.           |
| >                                                                    |
| > 4\. Add noodles and beef back to pan. Pour sauce over and toss     |
| > everything together 2 min.                                         |
| >                                                                    |
| > 5\. Garnish with scallion and serve warm or at room temperature.   |

| Carbs: 60g \| Fat: 11g                                               |

| cold the next day --- perfect packed lunch.*                         |
