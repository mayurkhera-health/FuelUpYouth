+----------------------------------------------------------------------+
| **\#57 Dal Makhani with Brown Rice** \| Indian · Dinner              |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 45 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 cup whole black lentils (urad dal), soaked overnight and       |
| > drained                                                            |
| >                                                                    |
| > • ¼ cup kidney beans, canned (rinsed and drained)                  |
| >                                                                    |
| > • 2 cups brown rice, cooked                                        |
| >                                                                    |
| > • 1 can (14 oz) diced tomatoes                                     |
| >                                                                    |
| > • ½ cup coconut milk                                               |
| >                                                                    |
| > • 1 medium onion, finely diced                                     |
| >                                                                    |
| > • 4 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tbsp fresh ginger, grated                                      |
| >                                                                    |
| > • 2 tbsp ghee or avocado oil                                       |
| >                                                                    |
| > • 1 tsp cumin seeds                                                |
| >                                                                    |
| > • 1 tsp garam masala                                               |
| >                                                                    |
| > • 1 tsp coriander powder                                           |
| >                                                                    |
| > • ½ tsp smoked paprika                                             |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • Fresh cilantro and lemon wedges to serve                         |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Pressure cook or boil soaked black lentils until very soft     |
| > (\~30--35 min). Drain and set aside.                               |
| >                                                                    |
| > 2\. Heat ghee in a heavy pot; add cumin seeds and sizzle 30        |
| > seconds.                                                           |
| >                                                                    |
| > 3\. Add onion; cook until deep golden (\~10 min). Add garlic and   |
| > ginger; cook 1 min.                                                |
| >                                                                    |
| > 4\. Stir in diced tomatoes and all spices; cook down 5--7 min      |
| > until oil separates.                                               |
| >                                                                    |
| > 5\. Add cooked lentils and kidney beans; stir well to combine.     |
| >                                                                    |
| > 6\. Pour in coconut milk; simmer on low 15--20 min, stirring       |
| > occasionally, until creamy.                                        |
| >                                                                    |
| > 7\. Adjust seasoning; serve over brown rice with cilantro and a    |
| > squeeze of lemon.                                                  |
