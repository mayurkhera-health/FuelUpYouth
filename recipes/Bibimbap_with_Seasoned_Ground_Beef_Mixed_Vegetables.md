+----------------------------------------------------------------------+
| **\#62 Bibimbap with Seasoned Ground Beef & Mixed Vegetables** \|    |
| Korean · Dinner                                                      |
+======================================================================+
| ⏱ Prep: 20 min \| 🍳 Cook: 20 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 lb lean ground beef                                            |
| >                                                                    |
| > • 2 cups short-grain white rice, cooked                            |
| >                                                                    |
| > • 2 cups baby spinach                                              |
| >                                                                    |
| > • 1 cup bean sprouts                                               |
| >                                                                    |
| > • 1 large carrot, julienned                                        |
| >                                                                    |
| > • 1 zucchini, julienned                                            |
| >                                                                    |
| > • 4 eggs (omit for EF)                                             |
| >                                                                    |
| > • 2 tbsp avocado oil                                               |
| >                                                                    |
| > • Gochujang sauce (serve on side; omit for low-spice): 2 tbsp      |
| > gochujang paste                                                    |
| >                                                                    |
| > • 1 tbsp sesame oil (omit for SesF)                                |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • 1 tsp rice vinegar                                               |
| >                                                                    |
| > • Beef seasoning: 2 tbsp low-sodium soy sauce, 1 tsp sesame oil, 1 |
| > tsp garlic, minced                                                 |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Season ground beef with soy sauce, sesame oil, and garlic;     |
| > cook in a skillet over medium-high until browned, \~7 min. Set     |
| > aside.                                                             |
| >                                                                    |
| > 2\. Sauté spinach in 1 tsp oil with a pinch of salt until wilted,  |
| > 1 min. Set aside.                                                  |
| >                                                                    |
| > 3\. Blanch bean sprouts in boiling water 1 min; drain and season   |
| > with a drop of sesame oil and salt.                                |
| >                                                                    |
| > 4\. Sauté carrot and zucchini separately in oil, each 2--3 min     |
| > until tender. Season with salt.                                    |
| >                                                                    |
| > 5\. Fry eggs sunny-side-up in remaining oil.                       |
| >                                                                    |
| > 6\. Arrange rice in bowls; top with sections of each vegetable and |
| > the beef.                                                          |
| >                                                                    |
| > 7\. Place fried egg on top. Drizzle gochujang sauce over and mix   |
| > everything together before eating.                                 |
