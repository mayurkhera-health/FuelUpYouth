+----------------------------------------------------------------------+
| **\#128 Tuna Salad Lettuce Cups (American)** \| American · Mid-Day   |
| Snacks                                                               |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cans (5 oz each) tuna in water, drained                        |
| >                                                                    |
| > • 2 tbsp plain Greek yogurt or light mayo                          |
| >                                                                    |
| > • 1 celery stalk, finely diced                                     |
| >                                                                    |
| > • 2 tbsp red onion, minced                                         |
| >                                                                    |
| > • 1 tsp lemon juice                                                |
| >                                                                    |
| > • Salt, pepper, and dill to taste                                  |
| >                                                                    |
| > • 6 large romaine or butter lettuce leaves (for cups)              |
| >                                                                    |
| > • Optional: sliced cucumber inside cups                            |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Mix tuna with yogurt/mayo, celery, onion, lemon juice, dill,   |
| > salt, and pepper.                                                  |
| >                                                                    |
| > 2\. Spoon 2--3 tbsp tuna salad into each lettuce cup.              |
| >                                                                    |
| > 3\. Add cucumber slices if using.                                  |
| >                                                                    |
| > 4\. Pack lettuce and filling separately to keep crisp; assemble    |
| > just before eating.                                                |
| >                                                                    |
| > 5\. Using Greek yogurt instead of mayo adds extra protein and cuts |
| > calories by 40%.                                                   |

| Carbs: 6g \| Fat: 4g                                                 |
