+----------------------------------------------------------------------+
| **\#119 Mini Cheese & Bean Quesadilla Triangles (Mexican)** \|       |
| Mexican · Mid-Day Snacks                                             |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 10 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 small whole wheat or corn tortillas (6-inch)                   |
| >                                                                    |
| > • ¼ cup canned black beans, drained and rinsed                     |
| >                                                                    |
| > • ½ cup shredded Monterey Jack or cheddar cheese                   |
| >                                                                    |
| > • 2 tbsp salsa (no added sugar)                                    |
| >                                                                    |
| > • Optional: 1 tbsp Greek yogurt as sour cream substitute           |
| >                                                                    |
| > • Optional: sliced avocado                                         |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Sprinkle cheese and black beans on one half of each tortilla.  |
| >                                                                    |
| > 2\. Fold over; cook in a dry skillet over medium heat 2--3 min per |
| > side until golden.                                                 |
| >                                                                    |
| > 3\. Let cool 2--3 min; cut into triangles.                         |
| >                                                                    |
| > 4\. Pack with salsa for dipping; add Greek yogurt and avocado if   |
| > desired.                                                           |
| >                                                                    |
| > 5\. Can be eaten warm or room temperature --- ideal for practice   |
| > bag.                                                               |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 16g \| Calories: 300 kcal \|      |
| Carbs: 32g \| Fat: 12g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Fish-Free\] \[Shellfish-Free\] \[Tree  |
| Nut-Free\] \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]              |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[SoyF\] \[SesF\] \[Low-Sugar /          |
| Diabetic-Friendly\]                                                  |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Calcium\] \[High Protein\] \[High Iron\]  |
| \[High Folate\] \[High Potassium\] \[Vitamin C\]                     |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains dairy (cheese) and gluten (flour           |
| tortilla). Use corn tortillas for GF; use dairy-free cheese for DF.  |
| Cheese provides calcium + casein protein; black beans add iron and   |
| fiber. One of the most popular after-school snacks for teen          |
| athletes.*                                                           |
