+----------------------------------------------------------------------+
| **\#7 Smoked Salmon Bagel with Cream Cheese & Cucumber** \| American |
| · Breakfast                                                          |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 1                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 whole wheat or gluten-free bagel                               |
| >                                                                    |
| > • 2 oz (55 g) cream cheese                                         |
| >                                                                    |
| > • 2 oz (55 g) smoked salmon                                        |
| >                                                                    |
| > • 4 thin cucumber slices                                           |
| >                                                                    |
| > • 1 tsp capers (optional)                                          |
| >                                                                    |
| > • Fresh dill or chives                                             |
| >                                                                    |
| > • Lemon wedge for serving                                          |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Slice and toast the bagel.                                     |
| >                                                                    |
| > 2\. Spread cream cheese generously on each half.                   |
| >                                                                    |
| > 3\. Layer smoked salmon over the cream cheese.                     |
| >                                                                    |
| > 4\. Top with cucumber, capers, and dill. Squeeze lemon over and    |
| > serve.                                                             |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 24g \| Calories: 430 kcal \|      |
| Carbs: 46g \| Fat: 14g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Shellfish-Free\] \[Tree Nut-Free\]     |
| \[Peanut-Free\] \[Sesame-Free\] \[Soy-Free\]                         |
|                                                                      |
| **DIETARY:** \[HighPro\] \[Low-Sodium\]                              |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[Omega-3\] \[Vitamin D\]       |
| \[B12-Rich\] \[High Calcium\]                                        |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains fish and dairy. Use GF bagel for           |
| Gluten-Free. Use dairy-free cream cheese spread for DF. Capers are   |
| high in sodium --- omit for strict low-sodium.*                      |
