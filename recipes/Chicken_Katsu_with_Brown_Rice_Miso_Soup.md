+----------------------------------------------------------------------+
| **\#54 Chicken Katsu with Brown Rice & Miso Soup** \| Japanese ·     |
| Dinner                                                               |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 20 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 boneless chicken breasts (5 oz each), pounded to ½-inch thick  |
| >                                                                    |
| > • 1 cup panko breadcrumbs (GF panko for GF)                        |
| >                                                                    |
| > • 2 eggs, beaten (or aquafaba for EF)                              |
| >                                                                    |
| > • ¼ cup all-purpose flour (or rice flour for GF/WF)                |
| >                                                                    |
| > • 1 tsp garlic powder                                              |
| >                                                                    |
| > • ½ tsp salt                                                       |
| >                                                                    |
| > • 2 tbsp avocado oil                                               |
| >                                                                    |
| > • 2 cups brown rice, cooked                                        |
| >                                                                    |
| > • 3 cups water                                                     |
| >                                                                    |
| > • 2 tsp white miso paste                                           |
| >                                                                    |
| > • 1 cup silken tofu, cubed                                         |
| >                                                                    |
| > • 2 green onions, sliced                                           |
| >                                                                    |
| > • 1 sheet dried nori, cut into strips                              |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Season chicken with garlic powder and salt.                    |
| >                                                                    |
| > 2\. Set up breading station: flour in one bowl, eggs in second,    |
| > panko in third.                                                    |
| >                                                                    |
| > 3\. Dredge chicken in flour, dip in egg, then coat thoroughly with |
| > panko.                                                             |
| >                                                                    |
| > 4\. Heat oil in a large oven-safe skillet over medium-high. Sear   |
| > chicken 3--4 min per side until golden.                            |
| >                                                                    |
| > 5\. Transfer to 375°F oven and bake 10--12 min until internal temp |
| > reaches 165°F.                                                     |
| >                                                                    |
| > 6\. For miso soup: heat water until just simmering. Whisk in miso  |
| > paste (do not boil).                                               |
| >                                                                    |
| > 7\. Add tofu and green onions; simmer 2 min. Stir in nori strips.  |
| >                                                                    |
| > 8\. Slice chicken katsu; serve over brown rice with miso soup on   |
| > the side.                                                          |
