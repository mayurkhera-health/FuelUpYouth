+----------------------------------------------------------------------+
| **\#22 Chinese Steamed Egg Custard (Zheng Shui Dan)** \| Chinese ·   |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 12 min \| 🍽 Serves: 1                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 3 large eggs                                                     |
| >                                                                    |
| > • ¾ cup low-sodium chicken or vegetable broth (warm, not boiling)  |
| >                                                                    |
| > • 1 tsp low-sodium soy sauce or tamari                             |
| >                                                                    |
| > • ½ tsp sesame oil (omit for SesF)                                 |
| >                                                                    |
| > • 1 scallion, sliced thin                                          |
| >                                                                    |
| > • Pinch of white pepper                                            |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Crack eggs into a bowl and beat gently --- avoid creating      |
| > foam.                                                              |
| >                                                                    |
| > 2\. Slowly pour warm broth into the eggs while stirring. Add white |
| > pepper.                                                            |
| >                                                                    |
| > 3\. Pour mixture through a fine sieve into a deep bowl or ramekin  |
| > to remove bubbles.                                                 |
| >                                                                    |
| > 4\. Cover tightly with cling wrap. Steam over medium-low heat for  |
| > 10--12 min until just set (should jiggle slightly in the centre).  |
| >                                                                    |
| > 5\. Drizzle with soy sauce and sesame oil; top with scallion.      |
| > Serve with steamed rice.                                           |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 19g \| Calories: 210 kcal \|      |
| Carbs: 3g \| Fat: 13g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Gluten-Free\] \[Dairy-Free\] \[Shellfish-Free\]     |
| \[Fish-Free\] \[Tree Nut-Free\] \[Peanut-Free\]                      |
|                                                                      |
| **DIETARY:** \[Low-Sugar / Diabetic-Friendly\] \[Low-Sodium\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[Vitamin D\] \[B12-Rich\]      |
| \[High Zinc\] \[High Iron\]                                          |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Use tamari for SoyF version; omit sesame oil for    |
| SesF. Silky texture makes this one of the easiest high-protein       |
| breakfasts. Add with rice to boost carbs to \~350 kcal.*             |
