+----------------------------------------------------------------------+
| **\#15 Besan Omelette (Chickpea Flour Pancake) --- Egg-Free** \|     |
| Indian · Breakfast                                                   |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 10 min \| 🍽 Serves: 1                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • ½ cup besan (chickpea flour)                                     |
| >                                                                    |
| > • ⅓ cup water (adjust for batter consistency)                      |
| >                                                                    |
| > • ¼ small onion, finely diced                                      |
| >                                                                    |
| > • ¼ small tomato, finely diced                                     |
| >                                                                    |
| > • 2 tbsp fresh spinach, chopped                                    |
| >                                                                    |
| > • ¼ tsp turmeric                                                   |
| >                                                                    |
| > • ¼ tsp cumin powder                                               |
| >                                                                    |
| > • Pinch of red chili flakes (optional)                             |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • 1 tsp olive or avocado oil                                       |
| >                                                                    |
| > • Fresh coriander to garnish                                       |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Whisk besan with water to make a smooth, lump-free batter      |
| > (slightly thinner than pancake batter).                            |
| >                                                                    |
| > 2\. Stir in onion, tomato, spinach, and all spices.                |
| >                                                                    |
| > 3\. Heat oil in a non-stick pan over medium. Pour batter and       |
| > spread into a round.                                               |
| >                                                                    |
| > 4\. Cook 3--4 min until edges firm up and bottom is golden. Flip   |
| > and cook 3 more min.                                               |
| >                                                                    |
| > 5\. Garnish with fresh coriander. Serve with a side of sliced      |
| > cucumber and tomato.                                               |
