+----------------------------------------------------------------------+
| **\#58 Carne Asada Tacos with Black Beans & Pico de Gallo** \|       |
| Mexican · Dinner                                                     |
+======================================================================+
| ⏱ Prep: 20 min \| 🍳 Cook: 15 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs skirt steak                                              |
| >                                                                    |
| > • 8 small corn tortillas (certified GF for GF)                     |
| >                                                                    |
| > • 1 can (15 oz) black beans, rinsed and drained                    |
| >                                                                    |
| > • 2 limes (juice)                                                  |
| >                                                                    |
| > • 1 orange (juice)                                                 |
| >                                                                    |
| > • 3 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • 1 tsp chili powder                                               |
| >                                                                    |
| > • 1 tbsp olive oil                                                 |
| >                                                                    |
| > • 3 roma tomatoes, diced                                           |
| >                                                                    |
| > • ½ white onion, finely diced                                      |
| >                                                                    |
| > • 1 jalapeño, seeded and minced                                    |
| >                                                                    |
| > • ½ cup fresh cilantro, chopped                                    |
| >                                                                    |
| > • Juice of 1 lime                                                  |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • 1 avocado, sliced                                                |
| >                                                                    |
| > • Crumbled cotija cheese (optional --- omit for DF)                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Marinate steak in lime juice, orange juice, garlic, cumin,     |
| > chili powder, and olive oil for at least 30 min.                   |
| >                                                                    |
| > 2\. Make pico de gallo: combine tomatoes, onion, jalapeño,         |
| > cilantro, lime juice, and salt; refrigerate.                       |
| >                                                                    |
| > 3\. Heat black beans in a small saucepan with a pinch of cumin and |
| > salt.                                                              |
| >                                                                    |
| > 4\. Heat a cast-iron skillet or grill over high heat. Cook steak   |
| > 3--4 min per side for medium-rare.                                 |
| >                                                                    |
| > 5\. Rest steak 5 min, then slice thinly against the grain.         |
| >                                                                    |
| > 6\. Warm tortillas on the skillet 30 seconds per side.             |
| >                                                                    |
| > 7\. Assemble tacos: tortilla, black beans, carne asada, pico de    |
| > gallo, avocado, and cotija (if using).                             |
