+----------------------------------------------------------------------+
| **\#95 White Flour Tortilla with Honey, Strawberries & Ricotta** \|  |
| American · Pre-Game Fueling                                          |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 2 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 small white flour tortillas (or GF tortilla for GF/WF)         |
| >                                                                    |
| > • ½ cup ricotta cheese (or coconut cream for DF)                   |
| >                                                                    |
| > • 1 cup fresh strawberries, sliced                                 |
| >                                                                    |
| > • 2 tbsp honey                                                     |
| >                                                                    |
| > • Optional: pinch of cinnamon, ¼ tsp vanilla in ricotta            |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Warm tortillas briefly in a dry skillet.                       |
| >                                                                    |
| > 2\. Spread ricotta on each tortilla.                               |
| >                                                                    |
| > 3\. Layer sliced strawberries over the ricotta.                    |
| >                                                                    |
| > 4\. Drizzle with honey; sprinkle with cinnamon if using.           |
| >                                                                    |
| > 5\. Roll or fold and eat immediately. Best 1--2 hours before game  |
| > time.                                                              |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 12g \| Calories: 320 kcal \|      |
| Carbs: 50g \| Fat: 7g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Fish-Free\] \[Shellfish-Free\] \[Tree  |
| Nut-Free\] \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]              |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Carb\] \[High Calcium\] \[Vitamin C\]     |
| \[High Potassium\] \[High Folate\]                                   |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains wheat and dairy. Use GF tortilla for       |
| GF/WF; use coconut cream for DF. Quick, no-cook (except warming),    |
| and sweet --- great for athletes who don\'t have appetite pre-game   |
| but need carbs.*                                                     |
