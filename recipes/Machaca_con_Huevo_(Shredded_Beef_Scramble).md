+----------------------------------------------------------------------+
| **\#24 Machaca con Huevo (Shredded Beef Scramble)** \| Mexican ·     |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 12 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 6 oz (170 g) machaca (dried shredded beef) or lean cooked        |
| > shredded beef                                                      |
| >                                                                    |
| > • 4 large eggs, beaten                                             |
| >                                                                    |
| > • ½ small white onion, diced                                       |
| >                                                                    |
| > • 1 small tomato, diced                                            |
| >                                                                    |
| > • 1 serrano or jalapeño chilli, finely diced (optional)            |
| >                                                                    |
| > • 1 tbsp avocado oil                                               |
| >                                                                    |
| > • Salt and black pepper to taste                                   |
| >                                                                    |
| > • 4 warm corn tortillas and fresh salsa to serve                   |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil in a skillet over medium. Sauté onion and chilli 3    |
| > min until translucent.                                             |
| >                                                                    |
| > 2\. Add tomato; cook 2 min until softened. Add machaca and stir to |
| > combine.                                                           |
| >                                                                    |
| > 3\. Pour eggs over the beef mixture. Cook, stirring gently, until  |
| > eggs are just set.                                                 |
| >                                                                    |
| > 4\. Season with salt and pepper.                                   |
| >                                                                    |
| > 5\. Serve immediately with warm corn tortillas and fresh salsa.    |
