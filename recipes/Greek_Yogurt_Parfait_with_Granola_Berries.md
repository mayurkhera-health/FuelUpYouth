+----------------------------------------------------------------------+
| **\#2 Greek Yogurt Parfait with Granola & Berries** \| American ·    |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 1                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 cup plain full-fat Greek yogurt                                |
| >                                                                    |
| > • ¼ cup granola (use nut-free/GF granola if needed)                |
| >                                                                    |
| > • ½ cup mixed berries (blueberries, strawberries, raspberries)     |
| >                                                                    |
| > • 1 tbsp raw honey                                                 |
| >                                                                    |
| > • 1 tbsp chia seeds                                                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Spoon Greek yogurt into a tall glass or bowl.                  |
| >                                                                    |
| > 2\. Layer berries and granola over the yogurt.                     |
| >                                                                    |
| > 3\. Sprinkle chia seeds on top and drizzle with honey.             |
| >                                                                    |
| > 4\. Serve immediately or refrigerate overnight for a softer        |
| > texture.                                                           |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 22g \| Calories: 360 kcal \|      |
| Carbs: 50g \| Fat: 7g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Shellfish-Free\] \[Fish-Free\]         |
| \[Peanut-Free\] \[Sesame-Free\] \[Soy-Free\]                         |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Calcium\] \[Vitamin C\]  |
| \[Omega-3\]                                                          |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Use TNF granola for tree-nut allergy. Use           |
| dairy-free coconut yogurt to make DF & VG.*                          |
+----------------------------------------------------------------------+
