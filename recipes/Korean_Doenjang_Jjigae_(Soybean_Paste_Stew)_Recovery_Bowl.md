+----------------------------------------------------------------------+
| **\#101 Korean Doenjang Jjigae (Soybean Paste Stew) Recovery Bowl**  |
| \| Korean · Post-Game Recovery                                       |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 tbsp doenjang (Korean fermented soybean paste)                 |
| >                                                                    |
| > • 6 cups low-sodium vegetable or anchovy broth                     |
| >                                                                    |
| > • 1 block (14 oz) firm tofu, cubed                                 |
| >                                                                    |
| > • 1 medium zucchini, diced                                         |
| >                                                                    |
| > • 1 cup mushrooms (shiitake or oyster), sliced                     |
| >                                                                    |
| > • 3 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 green chili (optional, for older athletes)                     |
| >                                                                    |
| > • 3 green onions, chopped                                          |
| >                                                                    |
| > • 2 cups cooked white rice (serve alongside)                       |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Bring broth to a simmer in a pot; whisk in doenjang until      |
| > dissolved.                                                         |
| >                                                                    |
| > 2\. Add garlic and mushrooms; cook 5 min.                          |
| >                                                                    |
| > 3\. Add zucchini and tofu; simmer 10 min until vegetables are      |
| > tender.                                                            |
| >                                                                    |
| > 4\. Top with green onions and serve with rice.                     |
| >                                                                    |
| > 5\. Eat within 45 min post-game; the fermented paste supports gut  |
| > health during recovery.                                            |
