+----------------------------------------------------------------------+
| **\#56 Butter Chicken (Murgh Makhani) with Basmati Rice** \| Indian  |
| · Dinner                                                             |
+======================================================================+
| ⏱ Prep: 20 min \| 🍳 Cook: 30 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs boneless chicken thighs, cut into chunks                 |
| >                                                                    |
| > • 1½ cups basmati rice, rinsed and cooked                          |
| >                                                                    |
| > • 1 can (14 oz) crushed tomatoes                                   |
| >                                                                    |
| > • ½ cup full-fat coconut milk (for DF version)                     |
| >                                                                    |
| > • 1 medium onion, diced                                            |
| >                                                                    |
| > • 4 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tbsp fresh ginger, grated                                      |
| >                                                                    |
| > • 2 tbsp ghee or avocado oil                                       |
| >                                                                    |
| > • 1 tsp garam masala                                               |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • 1 tsp turmeric                                                   |
| >                                                                    |
| > • 1 tsp coriander                                                  |
| >                                                                    |
| > • ½ tsp chili powder                                               |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • Fresh cilantro for garnish                                       |
| >                                                                    |
| > • ½ cup plain yogurt (or coconut yogurt for DF)                    |
| >                                                                    |
| > • 1 tbsp lemon juice                                               |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Marinate chicken in yogurt, lemon juice, half the garlic, half |
| > the ginger, turmeric, and salt for at least 30 min.                |
| >                                                                    |
| > 2\. Heat ghee in a heavy pot over medium-high; sear chicken pieces |
| > until golden on all sides. Remove and set aside.                   |
| >                                                                    |
| > 3\. In the same pot, sauté onion until golden (\~8 min). Add       |
| > remaining garlic and ginger; cook 1 min.                           |
| >                                                                    |
| > 4\. Add all spices; toast 30 seconds until fragrant.               |
| >                                                                    |
| > 5\. Pour in crushed tomatoes; simmer 10 min, stirring              |
| > occasionally.                                                      |
| >                                                                    |
| > 6\. Use an immersion blender to partially smooth the sauce         |
| > (optional).                                                        |
| >                                                                    |
| > 7\. Return chicken to the pot; stir in coconut milk. Simmer 10 min |
| > until chicken is cooked through.                                   |
| >                                                                    |
| > 8\. Serve over basmati rice, garnished with fresh cilantro.        |
