+----------------------------------------------------------------------+
| **\#104 Black Bean & Sweet Potato Burrito Bowl (Mexican)** \|        |
| Mexican · Post-Game Recovery                                         |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 cans (15 oz each) black beans, drained and rinsed              |
| >                                                                    |
| > • 2 medium sweet potatoes, peeled and diced                        |
| >                                                                    |
| > • 2 cups cooked brown rice                                         |
| >                                                                    |
| > • 1 cup corn kernels (fresh or frozen)                             |
| >                                                                    |
| > • 1 cup salsa (store-bought or fresh)                              |
| >                                                                    |
| > • 1 avocado, diced                                                 |
| >                                                                    |
| > • 1 lime, juiced                                                   |
| >                                                                    |
| > • 1 tsp cumin, 1 tsp chili powder                                  |
| >                                                                    |
| > • 2 tbsp olive oil                                                 |
| >                                                                    |
| > • Fresh cilantro, Greek yogurt or sour cream (optional garnish)    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Toss sweet potato with oil, cumin, chili powder; roast at      |
| > 400°F for 20 min until golden.                                     |
| >                                                                    |
| > 2\. Warm black beans in a small pot with ½ cup salsa and corn      |
| > until heated through.                                              |
| >                                                                    |
| > 3\. Build bowls: rice base, beans, roasted sweet potato, avocado,  |
| > salsa.                                                             |
| >                                                                    |
| > 4\. Squeeze lime over everything; top with cilantro and Greek      |
| > yogurt if using.                                                   |
| >                                                                    |
| > 5\. The complete amino acid pairing of rice + beans provides full  |
| > plant-based recovery protein.                                      |

| Carbs: 74g \| Fat: 10g                                               |
