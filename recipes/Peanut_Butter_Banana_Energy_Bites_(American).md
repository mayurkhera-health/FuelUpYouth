+----------------------------------------------------------------------+
| **\#124 Peanut Butter Banana Energy Bites (American)** \| American · |
| Mid-Day Snacks                                                       |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 0 min \| 🍽 Serves: 4                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 cup rolled oats (GF certified for GF)                          |
| >                                                                    |
| > • ½ cup natural peanut butter (or sunflower seed butter for PF)    |
| >                                                                    |
| > • 3 tbsp honey                                                     |
| >                                                                    |
| > • 1 ripe banana, mashed                                            |
| >                                                                    |
| > • ¼ cup dark chocolate chips                                       |
| >                                                                    |
| > • 2 tbsp ground flaxseed                                           |
| >                                                                    |
| > • ½ tsp vanilla extract                                            |
| >                                                                    |
| > • Pinch of salt                                                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Mix all ingredients in a bowl until well combined.             |
| >                                                                    |
| > 2\. Refrigerate dough 20 min until firm enough to roll.            |
| >                                                                    |
| > 3\. Roll into 16 balls (about 1 tbsp each); place on a lined       |
| > baking sheet.                                                      |
| >                                                                    |
| > 4\. Refrigerate at least 30 min before eating; store up to 1 week. |
| >                                                                    |
| > 5\. Two bites = ideal pre-practice snack; three bites =            |
| > post-practice recovery snack.                                      |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 10g \| Calories: 250 kcal \|      |
| Carbs: 30g \| Fat: 12g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Fish-Free\]             |
| \[Shellfish-Free\] \[Tree Nut-Free\] \[Gluten-Free\] \[Soy-Free\]    |
| \[Sesame-Free\]                                                      |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Vegan\] \[GF\] \[DF\] \[SoyF\]         |
| \[SesF\] \[Low-Sugar / Diabetic-Friendly\]                           |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Magnesium\] \[High       |
| Potassium\] \[Omega-3\] \[High Iron\] \[High Zinc\] \[High Folate\]  |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains peanuts and gluten (oats --- use GF oats   |
| for GF). Use sunflower seed butter for PF; use allergen-free         |
| chocolate chips for TNF. Flaxseed adds ALA Omega-3; banana provides  |
| potassium for muscle cramping prevention.*                           |
