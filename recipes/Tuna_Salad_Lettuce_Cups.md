+----------------------------------------------------------------------+
| **\#31 Tuna Salad Lettuce Cups** \| American · Lunch                 |
+======================================================================+
| ⏱ Prep: 8 min \| 🍳 Cook: 0 min \| 🍽 Serves: 1                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 can (5 oz / 142 g) water-packed tuna, drained                  |
| >                                                                    |
| > • 2 tbsp plain Greek yogurt (or light mayo)                        |
| >                                                                    |
| > • 1 stalk celery, finely diced                                     |
| >                                                                    |
| > • 2 tbsp diced red onion                                           |
| >                                                                    |
| > • 1 tsp Dijon mustard                                              |
| >                                                                    |
| > • 1 tsp lemon juice                                                |
| >                                                                    |
| > • Salt and black pepper to taste                                   |
| >                                                                    |
| > • 6 large romaine or butter lettuce leaves                         |
| >                                                                    |
| > • ½ avocado, sliced (for serving)                                  |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Drain tuna and flake into a bowl.                              |
| >                                                                    |
| > 2\. Mix in Greek yogurt, celery, red onion, mustard, and lemon     |
| > juice. Season with salt and pepper.                                |
| >                                                                    |
| > 3\. Spoon tuna mixture into lettuce cups.                          |
| >                                                                    |
| > 4\. Top each cup with avocado slices. Serve immediately.           |
