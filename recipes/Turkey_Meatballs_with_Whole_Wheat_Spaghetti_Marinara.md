+----------------------------------------------------------------------+
| **\#65 Turkey Meatballs with Whole Wheat Spaghetti & Marinara** \|   |
| American · Dinner                                                    |
+======================================================================+
| ⏱ Prep: 20 min \| 🍳 Cook: 25 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 lb lean ground turkey                                          |
| >                                                                    |
| > • 12 oz whole wheat spaghetti (or GF pasta for GF/WF)              |
| >                                                                    |
| > • 2 cups store-bought low-sodium marinara sauce                    |
| >                                                                    |
| > • ¼ cup breadcrumbs (GF breadcrumbs for GF)                        |
| >                                                                    |
| > • 1 egg (or 1 tbsp flaxseed + 3 tbsp water for EF)                 |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 2 tbsp fresh parsley, chopped                                    |
| >                                                                    |
| > • ¼ cup Parmesan cheese, grated (omit for DF)                      |
| >                                                                    |
| > • ½ tsp Italian seasoning                                          |
| >                                                                    |
| > • Salt and pepper to taste                                         |
| >                                                                    |
| > • 1 tbsp olive oil                                                 |
| >                                                                    |
| > • Fresh basil to garnish                                           |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 400°F. Mix ground turkey with breadcrumbs,     |
| > egg, garlic, parsley, Parmesan, Italian seasoning, salt, and       |
| > pepper.                                                            |
| >                                                                    |
| > 2\. Roll into 1.5-inch balls (\~20 meatballs); place on a lightly  |
| > oiled sheet pan.                                                   |
| >                                                                    |
| > 3\. Bake 18--20 min until golden and cooked through (internal temp |
| > 165°F).                                                            |
| >                                                                    |
| > 4\. Meanwhile, cook pasta per package directions; drain, reserving |
| > ½ cup pasta water.                                                 |
| >                                                                    |
| > 5\. Warm marinara in a saucepan; add meatballs and simmer 5 min.   |
| >                                                                    |
| > 6\. Toss pasta with sauce (add pasta water if needed to loosen).   |
| >                                                                    |
| > 7\. Serve topped with extra Parmesan and fresh basil.              |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 38g \| Calories: 540 kcal \|      |
| Carbs: 62g \| Fat: 14g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Fish-Free\] \[Shellfish-Free\] \[Tree Nut-Free\]    |
| \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]                         |
|                                                                      |
| **DIETARY:** \[Low-Sugar / Diabetic-Friendly\]                       |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Iron\] \[High Zinc\]     |
| \[B12-Rich\] \[High Folate\] \[High Potassium\] \[Vitamin C\]        |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains wheat, egg, and dairy. Use GF pasta + GF   |
| breadcrumbs for GF/WF; use flax egg for EF; omit Parmesan for DF.    |
| Turkey meatballs are leaner and lower in saturated fat than beef --- |
| better daily option for athletes.*                                   |
