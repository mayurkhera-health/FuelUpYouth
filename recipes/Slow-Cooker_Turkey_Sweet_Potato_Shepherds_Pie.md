+----------------------------------------------------------------------+
| **\#69 Slow-Cooker Turkey & Sweet Potato Shepherd\'s Pie** \|        |
| American · Dinner                                                    |
+======================================================================+
| ⏱ Prep: 20 min \| 🍳 Cook: 40 min \| 🍽 Serves: 6                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs lean ground turkey                                       |
| >                                                                    |
| > • 3 large sweet potatoes, peeled and cubed                         |
| >                                                                    |
| > • 1 cup frozen corn                                                |
| >                                                                    |
| > • 1 cup frozen peas                                                |
| >                                                                    |
| > • 2 medium carrots, diced                                          |
| >                                                                    |
| > • 1 medium onion, diced                                            |
| >                                                                    |
| > • 3 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 cup low-sodium chicken broth                                   |
| >                                                                    |
| > • 2 tbsp tomato paste                                              |
| >                                                                    |
| > • 1 tsp Worcestershire sauce (check GF label for GF)               |
| >                                                                    |
| > • 1 tsp thyme                                                      |
| >                                                                    |
| > • 1 tsp rosemary                                                   |
| >                                                                    |
| > • 2 tbsp avocado oil                                               |
| >                                                                    |
| > • 2 tbsp butter or ghee (optional, for mash --- omit for DF)       |
| >                                                                    |
| > • Salt and pepper to taste                                         |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 375°F.                                         |
| >                                                                    |
| > 2\. Boil sweet potatoes until tender, \~15 min. Mash with          |
| > butter/ghee (if using), salt, and pepper. Set aside.               |
| >                                                                    |
| > 3\. Heat oil in an oven-safe skillet over medium-high. Cook onion  |
| > and carrot 5 min. Add garlic; cook 1 min.                          |
| >                                                                    |
| > 4\. Add ground turkey; cook through, breaking up \~7 min. Season   |
| > with thyme, rosemary, salt, and pepper.                            |
| >                                                                    |
| > 5\. Stir in tomato paste, Worcestershire, broth, corn, and peas.   |
| > Simmer 5 min.                                                      |
| >                                                                    |
| > 6\. Spread sweet potato mash evenly over the turkey filling.       |
| >                                                                    |
| > 7\. Bake 20--25 min until top is lightly golden.                   |
| >                                                                    |
| > 8\. Let rest 5 min before serving.                                 |
