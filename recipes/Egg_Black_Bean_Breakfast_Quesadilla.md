+----------------------------------------------------------------------+
| **\#19 Egg & Black Bean Breakfast Quesadilla** \| Mexican ·          |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 10 min \| 🍽 Serves: 1                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 whole wheat flour tortillas (8-inch)                           |
| >                                                                    |
| > • 3 large eggs, beaten                                             |
| >                                                                    |
| > • ⅓ cup canned black beans, rinsed                                 |
| >                                                                    |
| > • ¼ cup shredded pepper jack or cheddar cheese                     |
| >                                                                    |
| > • ¼ cup diced red bell pepper                                      |
| >                                                                    |
| > • 2 tbsp low-sodium salsa                                          |
| >                                                                    |
| > • 1 tsp olive oil                                                  |
| >                                                                    |
| > • Salt and pepper to taste                                         |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil in a skillet. Add bell pepper; sauté 2 min. Pour in   |
| > eggs and scramble until just set. Season.                          |
| >                                                                    |
| > 2\. Stir in black beans and remove from heat.                      |
| >                                                                    |
| > 3\. Place one tortilla in the same pan over medium heat. Sprinkle  |
| > cheese over half, then add egg-bean filling. Fold in half.         |
| >                                                                    |
| > 4\. Cook 2--3 min per side until tortilla is golden and cheese     |
| > melts.                                                             |
| >                                                                    |
| > 5\. Slice into wedges. Serve with fresh salsa.                     |
