+----------------------------------------------------------------------+
| **\#117 Custom Athlete Trail Mix (American)** \| American · Mid-Day  |
| Snacks                                                               |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 4                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • ½ cup roasted pumpkin seeds (pepitas)                            |
| >                                                                    |
| > • ½ cup roasted almonds or cashews                                 |
| >                                                                    |
| > • ½ cup dried cranberries or raisins (no added sugar)              |
| >                                                                    |
| > • ¼ cup dark chocolate chips (70%+ cacao)                          |
| >                                                                    |
| > • ¼ cup sunflower seeds                                            |
| >                                                                    |
| > • Optional: ¼ cup coconut flakes (unsweetened)                     |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Combine all ingredients in a large bowl; mix evenly.           |
| >                                                                    |
| > 2\. Portion into 4 snack bags or containers (about ⅓ cup each).    |
| >                                                                    |
| > 3\. Refrigerate extra portions up to 2 weeks.                      |
| >                                                                    |
| > 4\. Grab one bag per athlete before practice or games.             |
| >                                                                    |
| > 5\. Pumpkin seeds are one of the highest zinc foods per ounce ---  |
| > critical for immune function and testosterone in teen athletes.    |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 12g \| Calories: 260 kcal \|      |
| Carbs: 26g \| Fat: 14g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Fish-Free\]             |
| \[Shellfish-Free\] \[Peanut-Free\] \[Gluten-Free\] \[Soy-Free\]      |
| \[Sesame-Free\]                                                      |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Vegan\] \[GF\] \[DF\] \[PF\]           |
| \[Low-Sugar / Diabetic-Friendly\]                                    |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Zinc\] \[High Magnesium\] \[Omega-3\]     |
| \[High Iron\] \[High Potassium\] \[High Calcium\]                    |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains tree nuts (almonds/cashews). Use all seeds |
| (pumpkin + sunflower + hemp) for TNF. Dark chocolate adds flavonoids |
| that reduce exercise-induced inflammation; pumpkin seeds are one of  |
| the most zinc-dense snack foods available.*                          |
