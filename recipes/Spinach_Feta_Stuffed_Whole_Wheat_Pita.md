+----------------------------------------------------------------------+
| **\#49 Spinach & Feta Stuffed Whole Wheat Pita** \| American · Lunch |
+======================================================================+
| ⏱ Prep: 8 min \| 🍳 Cook: 0 min \| 🍽 Serves: 1                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 large whole wheat pita pocket                                  |
| >                                                                    |
| > • 1 cup baby spinach                                               |
| >                                                                    |
| > • 3 tbsp crumbled feta cheese                                      |
| >                                                                    |
| > • ¼ cup canned chickpeas, rinsed                                   |
| >                                                                    |
| > • 4 cherry tomatoes, halved                                        |
| >                                                                    |
| > • 2 tbsp hummus                                                    |
| >                                                                    |
| > • 1 tsp olive oil, 1 tsp lemon juice                               |
| >                                                                    |
| > • Salt and pepper to taste                                         |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Gently warm pita in a dry pan or microwave 20 seconds.         |
| >                                                                    |
| > 2\. Toss spinach with olive oil, lemon juice, salt, and pepper.    |
| >                                                                    |
| > 3\. Spread hummus inside the pita pocket.                          |
| >                                                                    |
| > 4\. Stuff with spinach, feta, chickpeas, and tomatoes.             |
| >                                                                    |
| > 5\. Serve immediately or wrap in foil for a packed lunch.          |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 18g \| Calories: 380 kcal \|      |
| Carbs: 48g \| Fat: 13g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Shellfish-Free\] \[Fish-Free\] \[Tree  |
| Nut-Free\] \[Peanut-Free\] \[Soy-Free\]                              |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Calcium\] \[High Iron\]  |
| \[High Folate\] \[Vitamin C\] \[High Potassium\]                     |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains wheat (pita), dairy (feta), and sesame     |
| (hummus/tahini). Use GF flatbread for GF. Omit feta + use dairy-free |
| hummus for DF. Great fast school lunch in under 10 min.*             |
