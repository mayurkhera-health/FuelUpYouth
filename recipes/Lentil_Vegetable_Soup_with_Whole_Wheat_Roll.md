+----------------------------------------------------------------------+
| **\#42 Lentil Vegetable Soup with Whole Wheat Roll** \| American ·   |
| Lunch                                                                |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 30 min \| 🍽 Serves: 3                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 cup red lentils, rinsed                                        |
| >                                                                    |
| > • 1 medium carrot, diced                                           |
| >                                                                    |
| > • 2 stalks celery, diced                                           |
| >                                                                    |
| > • 1 small onion, diced                                             |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 can (14 oz / 400 g) diced tomatoes                             |
| >                                                                    |
| > • 4 cups low-sodium vegetable or chicken broth                     |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • ½ tsp turmeric                                                   |
| >                                                                    |
| > • ½ tsp smoked paprika                                             |
| >                                                                    |
| > • 1 tbsp olive oil                                                 |
| >                                                                    |
| > • Juice of ½ lemon                                                 |
| >                                                                    |
| > • Salt and pepper, fresh parsley to serve                          |
| >                                                                    |
| > • 2 whole wheat dinner rolls                                       |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat oil in a pot. Sauté onion, carrot, and celery 5 min. Add  |
| > garlic and spices; stir 1 min.                                     |
| >                                                                    |
| > 2\. Add lentils, tomatoes, and broth. Bring to a boil.             |
| >                                                                    |
| > 3\. Reduce heat; simmer 20--25 min until lentils are completely    |
| > soft.                                                              |
| >                                                                    |
| > 4\. Squeeze in lemon juice. Adjust salt and pepper.                |
| >                                                                    |
| > 5\. Serve in bowls topped with parsley, alongside warm whole wheat |
| > rolls.                                                             |
