+----------------------------------------------------------------------+
| **\#27 Chicken Teriyaki Bowl with Brown Rice & Edamame** \| Japanese |
| · Lunch                                                              |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 boneless chicken thighs (about 10 oz / 280 g total)            |
| >                                                                    |
| > • 3 tbsp low-sodium soy sauce or tamari                            |
| >                                                                    |
| > • 1½ tbsp honey                                                    |
| >                                                                    |
| > • 1 tsp rice vinegar                                               |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 1 tsp arrowroot or cornstarch (to thicken sauce)                 |
| >                                                                    |
| > • 1½ cups cooked brown rice                                        |
| >                                                                    |
| > • ½ cup frozen edamame, thawed                                     |
| >                                                                    |
| > • ½ cup shredded purple cabbage                                    |
| >                                                                    |
| > • 2 scallions, sliced                                              |
| >                                                                    |
| > • 1 tsp sesame seeds (optional --- omit for SesF)                  |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Whisk soy sauce, honey, rice vinegar, ginger, and cornstarch   |
| > in a small bowl.                                                   |
| >                                                                    |
| > 2\. Heat a non-stick pan over medium-high. Cook chicken thighs     |
| > 6--7 min per side until cooked through.                            |
| >                                                                    |
| > 3\. Pour teriyaki sauce over chicken; simmer 2 min until sauce     |
| > thickens and coats the chicken. Slice.                             |
| >                                                                    |
| > 4\. Divide rice between bowls. Top with chicken, edamame, and      |
| > cabbage.                                                           |
| >                                                                    |
| > 5\. Drizzle with remaining sauce; scatter scallions and sesame     |
| > seeds.                                                             |
+----------------------------------------------------------------------+

| Carbs: 50g \| Fat: 10g                                               |

|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Carb\] \[High Iron\]     |

| the highest-protein lunches in the database.*                        |
