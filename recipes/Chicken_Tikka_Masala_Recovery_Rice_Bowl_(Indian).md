+----------------------------------------------------------------------+
| **\#102 Chicken Tikka Masala Recovery Rice Bowl (Indian)** \| Indian |
| · Post-Game Recovery                                                 |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 25 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs boneless chicken thighs, cubed                           |
| >                                                                    |
| > • 1 cup plain Greek yogurt (for marinade)                          |
| >                                                                    |
| > • 2 tbsp tikka masala spice blend                                  |
| >                                                                    |
| > • 1 can (14 oz) crushed tomatoes                                   |
| >                                                                    |
| > • 1 cup full-fat coconut milk (DF option) or heavy cream           |
| >                                                                    |
| > • 1 large onion, diced                                             |
| >                                                                    |
| > • 3 cloves garlic, minced                                          |
| >                                                                    |
| > • 1-inch fresh ginger, grated                                      |
| >                                                                    |
| > • 2 tbsp olive oil                                                 |
| >                                                                    |
| > • 2 cups cooked basmati rice                                       |
| >                                                                    |
| > • Fresh cilantro for garnish                                       |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Marinate chicken in yogurt + 1 tbsp tikka spice for at least   |
| > 15 min (or overnight).                                             |
| >                                                                    |
| > 2\. Sauté onion in oil until golden; add garlic, ginger, remaining |
| > spice; cook 2 min.                                                 |
| >                                                                    |
| > 3\. Add marinated chicken; cook 7--8 min until browned.            |
| >                                                                    |
| > 4\. Stir in tomatoes and coconut milk; simmer 15 min until sauce   |
| > thickens.                                                          |
| >                                                                    |
| > 5\. Serve over basmati rice with cilantro; provides complete       |
| > protein + carb recovery.                                           |
+----------------------------------------------------------------------+

| Carbs: 48g \| Fat: 18g                                               |

| **TIP / SWAP:** *Contains dairy (yogurt/cream) --- use coconut milk  |
| for DF. Chicken thighs have more iron and zinc than breast meat;     |
| basmati rice is lower glycemic index than regular white rice;        |
| turmeric in tikka blend is anti-inflammatory.*                       |
