+----------------------------------------------------------------------+
| **\#67 Beef & Broccoli Stir-Fry with Garlic Brown Rice** \| Chinese  |
| · Dinner                                                             |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 15 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 lb sirloin or flank steak, thinly sliced                       |
| >                                                                    |
| > • 4 cups broccoli florets                                          |
| >                                                                    |
| > • 2 cups brown rice, cooked                                        |
| >                                                                    |
| > • 3 cloves garlic, minced (2 for sauce, 1 for rice)                |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 1 tbsp avocado oil                                               |
| >                                                                    |
| > • Sauce: 4 tbsp low-sodium soy sauce (or coconut aminos for SoyF)  |
| >                                                                    |
| > • 1 tbsp oyster sauce (omit for SoyF/SF --- use hoisin or extra    |
| > soy)                                                               |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • 1 tsp cornstarch dissolved in 2 tsp water                        |
| >                                                                    |
| > • ½ tsp sesame oil (omit for SesF)                                 |
| >                                                                    |
| > • 1 tbsp sesame seeds (omit for SesF)                              |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Whisk all sauce ingredients together; set aside.               |
| >                                                                    |
| > 2\. Blanch broccoli in boiling salted water 2 min; drain and set   |
| > aside.                                                             |
| >                                                                    |
| > 3\. Heat oil in a wok over high heat. Add beef in a single layer;  |
| > sear 1--2 min per side. Remove.                                    |
| >                                                                    |
| > 4\. Add garlic and ginger to the wok; stir-fry 30 sec until        |
| > fragrant.                                                          |
| >                                                                    |
| > 5\. Return beef and add broccoli; pour sauce over. Toss vigorously |
| > 1--2 min until sauce coats everything and thickens.                |
| >                                                                    |
| > 6\. Serve immediately over garlic brown rice (rice stir-fried with |
| > 1 clove garlic in a drizzle of oil).                               |
+----------------------------------------------------------------------+

| Carbs: 50g \| Fat: 15g                                               |

|                                                                      |

+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains soy and sesame. Use coconut aminos for     |
| SoyF; omit sesame for SesF; use tamari for GF. Oyster sauce adds     |
| depth --- omit for shellfish-free. Broccoli + beef = iron and        |
| Vitamin C in one pan.*                                               |
