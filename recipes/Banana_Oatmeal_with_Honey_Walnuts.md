+----------------------------------------------------------------------+
| **\#78 Banana Oatmeal with Honey & Walnuts** \| American · Pre-Game  |
| Fueling                                                              |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 10 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 cup old-fashioned rolled oats (certified GF for GF)            |
| >                                                                    |
| > • 2 cups water or low-fat milk (or oat milk for DF)                |
| >                                                                    |
| > • 2 ripe bananas --- 1 mashed in, 1 sliced on top                  |
| >                                                                    |
| > • 1 tbsp honey or maple syrup                                      |
| >                                                                    |
| > • 2 tbsp chopped walnuts (omit for TNF --- use sunflower seeds     |
| > instead)                                                           |
| >                                                                    |
| > • ½ tsp cinnamon                                                   |
| >                                                                    |
| > • Pinch of salt                                                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Bring liquid to a boil; add oats and salt. Reduce heat; cook 5 |
| > min, stirring occasionally.                                        |
| >                                                                    |
| > 2\. Mash one banana and stir into oats as it finishes cooking.     |
| >                                                                    |
| > 3\. Remove from heat; stir in honey and cinnamon.                  |
| >                                                                    |
| > 4\. Top with sliced banana and walnuts.                            |
| >                                                                    |
| > 5\. Serve warm --- ideal 2--2.5 hours before game time.            |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 12g \| Calories: 380 kcal \|      |
| Carbs: 64g \| Fat: 8g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Fish-Free\] \[Shellfish-Free\]         |
| \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]                         |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[Low-Sugar / Diabetic-Friendly\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Carb\] \[High Potassium\] \[High          |
| Magnesium\] \[High Folate\] \[High Iron\] \[Vitamin C\]              |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains tree nuts (walnuts) and may contain gluten |
| (oats). Use certified GF oats for GF; omit walnuts or use sunflower  |
| seeds for TNF; use oat milk for DF. Banana + oats = sustained energy |
| release --- classic pre-game fuel.*                                  |
