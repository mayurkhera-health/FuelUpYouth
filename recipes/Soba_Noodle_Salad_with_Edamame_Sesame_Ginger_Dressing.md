+----------------------------------------------------------------------+
| **\#40 Soba Noodle Salad with Edamame & Sesame Ginger Dressing** \|  |
| Japanese · Lunch                                                     |
+======================================================================+
| ⏱ Prep: 12 min \| 🍳 Cook: 8 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 6 oz (170 g) dried soba noodles (100% buckwheat for GF)          |
| >                                                                    |
| > • 1 cup frozen edamame, thawed                                     |
| >                                                                    |
| > • 1 medium carrot, julienned                                       |
| >                                                                    |
| > • ½ red bell pepper, thinly sliced                                 |
| >                                                                    |
| > • 2 scallions, sliced                                              |
| >                                                                    |
| > • 1 tbsp low-sodium soy sauce or tamari                            |
| >                                                                    |
| > • 1 tbsp rice vinegar                                              |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 1 tsp honey                                                      |
| >                                                                    |
| > • 1 tsp sesame oil (omit for SesF)                                 |
| >                                                                    |
| > • 1 tsp sesame seeds (omit for SesF)                               |
| >                                                                    |
| > • 1 tsp avocado oil                                                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Cook soba noodles per package instructions. Rinse under cold   |
| > water and drain well.                                              |
| >                                                                    |
| > 2\. Whisk soy sauce, rice vinegar, ginger, honey, sesame oil, and  |
| > avocado oil for the dressing.                                      |
| >                                                                    |
| > 3\. Toss noodles with edamame, carrot, and bell pepper.            |
| >                                                                    |
| > 4\. Pour dressing over and toss to coat. Top with scallions and    |
| > sesame seeds.                                                      |
| >                                                                    |
| > 5\. Serve cold or at room temperature. Keeps refrigerated up to 2  |
| > days.                                                              |

| for shelled peas for SoyF. Outstanding packed-lunch noodle salad.*   |
