+----------------------------------------------------------------------+
| **\#122 Mango Lassi with Chia Seeds (Indian)** \| Indian · Mid-Day   |
| Snacks                                                               |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 cup plain whole-milk yogurt (or coconut yogurt for DF)         |
| >                                                                    |
| > • 1 cup frozen or fresh mango chunks                               |
| >                                                                    |
| > • ½ cup milk (or oat milk for DF)                                  |
| >                                                                    |
| > • 1 tbsp chia seeds                                                |
| >                                                                    |
| > • 1 tsp honey                                                      |
| >                                                                    |
| > • ¼ tsp ground cardamom                                            |
| >                                                                    |
| > • Pinch of salt                                                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Blend yogurt, mango, milk, honey, cardamom, and salt until     |
| > smooth.                                                            |
| >                                                                    |
| > 2\. Pour into glasses; stir in chia seeds.                         |
| >                                                                    |
| > 3\. Let rest 5 min for chia to begin gelling (adds fiber and       |
| > Omega-3s).                                                         |
| >                                                                    |
| > 4\. Drink immediately or refrigerate up to 24 hours (chia will     |
| > thicken it).                                                       |
| >                                                                    |
| > 5\. Mango is one of the highest Vitamin C fruits --- supporting    |
| > collagen synthesis for joint recovery.                             |
