+----------------------------------------------------------------------+
| **\#81 Brown Rice Cakes with Almond Butter & Sliced Banana** \|      |
| American · Pre-Game Fueling                                          |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 6 plain brown rice cakes                                         |
| >                                                                    |
| > • 3 tbsp almond butter (or sunflower butter for TNF/PF)            |
| >                                                                    |
| > • 2 ripe bananas, sliced                                           |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • Optional: pinch of cinnamon                                      |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Spread almond butter evenly over each rice cake.               |
| >                                                                    |
| > 2\. Layer banana slices on top; drizzle with honey and a pinch of  |
| > cinnamon.                                                          |
| >                                                                    |
| > 3\. Serve immediately --- best eaten 45--90 min before game time.  |
| >                                                                    |
| > 4\. Pairs well with a glass of water or electrolyte drink.         |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 10g \| Calories: 320 kcal \|      |
| Carbs: 52g \| Fat: 8g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Fish-Free\]             |
| \[Shellfish-Free\] \[Peanut-Free\] \[Gluten-Free\] \[Soy-Free\]      |
| \[Sesame-Free\] \[WF\]                                               |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[GF\] \[DF\] \[Low-Sugar /              |
| Diabetic-Friendly\]                                                  |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Carb\] \[High Potassium\] \[High          |
| Magnesium\] \[Vitamin C\] \[High Folate\]                            |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains tree nuts (almond butter). Use sunflower   |
| butter for TNF/PF. Certified GF rice cakes for GF. Super portable    |
| pre-game snack --- no refrigeration needed. Ideal for travel days or |
| away games.*                                                         |
