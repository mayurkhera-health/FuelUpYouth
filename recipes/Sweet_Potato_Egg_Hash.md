+----------------------------------------------------------------------+
| **\#8 Sweet Potato & Egg Hash** \| American · Breakfast              |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 22 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 medium sweet potatoes, diced small (skin on or off)            |
| >                                                                    |
| > • 4 large eggs                                                     |
| >                                                                    |
| > • 1 red or green bell pepper, diced                                |
| >                                                                    |
| > • ½ yellow onion, diced                                            |
| >                                                                    |
| > • 2 tbsp olive oil                                                 |
| >                                                                    |
| > • ½ tsp smoked paprika                                             |
| >                                                                    |
| > • ½ tsp cumin                                                      |
| >                                                                    |
| > • Salt and black pepper to taste                                   |
| >                                                                    |
| > • Fresh cilantro or parsley to garnish                             |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Heat olive oil in a large skillet over medium-high. Add sweet  |
| > potato, cook 8--10 min, stirring occasionally.                     |
| >                                                                    |
| > 2\. Add bell pepper and onion; cook 5 more min until softened and  |
| > edges caramelise.                                                  |
| >                                                                    |
| > 3\. Season with paprika, cumin, salt, and pepper.                  |
| >                                                                    |
| > 4\. Make 4 wells in the hash; crack one egg into each. Cover and   |
| > cook 4--5 min until whites are set.                                |
| >                                                                    |
| > 5\. Garnish with cilantro and serve from the pan.                  |
