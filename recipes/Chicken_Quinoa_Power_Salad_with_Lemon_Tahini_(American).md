+----------------------------------------------------------------------+
| **\#109 Chicken & Quinoa Power Salad with Lemon Tahini (American)**  |
| \| American · Post-Game Recovery                                     |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 20 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs grilled chicken breast, sliced                           |
| >                                                                    |
| > • 2 cups cooked quinoa                                             |
| >                                                                    |
| > • 4 cups mixed greens (spinach, arugula, kale)                     |
| >                                                                    |
| > • 1 cup cherry tomatoes, halved                                    |
| >                                                                    |
| > • ½ cup cucumber, diced                                            |
| >                                                                    |
| > • ¼ cup red onion, thinly sliced                                   |
| >                                                                    |
| > • ¼ cup kalamata olives (optional)                                 |
| >                                                                    |
| > • 2 tbsp tahini (omit for SesF)                                    |
| >                                                                    |
| > • 2 tbsp lemon juice                                               |
| >                                                                    |
| > • 1 tbsp olive oil                                                 |
| >                                                                    |
| > • 1 clove garlic, minced                                           |
| >                                                                    |
| > • Salt and pepper                                                  |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Grill or pan-cook chicken until internal temp reaches 165°F;   |
| > slice and let rest.                                                |
| >                                                                    |
| > 2\. Whisk tahini, lemon juice, olive oil, garlic, and 2 tbsp water |
| > into dressing.                                                     |
| >                                                                    |
| > 3\. Toss greens with quinoa, tomatoes, cucumber, and onion.        |
| >                                                                    |
| > 4\. Top with chicken and olives; drizzle tahini dressing.          |
| >                                                                    |
| > 5\. Quinoa is the only plant grain with all 9 essential amino      |
| > acids --- ideal for recovery.                                      |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 42g \| Calories: 470 kcal \|      |
| Carbs: 36g \| Fat: 16g                                               |

+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains sesame (tahini) --- omit tahini for SesF   |
| and use lemon-olive oil dressing instead. Quinoa has 8g protein per  |
| cup and all essential amino acids; kale + spinach provide high       |
| Vitamin C for collagen synthesis and iron absorption.*               |
