+----------------------------------------------------------------------+
| **\#10 Chicken Congee with Soft-Boiled Egg** \| Chinese · Breakfast  |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 40 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • ½ cup jasmine or short-grain rice                                |
| >                                                                    |
| > • 4 cups low-sodium chicken broth                                  |
| >                                                                    |
| > • 1 cup cooked chicken breast, shredded                            |
| >                                                                    |
| > • 2 large eggs                                                     |
| >                                                                    |
| > • 1-inch piece fresh ginger, grated                                |
| >                                                                    |
| > • 2 scallions (spring onions), thinly sliced                       |
| >                                                                    |
| > • 1 tsp soy sauce or tamari (GF)                                   |
| >                                                                    |
| > • ½ tsp sesame oil                                                 |
| >                                                                    |
| > • White pepper to taste                                            |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Combine rice, broth, and ginger in a medium saucepan. Bring to |
| > a boil.                                                            |
| >                                                                    |
| > 2\. Reduce to a low simmer; cook uncovered 35--40 min, stirring    |
| > occasionally, until rice breaks down into a thick porridge.        |
| >                                                                    |
| > 3\. Stir in shredded chicken, soy sauce, and white pepper. Simmer  |
| > 3 more min.                                                        |
| >                                                                    |
| > 4\. Soft-boil eggs separately (6--7 min in simmering water), peel  |
| > and halve.                                                         |
| >                                                                    |
| > 5\. Ladle congee into bowls; top with soft-boiled egg, scallions,  |
| > and a few drops of sesame oil.                                     |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 26g \| Calories: 350 kcal \|      |
| Carbs: 40g \| Fat: 9g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Shellfish-Free\] \[Fish-Free\]       |
| \[Tree Nut-Free\] \[Peanut-Free\]                                    |
|                                                                      |
| **DIETARY:** \[Low-Sodium\]                                          |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[B12-Rich\] \[High Zinc\]      |
| \[High Iron\]                                                        |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Use tamari instead of soy sauce for Gluten-Free &   |
| Soy-Free alternative. Omit sesame oil for Sesame-Free. Egg is        |
| soft-boiled separately --- omit for EF version.*                     |
