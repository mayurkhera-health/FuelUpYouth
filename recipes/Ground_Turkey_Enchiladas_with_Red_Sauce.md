+----------------------------------------------------------------------+
| **\#59 Ground Turkey Enchiladas with Red Sauce** \| Mexican · Dinner |
+======================================================================+
| ⏱ Prep: 20 min \| 🍳 Cook: 30 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1 lb lean ground turkey                                          |
| >                                                                    |
| > • 8 corn tortillas (certified GF for GF)                           |
| >                                                                    |
| > • 1 can (15 oz) black beans, rinsed                                |
| >                                                                    |
| > • 1 cup frozen corn kernels                                        |
| >                                                                    |
| > • 1½ cups red enchilada sauce (store-bought low-sodium)            |
| >                                                                    |
| > • 1 cup shredded Mexican-blend cheese (omit for DF)                |
| >                                                                    |
| > • 1 small onion, diced                                             |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp cumin                                                      |
| >                                                                    |
| > • 1 tsp chili powder                                               |
| >                                                                    |
| > • ½ tsp oregano                                                    |
| >                                                                    |
| > • 1 tbsp olive oil                                                 |
| >                                                                    |
| > • Fresh cilantro and lime wedges to serve                          |
| >                                                                    |
| > • 1 avocado, sliced, to serve                                      |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Preheat oven to 375°F. Spread ¼ cup enchilada sauce in the     |
| > bottom of a 9×13 baking dish.                                      |
| >                                                                    |
| > 2\. Heat olive oil in a skillet; cook onion until soft. Add garlic |
| > and cook 1 min.                                                    |
| >                                                                    |
| > 3\. Add ground turkey; cook through, breaking up with a spoon (\~7 |
| > min). Season with cumin, chili powder, and oregano.                |
| >                                                                    |
| > 4\. Stir in black beans and corn; remove from heat.                |
| >                                                                    |
| > 5\. Warm tortillas briefly to make them pliable. Fill each with    |
| > turkey mixture, roll up, and place seam-side down in the baking    |
| > dish.                                                              |
| >                                                                    |
| > 6\. Pour remaining enchilada sauce over the top; scatter cheese if |
| > using.                                                             |
| >                                                                    |
| > 7\. Bake 20--25 min until bubbly and cheese is melted.             |
| >                                                                    |
| > 8\. Serve with cilantro, lime wedges, and avocado.                 |
