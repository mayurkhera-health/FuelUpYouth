+----------------------------------------------------------------------+
| **\#113 Birria-Style Beef & Bean Recovery Tacos (Mexican)** \|       |
| Mexican · Post-Game Recovery                                         |
+======================================================================+
| ⏱ Prep: 15 min \| 🍳 Cook: 30 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs beef chuck, cut into chunks                              |
| >                                                                    |
| > • 1 can (15 oz) pinto beans, drained                               |
| >                                                                    |
| > • 3 dried guajillo chiles, seeded and soaked                       |
| >                                                                    |
| > • 1 can (14 oz) diced tomatoes                                     |
| >                                                                    |
| > • 4 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp cumin, 1 tsp oregano, ½ tsp smoked paprika                 |
| >                                                                    |
| > • 2 cups low-sodium beef broth                                     |
| >                                                                    |
| > • 8 corn tortillas (GF)                                            |
| >                                                                    |
| > • ½ cup diced white onion                                          |
| >                                                                    |
| > • Fresh cilantro, lime wedges for serving                          |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Blend soaked chiles, tomatoes, garlic, and spices into a       |
| > smooth sauce.                                                      |
| >                                                                    |
| > 2\. Brown beef chunks in a pot; add chile sauce and broth; simmer  |
| > 25--30 min.                                                        |
| >                                                                    |
| > 3\. Shred beef; stir in pinto beans and heat through.              |
| >                                                                    |
| > 4\. Warm tortillas; fill with beef-bean mixture.                   |
| >                                                                    |
| > 5\. Top with onion and cilantro; dip in reserved consommé broth    |
| > for electrolytes.                                                  |
