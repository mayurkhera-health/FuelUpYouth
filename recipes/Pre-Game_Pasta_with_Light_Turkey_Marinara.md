+----------------------------------------------------------------------+
| **\#76 Pre-Game Pasta with Light Turkey Marinara** \| American ·     |
| Pre-Game Fueling                                                     |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 12 oz white spaghetti or penne (NOT whole wheat --- white pasta  |
| > digests faster pre-game)                                           |
| >                                                                    |
| > • ½ lb lean ground turkey                                          |
| >                                                                    |
| > • 1½ cups low-sodium marinara sauce                                |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • ½ tsp Italian seasoning                                          |
| >                                                                    |
| > • 1 tbsp olive oil                                                 |
| >                                                                    |
| > • Salt to taste                                                    |
| >                                                                    |
| > • Optional: 2 tbsp Parmesan (omit for DF)                          |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Cook pasta per package directions until al dente; drain.       |
| >                                                                    |
| > 2\. Heat oil; cook garlic 30 sec. Add ground turkey; cook through  |
| > \~7 min, breaking up.                                              |
| >                                                                    |
| > 3\. Add marinara and Italian seasoning; simmer 5 min.              |
| >                                                                    |
| > 4\. Toss pasta with sauce --- keep it light, not heavy.            |
| >                                                                    |
| > 5\. Serve in moderate portions (don\'t overfill --- pre-game meals |
| > should leave room for performance).                                |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 30g \| Calories: 500 kcal \|      |
| Carbs: 70g \| Fat: 8g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Fish-Free\] \[Shellfish-Free\] \[Tree  |
| Nut-Free\] \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]              |
|                                                                      |
| **DIETARY:** \[Low-Sugar / Diabetic-Friendly\] \[Low-Sodium\]        |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Carb\] \[High Iron\]     |
| \[B12-Rich\] \[High Potassium\] \[High Folate\]                      |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains wheat. Use GF pasta for GF/WF. Omit        |
| Parmesan for DF. White pasta = faster-digesting carbs ideal for      |
| pre-game. Eat 2--3 hours before competition. Keep fat low; avoid     |
| heavy cream sauces.*                                                 |
