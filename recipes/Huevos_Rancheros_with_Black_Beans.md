+----------------------------------------------------------------------+
| **\#18 Huevos Rancheros with Black Beans** \| Mexican · Breakfast    |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 15 min \| 🍽 Serves: 2                       |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 4 large eggs                                                     |
| >                                                                    |
| > • 4 corn tortillas (6-inch)                                        |
| >                                                                    |
| > • 1 cup canned black beans, rinsed and warmed                      |
| >                                                                    |
| > • ½ cup fresh or jarred low-sodium salsa                           |
| >                                                                    |
| > • ¼ cup crumbled cotija or shredded pepper jack cheese             |
| >                                                                    |
| > • 1 ripe avocado, sliced                                           |
| >                                                                    |
| > • 2 tbsp plain Greek yogurt or sour cream                          |
| >                                                                    |
| > • Fresh cilantro and lime wedge to serve                           |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Warm corn tortillas in a dry skillet or directly on a gas      |
| > flame for 30 seconds each side. Stack and cover.                   |
| >                                                                    |
| > 2\. Heat a non-stick pan with a little oil. Fry eggs sunny-side up |
| > or over-easy, seasoning with salt and pepper.                      |
| >                                                                    |
| > 3\. Warm black beans in a small pot; season with cumin and a pinch |
| > of salt.                                                           |
| >                                                                    |
| > 4\. Layer each plate: 2 tortillas, spoonful of beans, 2 fried      |
| > eggs, salsa, cheese, avocado, and a dollop of Greek yogurt.        |
| >                                                                    |
| > 5\. Scatter cilantro and squeeze lime over. Serve immediately.     |
