+----------------------------------------------------------------------+
| **\#3 Overnight Oats with Almond Butter & Banana** \| American ·     |
| Breakfast                                                            |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 1                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • ½ cup rolled oats (use certified GF oats if needed)              |
| >                                                                    |
| > • ¾ cup whole milk or oat milk                                     |
| >                                                                    |
| > • 2 tbsp natural almond butter                                     |
| >                                                                    |
| > • 1 ripe banana, sliced                                            |
| >                                                                    |
| > • 1 tbsp maple syrup or honey                                      |
| >                                                                    |
| > • ¼ tsp cinnamon, pinch of salt                                    |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Mix oats, milk, maple syrup, cinnamon, and salt in a jar or    |
| > container.                                                         |
| >                                                                    |
| > 2\. Stir well, seal, and refrigerate overnight (minimum 4 hours).  |
| >                                                                    |
| > 3\. In the morning, top with almond butter and sliced banana.      |
| >                                                                    |
| > 4\. Add a splash more milk if too thick. Serve cold or warm        |
| > briefly in microwave.                                              |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 16g \| Calories: 480 kcal \|      |
| Carbs: 65g \| Fat: 16g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Egg-Free\] \[Shellfish-Free\] \[Fish-Free\]         |
| \[Peanut-Free\] \[Sesame-Free\] \[Soy-Free\]                         |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[HighCarb\]                             |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Carb\] \[High Magnesium\] \[High          |
| Potassium\] \[Omega-3\]                                              |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains tree nuts (almond butter). Swap for        |
| sunflower seed butter to make TNF & PF. Use oat milk to make DF.*    |
