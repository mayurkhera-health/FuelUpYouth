+----------------------------------------------------------------------+
| **\#93 Fruit & Nut Butter Energy Smoothie** \| American · Pre-Game   |
| Fueling                                                              |
+======================================================================+
| ⏱ Prep: 5 min \| 🍳 Cook: 0 min \| 🍽 Serves: 2                        |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 ripe bananas (1 frozen)                                        |
| >                                                                    |
| > • 1 cup frozen mango chunks                                        |
| >                                                                    |
| > • 1 tbsp almond butter or peanut butter (omit for TNF/PF --- use   |
| > sunflower butter)                                                  |
| >                                                                    |
| > • 1 cup oat milk or low-fat milk                                   |
| >                                                                    |
| > • 1 tbsp honey                                                     |
| >                                                                    |
| > • ½ tsp vanilla extract                                            |
| >                                                                    |
| > • Pinch of cinnamon                                                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Blend all ingredients until smooth.                            |
| >                                                                    |
| > 2\. Pour into glasses; drink immediately.                          |
| >                                                                    |
| > 3\. For pre-game within 30--60 min: skip nut butter (too much fat  |
| > close to game time).                                               |
| >                                                                    |
| > 4\. For 2-hour window: include nut butter for sustained energy.    |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 8g \| Calories: 340 kcal \|       |
| Carbs: 62g \| Fat: 7g                                                |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Fish-Free\]             |
| \[Shellfish-Free\] \[Peanut-Free\] \[Soy-Free\] \[Sesame-Free\]      |
|                                                                      |
| **DIETARY:** \[Vegetarian\] \[DF\] \[Low-Sugar / Diabetic-Friendly\] |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Carb\] \[High Potassium\] \[Vitamin C\]   |
| \[High Magnesium\] \[High Folate\]                                   |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains tree nuts (almond/peanut butter). Use      |
| sunflower butter for TNF/PF. Use oat milk for DF. Fast to make on    |
| competition mornings --- all ingredients can be pre-frozen in bags   |
| for instant blending.*                                               |
+----------------------------------------------------------------------+
