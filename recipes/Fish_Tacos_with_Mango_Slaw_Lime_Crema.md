+----------------------------------------------------------------------+
| **\#47 Fish Tacos with Mango Slaw & Lime Crema** \| Mexican · Lunch  |
+======================================================================+
| ⏱ Prep: 12 min \| 🍳 Cook: 10 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 10 oz (280 g) tilapia or mahi-mahi fillets                       |
| >                                                                    |
| > • 1 tsp cumin, ½ tsp smoked paprika, ½ tsp garlic powder           |
| >                                                                    |
| > • Salt and pepper, 1 tbsp avocado oil                              |
| >                                                                    |
| > • 6 small corn tortillas                                           |
| >                                                                    |
| > • 1 cup shredded purple cabbage                                    |
| >                                                                    |
| > • ½ ripe mango, diced                                              |
| >                                                                    |
| > • Juice of 1 lime                                                  |
| >                                                                    |
| > • ¼ cup plain Greek yogurt + 1 tsp lime juice + pinch of salt      |
| >                                                                    |
| > • Fresh cilantro                                                   |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Season fish with cumin, paprika, garlic powder, salt, and      |
| > pepper.                                                            |
| >                                                                    |
| > 2\. Heat oil in a skillet over medium-high. Cook fish 3--4 min per |
| > side until flaky.                                                  |
| >                                                                    |
| > 3\. Toss cabbage and mango with lime juice and a pinch of salt for |
| > the slaw.                                                          |
| >                                                                    |
| > 4\. Mix Greek yogurt, lime juice, and salt for lime crema.         |
| >                                                                    |
| > 5\. Warm corn tortillas. Break fish into chunks; assemble tacos    |
| > with slaw, crema, and cilantro.                                    |
