+----------------------------------------------------------------------+
| **\#105 Teriyaki Beef & Broccoli Recovery Bowl (Japanese)** \|       |
| Japanese · Post-Game Recovery                                        |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 15 min \| 🍽 Serves: 4                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 1.5 lbs flank steak or sirloin, thinly sliced                    |
| >                                                                    |
| > • 4 cups broccoli florets                                          |
| >                                                                    |
| > • 2 cups cooked white rice                                         |
| >                                                                    |
| > • 3 tbsp low-sodium soy sauce (or tamari for GF)                   |
| >                                                                    |
| > • 2 tbsp honey                                                     |
| >                                                                    |
| > • 1 tbsp rice vinegar                                              |
| >                                                                    |
| > • 1 tsp sesame oil (omit for SesF)                                 |
| >                                                                    |
| > • 2 cloves garlic, minced                                          |
| >                                                                    |
| > • 1 tsp fresh ginger, grated                                       |
| >                                                                    |
| > • 1 tbsp sesame seeds (omit for SesF)                              |
| >                                                                    |
| > • 1 tbsp neutral oil                                               |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Whisk soy sauce, honey, rice vinegar, sesame oil, garlic,      |
| > ginger into teriyaki sauce.                                        |
| >                                                                    |
| > 2\. Heat oil in a large skillet or wok over high heat; sear beef   |
| > slices 2--3 min per side.                                          |
| >                                                                    |
| > 3\. Remove beef; stir-fry broccoli 4--5 min until bright green and |
| > slightly tender.                                                   |
| >                                                                    |
| > 4\. Return beef; pour sauce over and toss 1--2 min until           |
| > caramelized.                                                       |
| >                                                                    |
| > 5\. Serve over rice; garnish with sesame seeds.                    |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 40g \| Calories: 490 kcal \|      |
| Carbs: 46g \| Fat: 14g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Egg-Free\] \[Fish-Free\]             |
| \[Shellfish-Free\] \[Tree Nut-Free\] \[Peanut-Free\] \[Gluten-Free\] |
|                                                                      |
| **DIETARY:** \[GF\] \[DF\] \[Low-Sugar / Diabetic-Friendly\]         |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Iron\] \[High Zinc\]     |
| \[B12-Rich\] \[Vitamin C\] \[High Potassium\] \[High Magnesium\]     |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Contains soy and sesame. Use tamari for GF; coconut |
| aminos for SoyF; omit sesame for SesF. Beef is one of the richest    |
| bioavailable iron and zinc sources; broccoli adds Vitamin C which    |
| enhances iron absorption by up to 3x.*                               |
