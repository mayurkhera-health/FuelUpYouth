+----------------------------------------------------------------------+
| **\#34 Miso Ramen with Soft-Boiled Egg & Chicken** \| Japanese ·     |
| Lunch                                                                |
+======================================================================+
| ⏱ Prep: 10 min \| 🍳 Cook: 20 min \| 🍽 Serves: 2                      |
+----------------------------------------------------------------------+
| **INGREDIENTS**                                                      |
|                                                                      |
| > • 2 portions buckwheat soba noodles (or ramen noodles)             |
| >                                                                    |
| > • 2 boneless chicken thighs, poached and shredded                  |
| >                                                                    |
| > • 2 large eggs, soft-boiled (6--7 min)                             |
| >                                                                    |
| > • 3 cups low-sodium chicken broth                                  |
| >                                                                    |
| > • 2 tbsp white miso paste                                          |
| >                                                                    |
| > • 1 tsp low-sodium soy sauce or tamari                             |
| >                                                                    |
| > • 1 tsp sesame oil (omit for SesF)                                 |
| >                                                                    |
| > • 1 cup baby bok choy, halved and blanched                         |
| >                                                                    |
| > • 2 scallions, sliced                                              |
| >                                                                    |
| > • 4 slices bamboo shoots (optional)                                |
|                                                                      |
| **INSTRUCTIONS**                                                     |
|                                                                      |
| > 1\. Poach chicken thighs in simmering water 12--15 min. Remove and |
| > shred. Soft-boil eggs simultaneously.                              |
| >                                                                    |
| > 2\. Heat broth in a pot. Whisk miso and soy sauce into the broth   |
| > (do not boil after adding miso).                                   |
| >                                                                    |
| > 3\. Cook noodles per package instructions; drain and divide into   |
| > bowls.                                                             |
| >                                                                    |
| > 4\. Pour hot miso broth over noodles. Top with shredded chicken,   |
| > halved soft-boiled egg, bok choy, and scallions.                   |
| >                                                                    |
| > 5\. Drizzle sesame oil and garnish with bamboo shoots.             |
+----------------------------------------------------------------------+
| **NUTRITION PER SERVING** Protein: 34g \| Calories: 490 kcal \|      |
| Carbs: 50g \| Fat: 12g                                               |
+----------------------------------------------------------------------+
| **FREE FROM:** \[Dairy-Free\] \[Shellfish-Free\] \[Fish-Free\]       |
| \[Tree Nut-Free\] \[Peanut-Free\]                                    |
|                                                                      |
| **DIETARY:** \[Low-Sugar / Diabetic-Friendly\]                       |
|                                                                      |
| **NUTRIENT FOCUS:** \[High Protein\] \[High Carb\] \[B12-Rich\]      |
| \[High Iron\] \[High Zinc\] \[Vitamin D\]                            |
+----------------------------------------------------------------------+
| **TIP / SWAP:** *Use tamari and GF soba (100% buckwheat) for GF &    |
| SoyF. Omit sesame oil for SesF. Miso is fermented soy --- use        |
| chickpea miso for SoyF. One of the most warming recovery lunches.*   |
